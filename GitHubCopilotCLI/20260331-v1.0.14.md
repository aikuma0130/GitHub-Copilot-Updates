# GitHub Copilot CLI v1.0.14 - 2026-03-31 アップデート情報

## リリース日

2026-03-31

## GitHub Copilot CLI バージョン

1.0.14

## 概要

v1.0.14 では2件の新機能、4件の改善、20件のバグ修正、1件の機能削除が含まれています。CLI起動速度の向上、CPU使用率の削減、多数のMCPサーバー・クリップボード関連のバグ修正が主な変更点です。

## 変更点

### 新機能
- SDK セッション参加者が handlePendingElicitation API を通じて elicitation リクエストに応答可能に
- Kitty キーボードプロトコル対応ターミナルで Shift+Enter による改行挿入をサポート

### 改善
- スピナーレンダリングとタスクポーリングの最適化によりストリーミング中のCPU使用率を削減
- ターミナル検出・認証・git操作の並列実行によりCLI起動時間を短縮
- 自動リトライとリクエストタイムアウトによりMCPレジストリ検索の信頼性が向上
- V8コンパイルキャッシュにより繰り返し起動時のパース・コンパイル時間を短縮

### 削除
- gemini-3-pro-preview モデルのサポートを削除

### バグ修正
- BYOM使用時にAnthropicモデルへ画像が正しく送信されるように修正
- モデルピッカーの選択が現在のセッションで --model フラグを正しくオーバーライドするように修正
- エラー終了時にターミナル出力がクリアされたりジャンプしたりしなくなった
- Git マーケットプレイスURLのクローン失敗時に基盤となるエラー詳細を表示
- macOSでの一時ファイル操作で不要な権限プロンプトが表示されなくなった
- セッション終了時にシェルプロセスが適切にクリーンアップされるように修正
- SDK exit_plan_mode.requested イベントが、直接コールバック設定の有無にかかわらず常に発行されるように修正
- Microsoft Entra ID認証を使用するMCPサーバーでログインごとに同意画面が表示されなくなった
- タイムアウト到達時にgrep・glob検索結果が速やかに返されるように修正
- elicitation ダイアログで素早く入力した際にキーストロークが失われなくなった
- ネイティブWindowsでのクリップボードコピーで貼り付けテキスト先頭の不要な U+FEFF 文字が含まれなくなった
- セッション再開時に --config-dir が無視されてパスが ~/.copilot にフォールバックする問題を修正
- 許可リストポリシーでブロックされたMCPサーバーが /mcp show から非表示に
- Bring Your Own Model (BYOM) プロバイダー使用時に推論努力設定が正しく適用されるように修正
- クラシックPAT使用時のエラーメッセージを明確化
- grep ツールが大きなファイルや長い行でメモリ不足にならないように修正
- ACPモードでのMCPサーバーOAuth認証が動作するように修正
- $BROWSER をスペースで分割するように修正
- マウスサポートが有効な場合に貼り付けテキストが破損しなくなった
- マーケットプレイスプラグインのアンインストール時にキャッシュデータがディスクから削除されるように修正
