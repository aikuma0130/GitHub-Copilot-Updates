# GitHub Copilot CLI v1.0.15 - 2026-04-01 アップデート情報

## リリース日

2026-04-01

## GitHub Copilot CLI バージョン

1.0.15

## 概要

v1.0.15 では5件の新機能、6件の改善、2件の機能削除、5件のバグ修正が含まれています。MCP OAuth認証の強化、永続的なMCPサーバー設定管理RPCの追加、設定キーのキャメルケースへの移行が主な変更点です。

## 変更点

### 新機能
- mcp.config.list、mcp.config.add、mcp.config.update、mcp.config.remove サーバーRPCを追加し、永続的なMCPサーバー設定の管理が可能に
- ヘッドレスおよびCI環境でのMCP OAuthフォールバックとしてデバイスコードフロー（RFC 8628）を追加
- `/mcp auth` コマンドおよびMCP OAuthサーバーのアカウント切り替え対応の再認証UIを追加
- ツールエラー時の postToolUseFailure フックを追加し、postToolUse はツール呼び出し成功後のみ実行されるように変更
- `/share html` コマンドでセッションとリサーチレポートを自己完結型インタラクティブHTMLファイルとしてエクスポート可能に

### 改善
- Copilotマスコットがインタラクティブモードで微細な目のアニメーション（まばたき）を表示
- ユーザー切り替えおよび `/user list` でアカウントがアルファベット順に表示されるように改善
- diff ビューアーに Home/End および Page Up/Page Down ナビゲーションを追加
- セッション終了後にCLIが最大10秒待機せずすぐに終了するように改善
- askUser、autoUpdate、storeTokenPlaintext、logLevel、skillDirectories、disabledSkills などの設定がキャメルケース名に変更（スネークケースも引き続きサポート）
- 多くの設定キーがキャメルケース名を優先するように変更（スネークケース名も引き続き使用可能）

### 削除
- gpt-5.1-codex、gpt-5.1-codex-mini、gpt-5.1-codex-max モデルのサポートを削除
- Ctrl+D でメッセージをキューに入れる機能を削除（代わりに Ctrl+Q または Ctrl+Enter を使用）

### バグ修正
- Escape または Ctrl+C でキャンセル後に Autopilot が継続しなくなった
- CLI読み込み中に入力したキーストロークが失われなくなった
- 大きなツール出力プレビューが正しい文字数を表示し、最大500文字まで表示されるように修正
- 接続が遅いMCPサーバーがエージェントの開始をブロックしなくなった
- Windowsクリップボードからの画像貼り付けがWSL環境で動作するように修正
