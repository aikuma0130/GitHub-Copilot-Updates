# GitHub Copilot CLI v1.0.16 - 2026-04-02 アップデート情報

## リリース日

2026-04-02

## GitHub Copilot CLI バージョン

1.0.16

## 概要

v1.0.16 ではスクリプトによるツール権限リクエストの制御を可能にする PermissionRequest フックの追加と、MCPサーバーの安定性向上を中心としたバグ修正が含まれています。

## 変更点

### 新機能
- スクリプトからツールの権限リクエストをプログラム的に承認または拒否できる PermissionRequest フックを追加

### 改善
- MCPツール呼び出し時にタイムラインにツール名とパラメータの概要が表示されるように改善

### 削除
- 非推奨の `marketplaces` リポジトリ設定を削除（代わりに `extraKnownMarketplaces` を使用）

### バグ修正
- excludedTools または availableTools で sql ツールを除外した際に SQL プロンプトタグが表示されなくなった
- 作業ディレクトリ変更時に MCP サーバーが有効な認証で正しく再接続されるように修正
- ログイン、ユーザー切り替え、/mcp reload 後に MCP サーバーが正しく読み込まれるように修正
- BYOK Anthropic プロバイダーが設定された maxOutputTokens 制限を正しく適用するように修正
