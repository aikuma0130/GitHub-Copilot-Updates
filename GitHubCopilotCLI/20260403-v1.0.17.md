# GitHub Copilot CLI v1.0.17 - 2026-04-03 アップデート情報

## リリース日

2026-04-03

## GitHub Copilot CLI バージョン

1.0.17

## 概要

v1.0.17 では CLI にビルトインスキルが同梱されるようになり、MCP OAuth の HTTPS リダイレクト URI 対応やセッション復帰のパフォーマンス改善が行われました。

## 変更点

### 新機能
- CLI にビルトインスキルが同梱されるようになった（最初のスキルとして Copilot クラウドエージェント環境のカスタマイズガイドを搭載）

### 改善
- MCP OAuth フローで自己署名証明書フォールバックによる HTTPS リダイレクト URI をサポートし、HTTPS を必要とする OAuth プロバイダー（例：Slack）との互換性が向上
- /resume のセッション選択画面の読み込みが大幅に高速化（特に大量のセッション履歴がある場合）
