# GitHub Copilot CLI v1.0.18 - 2026-04-04 アップデート情報

## リリース日

2026-04-04

## GitHub Copilot CLI バージョン

1.0.18

## 概要

v1.0.18 では新たに Critic エージェントが導入され、プランや複雑な実装を補完モデルで自動レビューしてエラーを早期に検出できるようになりました。また、通知フックイベントの追加やセッション復帰の改善が含まれています。

## 変更点

### 新機能
- 新しい Critic エージェントが、補完モデルを使用してプランや複雑な実装を自動レビューし、エラーを早期に検出するようになった（Claude モデルの実験モードで利用可能）
- シェル完了・パーミッションプロンプト・エリシテーションダイアログ・エージェント完了時に非同期で発火する通知フックイベントを追加

### 改善
- preToolUse フックの permissionDecision に 'allow' を設定すると、ツール承認プロンプトが表示されなくなった

### バグ修正
- セッション復帰のピッカーが初回使用時にブランチとリポジトリごとに正しくグループ化されるようになった
