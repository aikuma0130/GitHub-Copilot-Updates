# GitHub Copilot CLI v1.0.19 - 2026-04-06 アップデート情報

## リリース日

2026-04-06

## GitHub Copilot CLI バージョン

1.0.19

## 概要

v1.0.19 では `/mcp enable` および `/mcp disable` コマンドがセッション間で永続化されるようになり、MCP サーバーの管理が改善されました。また、OpenTelemetry モニタリングの強化やプラグインフックスクリプトのバグ修正が含まれています。

## 変更点

### 新機能
- `/mcp enable` と `/mcp disable` がセッション間で永続化されるようになった

### 改善
- OpenTelemetry モニタリング：サブエージェントスパンが INTERNAL スパンカインドを使用するようになり、チャットスパンに `github.copilot.time_to_first_chunk` 属性が含まれるようになった（ストリーミング時のみ）
- セッションが別のクライアントで使用中の場合、IDE 自動接続をスキップするようになった
- スラッシュコマンドのタイムラインエントリにコマンド名（例：「Review」「Plan」）が含まれるようになり、コンテキストが改善された

### バグ修正
- macOS でプラグインフックスクリプトに実行権限がない場合でも正しく実行されるようになった
- エージェントの表示名がファイル名と異なるセッションを再開した際に、カスタムエージェントが正しく復元されるようになった
