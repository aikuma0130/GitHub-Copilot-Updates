# GitHub Copilot CLI v1.0.20 - 2026-04-07 アップデート情報

## リリース日

2026-04-07

## GitHub Copilot CLI バージョン

1.0.20

## 概要

v1.0.20 では `copilot help monitoring` トピックが追加され、OpenTelemetry の設定詳細と使用例が参照できるようになりました。また、スピナー表示やライブストリーミング中の UI パフォーマンスが改善されています。

## 変更点

### 新機能
- OpenTelemetry の設定詳細と使用例を含む `copilot help monitoring` トピックが追加された

### 改善
- バックグラウンドエージェントやシェルコマンドが完了するまでスピナーがアクティブなまま維持され、その間もユーザー入力が可能になった
- Azure OpenAI BYOK が API バージョン未設定時に GA バージョンレス v1 ルートをデフォルトで使用するようになった
- ライブレスポンスストリーミング中の UI のもたつきが軽減された
- `/yolo` と `--yolo` の動作が統一され、`/yolo` の状態が `/restart` 後も永続化されるようになった
