# GitHub Copilot CLI v1.0.21 - 2026-04-07 アップデート情報

## リリース日

2026-04-07

## GitHub Copilot CLI バージョン

1.0.21

## 概要

v1.0.21 では MCP サーバー管理用の `copilot mcp` コマンドが新たに追加されました。また、スピナー表示やタイムラインの不具合修正、メモリ使用量の削減など、安定性とパフォーマンスが向上しています。

## 変更点

### 新機能
- MCP サーバーを管理するための `copilot mcp` コマンドが追加された

### 改善
- Enterprise GitHub URL のログインフローでキーボード入力が受け付けられ、Enter キーで送信できるようになった
- プランモードのタイムライン表示で、冗長な「Plan」プレフィックスなしにユーザーテキストが表示されるようになった
- 不要になったシェルセッションを自動的にシャットダウンすることでメモリ使用量が削減された
- PascalCase で設定されたフックイベント名が、VS Code 互換の snake_case ペイロード（hook_event_name、session_id、ISO 8601 タイムスタンプ）で送信されるようになった

### バグ修正
- 長時間実行される非同期シェルコマンド実行中にスピナーが停止しているように見える問題が修正された
- スラッシュコマンドピッカーがフィルタリング中にちらつきや位置ずれを起こす問題が修正された
- コンテンツが縮小された際（キャンセルやツール完了後など）にタイムラインが空白になる問題が修正された
