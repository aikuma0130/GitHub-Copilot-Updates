# GitHub Copilot CLI v1.0.23 - 2026-04-10 アップデート情報

## リリース日

2026-04-10

## GitHub Copilot CLI バージョン

1.0.23

## 概要

v1.0.23 では、`--mode`、`--autopilot`、`--plan` フラグの追加により CLI を特定のエージェントモードで直接起動できるようになりました。また、スラッシュコマンドピッカーの改善やエージェント実行中のコマンド動作修正など、操作性の向上とバグ修正が行われています。

## 変更点

### 新機能
- `--mode`、`--autopilot`、`--plan` フラグが追加され、CLI を特定のエージェントモードで直接起動できるようになった
- Ctrl+L で会話セッションをクリアせずにターミナル画面をクリアできるようになった

### 改善
- スラッシュコマンドピッカーがスキルの完全な説明と改善されたスクロールバーを表示するようになった
- 推論トークンの使用量がゼロでない場合、モデルごとのトークン内訳に表示されるようになった
- リモートタブが Copilot コーディングエージェントのタスクを正しく表示し、Tasks API を通じたステアリングをサポートするようになった
- `.vscode/mcp.json` の移行通知に設定を `.mcp.json` に移行するための jq コマンドが含まれるようになった

### バグ修正
- メモリバックエンドが利用不可の場合、エージェントが最初のターンでハングする問題が修正された
- Bazel/Buck ビルドターゲットラベル（例: //package:target）がファイルパスとして誤認識される問題が修正された
- /diff、/agent、/feedback、/ide、/tuikit がエージェント実行中に動作するようになった
- BEL 文字を含むシェル出力がターミナルのビープ音を繰り返し鳴らす問題が修正された
