# GitHub Copilot CLI v1.0.24 - 2026-04-10 アップデート情報

## リリース日

2026-04-10

## GitHub Copilot CLI バージョン

1.0.24

## 概要

v1.0.24 では、preToolUse フックの機能拡張やカスタムエージェントのモデル指定の柔軟性向上などの改善が行われました。また、CLI クラッシュ後のターミナル状態の復元や --remote フラグの動作修正も含まれています。

## 変更点

### 改善
- preToolUse フックが modifiedArgs/updatedInput および additionalContext フィールドを正しく尊重するようになった
- カスタムエージェントのモデルフィールドが VS Code の表示名やベンダーサフィックスを受け付けるようになった（例: "Claude Sonnet 4.5"、"GPT-5.4 (copilot)"）
- 終了画面が Copilot マスコットとよりクリーンな使用状況サマリーレイアウトでリデザインされた

### バグ修正
- CLI が OOM やセグフォルトでクラッシュした後、ターミナルの状態（代替画面、カーソル、raw モード）が正しく復元されるようになった
- GitHub リポジトリで初回実行時のセッション同期プロンプトが表示される際に --remote フラグが正しく尊重されるようになった
