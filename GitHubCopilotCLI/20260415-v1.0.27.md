# GitHub Copilot CLI v1.0.27 - 2026-04-15 アップデート情報

## リリース日

2026-04-15

## GitHub Copilot CLI バージョン

1.0.27

## 概要

v1.0.27 では、会話履歴に影響を与えずに質問できる `/ask` コマンドやプラグインカタログ更新コマンドが追加されました。また、ステータスバーのヒント表示の改善や WSL でのクリップボードに関するバグ修正が含まれています。

## 変更点

### 新機能
- 会話履歴に影響を与えずに簡単な質問ができる `/ask` コマンドが追加された
- プラグインカタログを更新する `copilot plugin marketplace update` コマンドが追加された

### 改善
- Copilot Pro トライアルが一時停止されている場合に、汎用的なポリシーエラーではなく明確なメッセージが表示されるようになった
- ステータスバーが入力中に @files や #issues のヒントを表示し、スラッシュコマンドピッカーが開いているときに /help のヒントを表示するようになった

### バグ修正
- WSL でのクリップボードコピーが、貼り付けたテキストに不可視の BOM 文字をリークしなくなった
