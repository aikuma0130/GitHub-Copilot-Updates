# GitHub Copilot CLI v1.0.28 - 2026-04-16 アップデート情報

## リリース日

2026-04-16

## GitHub Copilot CLI バージョン

1.0.28

## 概要

v1.0.28 では、--resume ピッカーからリモートコントロールセッションへの接続やターミナルタイトル更新の無効化オプションが追加されました。また、権限プロンプトや巻き戻しピッカーの改善、マスコットアニメーションの変更など多くの修正が含まれています。

## 変更点

### 新機能
- --resume ピッカーから CLI リモートコントロールセッションに接続できるようになった
- ターミナルタイトルの更新を無効化する `COPILOT_DISABLE_TERMINAL_TITLE` 環境変数がサポートされた

### 改善
- git サブモジュール内で作業する際に、権限プロンプトが正しいリポジトリパスを表示するようになった
- MCP 移行ヒントが、インラインのシェルコマンドではなくプラットフォーム別の手順を含むドキュメントにリンクするようになった
- 巻き戻しピッカーのナビゲーションが矢印キーと Enter に簡素化され、紛らわしい 1-9 クイックセレクトショートカットが削除された
- 設定されたエディターが起動できない場合に明確なエラーメッセージが表示されるようになった
- マスコットが起動時に連続的に瞬きする代わりに短い瞬きシーケンスを再生するようになった
- `/clear` または `/new` の後にカスタムインストラクションとスキルがディスクから再読み込みされるようになった

### バグ修正
- `read_agent` が既に結果を待っている場合に、バックグラウンドエージェントの完了通知が冗長に送信されなくなった
- Azure リソース ID が az CLI コマンド実行時に誤ったパスセキュリティ警告をトリガーしなくなった
