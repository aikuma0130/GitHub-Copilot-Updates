# GitHub Copilot CLI v1.0.29 - 2026-04-16 アップデート情報

## リリース日

2026-04-16

## GitHub Copilot CLI バージョン

1.0.29

## 概要

v1.0.29 では、Claude Opus 4.7 のサポートや CI パイプラインでの環境検証用 `--list-env` フラグが追加されました。また、リモート MCP サーバー設定の簡素化やカーソル表示の安定化などの改善が含まれています。

## 変更点

### 新機能
- Claude Opus 4.7 モデルのサポートが追加された
- プロンプトモードで読み込まれたプラグイン、エージェント、スキル、MCP サーバーをログに記録する `--list-env` フラグが追加された
- シェルコマンドと MCP サーバーに `COPILOT_AGENT_SESSION_ID` 環境変数が提供されるようになった

### 改善
- リモート MCP サーバー設定で type フィールドを省略でき、デフォルトで http が使用されるようになった
- 点滅カーソルが安定した幅を維持し、点滅中にテキストがシフトしなくなった

### バグ修正
- エージェントがローカルユーザー名ではなく git リモート URL からリポジトリオーナーを正しく識別するようになった
- Windows でクラッシュ終了後にターミナル状態が正しく復元されるようになった
