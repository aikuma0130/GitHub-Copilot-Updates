# GitHub Copilot CLI v1.0.30 - 2026-04-16 アップデート情報

## リリース日

2026-04-16

## GitHub Copilot CLI バージョン

1.0.30

## 概要

v1.0.30 では、ステータスバーの表示項目をカスタマイズできる `/statusline` コマンドが追加されました。また、`--list-env` フラグの削除、画像貼り付けの回帰修正、プラグインスキルの検出修正などが含まれています。

## 変更点

### 新機能
- ステータスバーに表示する項目（ディレクトリ、ブランチ、effort、コンテキストウィンドウ、クォータ）をカスタマイズできる `/statusline` コマンド（`/footer` エイリアス付き）が追加された

### 改善
- フィードバックフォームが正しい GitHub リポジトリにリンクするようになった
- `/undo` が巻き戻しが利用できない場合（git リポジトリ外やコミットがない場合など）に説明メッセージを表示するようになった
- Ctrl+V と Meta+V の両方がすべてのプラットフォームで画像貼り付けをトリガーするようになった

### 削除
- プロンプトモードで読み込まれたプラグイン、エージェント、スキル、MCP サーバーをログに記録していた `--list-env` フラグが削除された

### バグ修正
- `skills.discover` 使用時にプラグインスキルとコマンドが正しく検出されるようになった
- ブラケットペースト処理の回帰後に、クリップボードからの画像貼り付けが再び動作するようになった
