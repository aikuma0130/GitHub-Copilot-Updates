# GitHub Copilot CLI v1.0.32 - 2026-04-17 アップデート情報

## リリース日

2026-04-17

## GitHub Copilot CLI バージョン

1.0.32

## 概要

モデル自動選択（`auto`）やドキュメントファイル添付、セッション再開時の短縮IDサポートなどの新機能が追加されました。また、レート制限時の自動リトライやテーブル描画の改善に加え、GHE認証やリワインド操作など多数のバグが修正されています。

## 変更点

### 新機能
- `--resume` および `/resume` でセッションIDの短縮プレフィックス（7文字以上の16進数）が使用可能に
- モデルに `auto` を選択すると、Copilotがセッションごとに最適なモデルを自動選択
- `--print-debug-info` フラグを追加し、バージョン、ターミナル機能、環境変数を表示
- 週間使用量の75%および90%に達した際に警告を表示
- プロンプトにサポートされたドキュメントファイルを添付してエージェントに読み込ませることが可能に
- `--connect` フラグを追加し、リモートセッションにIDで直接接続可能に

### 改善
- レート制限されたセッションでキューに入ったメッセージを一時停止し、自動的にリトライするように変更
- テーブルの列幅、絵文字サポート、ターミナルリサイズ時のボーダー安定性を改善
- レート制限エラーメッセージに、到達した制限の種類に応じた具体的なコンテキストを表示
- セッションアイドルタイムアウトが `--session-idle-timeout` で設定可能に（デフォルトは無効）
- トークン制限を超えるスキルでも名前で検出・呼び出しが可能に

### バグ修正
- `/feedback` が作業ディレクトリに書き込めない場合、バンドルをTEMPに保存するように修正
- `copilot login --host` がGitHub Enterprise Cloud（GHE）インスタンスで正しく認証されるように修正
- エージェントコンテキストの日時にローカルタイムゾーンオフセットが含まれるように修正
- ターミナルの進行インジケーターがエージェント処理中に表示され続けるように修正
- `/clear` 後にNeovimなどのターミナルでステータスラインに不要なUnicodeグリフが表示される問題を修正
- `/cd` でディレクトリ変更後にリワインドが正しく動作するように修正
- `/plan` およびプランモードでマルチライン入力が保持されるように修正
- 入力が空の場合のみBackspaceでシェルモードを終了するように修正
- `/ask` ダイアログでマウスホイールスクロールが正しく動作するように修正
