# GitHub Copilot CLI v1.0.33 - 2026-04-20 アップデート情報

## リリース日

2026-04-20

## GitHub Copilot CLI バージョン

1.0.33

## 概要

スラッシュコマンドのエイリアス追加やコマンド候補のサジェスト機能など、コマンド入力の利便性が向上しました。また、使用量制限の早期警告表示やvimスタイルのナビゲーション対応に加え、grepのタイムアウトやスキルピッカーの表示崩れなど複数のバグが修正されています。

## 変更点

### 新機能
- `--resume` または `--continue` でリモートセッションを再開する際に `--remote` フラグが自動的に引き継がれるように
- `/bug`、`/continue`、`/release-notes`、`/export`、`/reset` をコマンドエイリアスとして追加
- スラッシュコマンドピッカーが未認識またはスペルミスのコマンド入力時に類似コマンドを提案
- `/upgrade` を `/update` コマンドのエイリアスとして追加

### 改善
- `ctrl+t` による推論表示の切り替えが `/help` および `?` オーバーレイに表示されるように
- autoモードのサブエージェントがセッションモデルを引き継ぐように変更
- 使用量制限の50%および95%到達時に警告を表示し、レート制限到達前に早期通知
- タスクダイアログで `j`/`k` によるvimスタイルのナビゲーションと `x` によるタスク終了に対応

### バグ修正
- コンテンツ除外ポリシーが有効な大規模リポジトリでgrepがタイムアウトする問題を修正
- 非インタラクティブモードでバックグラウンドエージェントの完了を待機するように修正
- スキルピッカーがCJK/日本語の説明文や長いスキル名を折り返さず正しく切り詰めるように修正
- スラッシュコマンドピッカーでEnterキー押下時にハイライトされたコマンドが選択されるように修正
