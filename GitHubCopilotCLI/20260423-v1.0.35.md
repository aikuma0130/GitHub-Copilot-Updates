# GitHub Copilot CLI v1.0.35 - 2026-04-23 アップデート情報

## リリース日

2026-04-23

## GitHub Copilot CLI バージョン

1.0.35

## 概要

セッション管理の強化（名前付きセッション、削除サブコマンド）、HTTP フック対応、/usage の貢献グラフ追加など多数の新機能が追加されたほか、UI レンダリングやプラグイン即時反映などの改善、絵文字入力やサブエージェント表示など多数のバグ修正が含まれる大規模アップデートです。

## 変更点

### 新機能
- `COPILOT_GH_HOST` 環境変数による GitHub ホスト名の指定をサポート（`GH_HOST` より優先）
- Ctrl+Y でも補完ポップアップ（@メンション、パス補完、スラッシュコマンド）のハイライト項目を確定可能に
- `/session delete`、`delete <id>`、`delete-all` サブコマンドを追加し、セッションピッカーで x キーによる削除に対応
- `--name` でセッションに名前を付け、`--resume=<name>` で名前指定による再開が可能に
- `/usage` に GitHub スタイルの貢献グラフを追加（ターミナルのカラーモードに適応し、無色ターミナルでは異なるグリフにフォールバック）
- `continueOnAutoMode` 設定オプションを追加し、レート制限時に一時停止せず自動的に auto モデルに切り替え可能に
- HTTP フックサポートを追加し、ローカルコマンド実行の代わりに設定 URL へ JSON ペイロードを POST 可能に

### 改善
- スラッシュコマンドが引数とサブコマンドのタブ補完に対応
- シェルエスケープコマンド（!）が `$SHELL` 設定時にその値を使用するように変更（従来は常に /bin/sh を使用）
- リモートセッションでの CLI TUI における権限プロンプトが正しく表示されるように改善
- セッションセレクターにブランチ名、アイドル/使用中ステータスを表示し、カーソル対応の検索機能を改善
- モデル変更通知で変更前と変更後の両方のモデル名を表示
- `/update` と `/version` コマンドが設定済みの更新チャネルに従うように変更
- セッション同期プロンプトがより明確なラベルを使用し、GitHub.com のクロスデバイス同期について説明を追加
- `--continue` が最後に操作したセッションではなく、現在の作業ディレクトリのセッションを優先して再開
- ステータスラインスクリプトにモデルバッジおよび `/context` 出力と一致するコンテキストウィンドウフィールドを追加
- ユーザー設定が `~/.copilot/settings.json` に保存されるようになり、config.json の内部状態と分離
- lsp.json の LSP サーバーエントリでスポーン、初期化、ウォームアップのタイムアウトを設定可能に
- ステータスラインのコンテキストウィンドウインジケーターがデフォルトで非表示に変更
- MCP OAuth を共有ランタイムフローに移行し、MCP サーバー削除時に関連 OAuth 状態もクリア
- 大量テキスト表示時のタイムラインレンダリングパフォーマンスを改善
- `MULTI_TURN_AGENTS` 配下での同期タスク呼び出しが完了までブロックするように変更（60秒後の自動バックグラウンド昇格を廃止、フォローアップには `mode: "background"` を使用）
- タブナビゲーションで Home/End キーによる最初・最後のタブへのジャンプに対応
- プラグインがインストール後に再起動なしで即座に有効化
- grep および glob ツールが複数の検索パスを受け入れるように拡張

### バグ修正
- スペースや特殊文字を含む MCP サーバー名が正しくサポートされるように修正
- `-i` で初期プロンプトとして渡されたスキルスラッシュコマンド（例: /skill-name）が起動時に正しく認識されるよう修正
- `read_bash` が結果を既に返している場合にシェル完了通知が重複しないよう修正
- Windows で Copilot エージェントの設定がシェルアクセスを持つように修正
- Linux でクリップボードユーティリティ（wl-clipboard または xclip）が未インストール時にインストール手順付きのエラーメッセージを表示
- エージェントループ内でのカスタムツール呼び出しの自己修正に対応
- 絵文字やマルチコードポイント文字の入力時のカーソル移動、削除、レンダリングが正しく動作するよう修正
- Windows でのツール利用可能性検出が正しく動作するよう修正
- ターン中のセッショントークン期限切れが自動的に処理され、メッセージの再送信が不要に
- `/cwd` と `/add-dir` パスピッカーでの初回タブ・矢印キーナビゲーションが正しい項目を選択するよう修正
- IDE やエクステンション切断時に一時的な I/O エラーがタイムラインに赤いエラーエントリとして表示されないよう修正
- `~/.claude/` 内のカスタムエージェントとスキルが Copilot プロジェクト設定として誤って読み込まれないよう修正
- 認証後にログインコマンドがインタラクティブ入力を正しく復元するよう修正
- 設定済みの推論エフォートをサポートしないモデルへの切り替え時に auto モードがエラーで失敗しないよう修正
- パターン固有の instruction ファイル（.github/instructions/*.instructions.md）がセッションごとにシステムプロンプトに全文を含めないよう修正
- エクステンションのシャットダウンエラーがセッション終了時にエラーレベルのログノイズとして表示されないよう修正
- LSP 設定が存在する場合に LSP リファクタリングツールが最初のターンで正しく登録されるよう修正
- サブエージェントの思考がタイムラインから非表示に変更
- カスタムエージェント名がステータスラインのフッターに表示され、`/statusline` で切り替え可能に
- 起動ダイアログでの Escape キー押下時にレースコンディションが発生しないよう修正
