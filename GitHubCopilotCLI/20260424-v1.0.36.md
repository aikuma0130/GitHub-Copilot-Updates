# GitHub Copilot CLI v1.0.36 - 2026-04-24 アップデート情報

## リリース日

2026-04-24

## GitHub Copilot CLI バージョン

1.0.36

## 概要

`/keep-alive` コマンドの正式公開、`/remote` コマンドの拡張、ステータスラインへの変更行数表示トグル追加などの新機能に加え、サブコマンドピッカーの UI 改善やダブル Esc によるキャンセル誤操作防止など、操作性とセキュリティ面の改善が含まれるアップデートです。

## 変更点

### 新機能
- `/keep-alive` が実験モードなしで利用可能に（Copilot CLI アクティブ時のシステムスリープを防止）
- `/remote` コマンドが現在のステータスを表示し、`/remote on` と `/remote off` でリモートコントロールの切り替えに対応
- ステータスラインに 'changes' トグルを追加し、セッションの追加/削除行数を表示

### 改善
- サブコマンドピッカーでハイライト項目の横に選択インジケーター（❯）を表示
- 複数の Copilot ライセンスが検出された場合に直接リンク付きのより明確なエラーメッセージを表示
- 無効化されたスキルがスラッシュコマンドリストに表示されないように変更
- 実行中の作業をキャンセルするために Esc のダブルタップが必要に変更（誤操作防止）
- `~/.claude/` からのカスタムエージェント、スキル、コマンドが Copilot CLI に読み込まれないように変更
- Claude Opus 4.6 がデフォルトで中程度の推論エフォートを使用するように変更

### バグ修正
- `preToolUse.matcher` が無視される問題を修正（アップグレード後、matcher 付きフックは正規表現に完全一致するツール名のみで実行）
- `.gitignore` されたディレクトリ（例: `.github/instructions/`）内のカスタム instruction ファイルが正しく読み込まれるよう修正
- デバッグログやフィードバックバンドルの保存時に既存のアーカイブファイルを上書きしないよう修正
