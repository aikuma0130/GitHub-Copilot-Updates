# GitHub Copilot CLI v1.0.37 - 2026-04-27 アップデート情報

## リリース日

2026-04-27

## GitHub Copilot CLI バージョン

1.0.37

## 概要

ロケーションベースの権限永続化がデフォルト有効化され、`copilot completion` サブコマンドによるシェル補完スクリプト生成機能が追加されたほか、セッションピッカーのソート順切り替えや `/ask` のマークダウンレンダリング対応など、利便性の向上とバグ修正が含まれるアップデートです。

## 変更点

### 新機能
- ロケーションベースの権限永続化がデフォルトで有効化（同じディレクトリで承認がセッション間で引き継がれるように）
- `copilot completion <bash|zsh|fish>` サブコマンドを追加し、サブコマンド・フラグ・選択肢の静的シェル補完スクリプトを生成可能に
- セッションピッカーで `s` キーを押すとソート順を切り替え可能に（関連度、最終使用、作成日、名前）

### 改善
- ACP モデル設定オプションに configOptions API を使用するクライアント向けの説明とメタデータを追加
- モデルやエフォートの変更通知が同じモデルやエフォートレベルを再選択した際に表示されないように改善
- `/ask` のレスポンスがテーブルやフォーマット済みリンクを含むマークダウンをレンダリングするように改善

### バグ修正
- Linux でクリップボード書き込み時に X11 ハンドルがリークする問題を修正
- 保留中メッセージインジケーターがプロンプトフレームと共に正しく表示されるよう修正
- `git branch --show-current` への切り替え後に detached HEAD 検出が常に false を返す問題を修正
- スキルにエラーや警告がある場合にスキルピッカーリストが完全に表示されるよう修正
