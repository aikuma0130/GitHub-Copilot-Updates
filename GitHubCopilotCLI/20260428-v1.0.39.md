# GitHub Copilot CLI v1.0.39 - 2026-04-28 アップデート情報

## リリース日

2026-04-28

## GitHub Copilot CLI バージョン

1.0.39

## 概要

ACP セッション向けのスラッシュコマンド追加や allow-all 権限モードの切り替え機能が導入されたほか、実行中タスクのバックグラウンド移動、セッションピッカーの改善、パイプエラーによるクラッシュ修正などが含まれるアップデートです。

## 変更点

### 新機能
- ACP クライアントがセッション設定で allow-all 権限モードを切り替え可能に
- ACP セッション向けに `/compact`、`/context`、`/usage`、`/env` スラッシュコマンドを追加
- `ctrl+x → b` で実行中のタスクやシェルコマンドをバックグラウンドに移動可能に

### 改善
- `/remote` ステータス出力に各接続状態のアクション可能なヒントを表示するように改善
- `--resume` セッションピッカーのタブレイアウト、ステータス表示、プログレッシブローディングを改善
- スラッシュコマンドの引数ピッカーがコマンド境界で即座に開くように改善（末尾スペースが不要に）

### バグ修正
- 子プロセスの stdio ストリームでの一時的なパイプエラーがクラッシュや誤ったクラッシュレポートを引き起こす問題を修正
