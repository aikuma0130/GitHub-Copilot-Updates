# GitHub Copilot CLI v1.0.42 - 2026-05-06 アップデート情報

## リリース日

2026-05-06

## GitHub Copilot CLI バージョン

1.0.42

## 概要

MCP サーバーのエラー診断の改善、`-C` フラグによる作業ディレクトリ変更のサポート、実験的なラバーダックエージェントの追加など、利便性と安定性を向上させるリリースです。

## 変更点

### 新機能
- GPT セッション向けのラバーダックエージェントを追加（Claude を利用、`/experimental` で利用可能）
- `git -C` と同様に、起動前に作業ディレクトリを変更する `-C` フラグを追加

### 改善
- MCP サーバー障害の警告で、サーバー名にスペースが含まれる場合に直接実行可能な `/mcp show` コマンドを提案するように
- MCP サーバー障害の警告に stderr 出力を含め、接続エラーの診断を支援
- 終了メッセージのレジュームコマンドで、セッション名が変更されていない場合にセッション ID を表示するように
- リモートセッションのエクスポートが GitHub 以外のリポジトリやリポジトリなしのディレクトリをサポート

### バグ修正
- セッション再開時に「Go back」を選択した後に誤った「session in use」警告が表示されなくなった
- リクエストキャンセル後に Enter キーが永久にスタックしなくなった
- ユーザーメッセージがなく保存済みセッションもない場合に終了サマリーを非表示に
- Windows での CLI アップデート時にパッケージ展開中の一時的な EPERM エラーで ENOENT が発生しなくなった
