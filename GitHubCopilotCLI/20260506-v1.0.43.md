# GitHub Copilot CLI v1.0.43 - 2026-05-06 アップデート情報

## リリース日

2026-05-06

## GitHub Copilot CLI バージョン

1.0.43

## 概要

ステータスラインへのユーザー名表示の追加、自動モードでのサーバーサイドモデルルーティングの採用、悪意のあるベアリポジトリに対する RCE 脆弱性の修正を含むセキュリティおよび機能改善リリースです。

## 変更点

### 新機能
- `/statusline` ピッカーにユーザー名トグルを追加し、フッターにアクティブなアカウントを表示可能に

### 改善
- 自動モードでサーバーサイドモデルルーティングを使用し、リアルタイムのモデル選択を改善
- アップデートコマンド実行時にダウンロード進捗を表示するように

### バグ修正
- 複数セッションがアクティブな場合にレジュームプロンプトで正しいセッション名を表示するように修正
- プロジェクト内にネストされた悪意のあるベアリポジトリからの RCE（リモートコード実行）を防止（[GHSA-9ccr-r5hg-74gf](https://github.com/github/copilot-cli/security/advisories/GHSA-9ccr-r5hg-74gf)）
- MCP サーバーの子プロセス（npx や uvx 経由で起動されたもの）がセッション終了時に完全に終了されるように
