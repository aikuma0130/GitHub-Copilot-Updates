# GitHub Copilot CLI v1.0.44 - 2026-05-08 アップデート情報

## リリース日

2026-05-08

## GitHub Copilot CLI バージョン

1.0.44

## 概要

スラッシュコマンドの入力中使用やプレリリースビルドの取得機能の追加、シェルコマンドやクォータ表示など多数のバグ修正を含むリリースです。

## 変更点

### 新機能

- スラッシュコマンドが入力途中でも使用可能になり、1つのメッセージで複数のスキルを呼び出せるように
- `userPromptSubmitted` フックがリクエストを直接処理できるようになり、LLM を経由せずにレスポンスを返すことが可能に
- `copilot update` および `/update` に任意の `prerelease` 引数を追加し、最新のプレリリースビルドを取得可能に

### 改善

- マルチアカウントユーザー向けに `/user list` および `/user switch` の動作を高速化
- シェルエイリアスや rc ファイルの設定が `!` コマンドで正しく動作するように
- タイムラインで rubber-duck サブエージェントの解決済みモデルを表示するように（例: Rubber-duck(claude-opus-4.7)）

### バグ修正

- `/add-dir` のパス補完がちらついたり、`@` や `#` ピッカーに横取りされる問題を修正
- `!` プレフィックスによるシェルコマンドがすべてのシェル設定で正しく動作するように修正
- Free ユーザーのクォータ表示が常に100%使用済みと表示される代わりに、正しい残量を表示するように修正
- オートパイロットモードで許可したツール権限が `/clear` 後も保持されるように修正
- `/model` ピッカーでモデルを切り替えた際にエフォートレベルが正しく適用されるように修正
- パーミッションプロンプト表示中に Ctrl+C を押しても CLI がハングしなくなるように修正
- スラッシュコマンドピッカーで結果が一致しない場合もプロジェクト情報が表示されるように修正
- `settings.json` 内の無効な URL エントリが CLI の起動をクラッシュさせず、警告付きでスキップされるように修正
