# GitHub Copilot CLI v1.0.45 - 2026-05-11 アップデート情報

## リリース日

2026-05-11

## GitHub Copilot CLI バージョン

1.0.45

## 概要

`/autopilot` や `/fork` などの新しいスラッシュコマンドの追加、OpenTelemetry の GenAI セマンティック規約への準拠、CLI 起動速度の改善を含むリリースです。

## 変更点

### 新機能

- `/autopilot` スラッシュコマンドを追加し、インタラクティブモードとオートパイロットモードの切り替えが可能に
- `/fork` コマンドを追加し、現在のセッションを新しい独立したセッションにフォーク可能に

### 改善

- Windows で PowerShell 7+（pwsh）が利用できない場合に Windows PowerShell（powershell.exe）にフォールバックするように
- OpenTelemetry 出力が GenAI セマンティック規約に準拠：MCP ツール呼び出しが標準の tool_call スパンを使用し、新しい gen_ai.client.operation.duration メトリクスでツール実行時間を追跡
- OSC カラークエリのサポートが限定的なターミナルで CLI の起動が高速化し、起動時間を最大約1.5秒短縮

### バグ修正

- 拡張機能のパーミッションプロンプトを含むセッションが「Session file is corrupted」エラーなしで再開できるように修正
- `task_complete` でエージェントが停止した際に `agentStop` フックが正しく発火するように修正
