# GitHub Copilot CLI v1.0.46 - 2026-05-12 アップデート情報

## リリース日

2026-05-12

## GitHub Copilot CLI バージョン

1.0.46

## 概要

CLI バージョンが非推奨になった際の警告表示の追加、diff ビューの長い行の折り返し対応、読み取り専用の gh CLI コマンドの自動承認など、利便性とセッション安定性を向上させるリリースです。

## 変更点

### 新機能

- CLI バージョンが非推奨になった場合にプレミアムモデルへのアクセスが失われる可能性がある旨の警告を表示するように

### 改善

- diff ビューで長い行がターミナル幅で折り返されるようになり、切り詰められなくなった
- 読み取り専用の gh CLI コマンド（list、view、status、diff など）がユーザー確認のプロンプトなしで自動承認されるように

### バグ修正

- pwsh が .NET グローバルツール shim としてインストールされている場合に PowerShell が正しく起動するように修正
- ターン途中で ERR_HTTP2_INVALID_SESSION エラーによりセッションがクラッシュする問題を修正
