# GitHub Copilot CLI v1.0.47 - 2026-05-13 アップデート情報

## リリース日

2026-05-13

## GitHub Copilot CLI バージョン

1.0.47

## 概要

/fork コマンドへのオプション名指定の追加、diff ビューでの j/k キーナビゲーション対応、Copilot cloud agent セッションの --resume サポートなど、操作性を向上させるリリースです。

## 変更点

### 新機能

- /diff ビューで j/k キーによる上下ナビゲーションをサポート

### 改善

- /fork コマンドでオプションの名前を指定可能になり、フォークされたセッションがセッションダイアログで元のセッションを表示するように
- --resume がエージェントがブランチにまだ変更をプッシュしていない Copilot cloud agent セッションをサポートするように

### バグ修正

- Copilot Max サブスクライバーがサブスクリプションティアで利用可能な正しいモデルを表示するように修正
