# GitHub Copilot CLI v1.0.48 - 2026-05-14 アップデート情報

## リリース日

2026-05-14

## GitHub Copilot CLI バージョン

1.0.48

## 概要

モデルピッカーでのトークン価格表示の改善、CJK 文字や絵文字の表示修正、指示ファイルの glob パターン処理修正など、課金表示の改善と複数のバグ修正を含むリリースです。

## 変更点

### 改善

- モデルピッカーがトークンベース課金ユーザーに対してドットインジケーターの代わりに実際のトークン価格を表示するように改善
- /ask ダイアログが受信できないフォローアップ返信のプロンプトを表示しなくなった
- モデルに注入されるスキルコンテンツに YAML フロントマターメタデータが含まれなくなった

### バグ修正

- applyTo フロントマターで引用符なしの glob パターン（例: `applyTo: **/*.ts`）を持つ指示ファイルが正しく適用されるように修正
- CJK 文字や絵文字を含む入力テキストが行間に空白のギャップなしでレンダリングされるように修正
- /context がすべてのモデルに対して常に 128k を表示する代わりに正しいトークン制限を表示するように修正
- プロンプト/ヘッドレスモードで Azure DevOps 専用ワークスペースの場合にビルトインの github-mcp-server を自動無効化するように修正（インタラクティブモードの動作と一致）
- ターミナルカーソルが装飾要素ではなく入力フィールド上に正しく配置されるように修正
- アクティブモデル変更時に ACP クライアントが更新された設定オプションを受信するように修正
