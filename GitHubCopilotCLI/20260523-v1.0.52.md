# GitHub Copilot CLI v1.0.52 - 2026-05-23 アップデート情報

## リリース日

2026-05-23

## GitHub Copilot CLI バージョン

1.0.52

## 概要

会話ビューへのスクロールバー追加、セッション再開時の作業ディレクトリ復元、コンテキストウィンドウティア選択のエンドツーエンド適用、カスタムエージェントの遅延ツール読み込みサポートなど、UI・セッション管理・MCP 連携の改善を含むリリースです。

## 変更点

### 新機能

- メイン会話ビューにマウスドラッグ対応の縦スクロールバーを追加
- カスタムエージェントがエージェントフロントマターの `deferred-tool-loading` によるオプトイン遅延ツール読み込みをサポートし、大量のツールリストを持つエージェントでのツール検索ディスカバリーが可能に
- コンテキストウィンドウティア選択（デフォルト約200Kトークン vs 1Mトークン）がエンドツーエンドで適用され、ティアの選択がコンパクション・トランケーション・トークン表示を実際に制約するように

### 改善

- 非インタラクティブサブコマンド（plugin list、mcp list、help、version）が stdin を消費しなくなった
- 推論トークンがトークン使用量サマリーの出力トークン数に括弧表記で表示されるように改善
- ステータスラインコマンドが実行可能スクリプトパスに加えてプレーンシェルコマンドもサポート
- 起動時に ~/.copilot/logs/ の古いプロセスログファイルを自動的に削除し、無制限のディスク増大を防止
- /statusline ピッカーのアイテム説明をクリーンにし、スペーシングを改善
- ピッカーチェックボックスが単一セルの ▣/▢ グリフを使用し、ピッカー全体でよりタイトで一貫した行を実現
- /compact がオプションのフォーカス指示を受け付け、コンパクションサマリーの形成が可能に
- 汎用サブエージェントが GPT-5.4 または GPT-5.5 を利用可能な場合に使用するように改善
- /usage がセッションおよび週間制限のクォータプログレスバーを表示するように改善
- AI クレジットエラーメッセージがより明確な文言と「Manage budget」リンクを含むように更新

### バグ修正

- Autopilot モードへの切り替え時にツール・パス・URL アクセスの予期しない許可プロンプトが表示されなくなるように修正
- `copilot --continue` でセッションの保存ディレクトリから実行した際に、保存されたブランチと git コンテキストが古いままにならず更新されるように修正
- kill コマンドの安全フィルターが `kill -0 <PID> 2>/dev/null` のようなシェルリダイレクションを含む有効なコマンドを拒否しなくなるように修正
- セッションが保存された作業ディレクトリで再開されるように修正（-C で上書き可能、相対パスを値に持つフラグは保存された cwd から解決）
- Responses API 使用後のセッションで AI クレジット使用量が正しく表示されるように修正
- tmux on Cygwin または mintty 使用時のレンダリングのカクつきを修正
- スラッシュコマンドピッカーで行が選択された際に (experimental) および (staff) ラベルがオレンジ色を維持するように修正
- URL/URI フィールドに非 URL 文字列を含むイベントがあるセッションが「Session file is corrupted」エラーなしで再開されるように修正
- HTTP/2 アップロードの停滞によりタイムアウトしたリクエストが自動的に HTTP/1.1 で再試行されるように修正
- Windows でプロセスが高ビット終了コード（例：.NET 未処理例外）で終了した場合にセッションの読み込みが失敗しなくなるように修正
- タイムラインエントリのコネクタカラーが展開時に周囲の要素と一致するように修正
- truecolor 非対応ターミナルでユーザーメッセージの背後に灰色の背景バーが表示されなくなるように修正
- 終了サマリーで `AI Credits` ラベルが値の前に正しいスペーシングで表示されるように修正
- /restart および /update が再起動後に現在のセッション ID を保持するように修正
- MCP サーバー設定のレガシーなネストされた `oauth.clientId` および `oauth.callbackPort` キーがサイレントに破棄される代わりにサポートされる `oauthClientId` および `auth.redirectPort` キーに移行されるように修正
- MCP OAuth の再認証が設定された redirectPort を尊重するように修正
- Windows で PowerShell の除算演算子が誤って「Allow directory access」プロンプトをトリガーしなくなるように修正
