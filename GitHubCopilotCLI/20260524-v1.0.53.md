# GitHub Copilot CLI v1.0.53 - 2026-05-24 アップデート情報

## リリース日

2026-05-24

## GitHub Copilot CLI バージョン

1.0.53

## 概要

マルチラインプロンプトの表示問題、/skills ピッカーの設定ディレクトリ対応、Bash シェルセッションのハング問題を修正したバグ修正リリースです。

## 変更点

### バグ修正

- マルチラインプロンプトがコンテンツのクリッピングや選択オフセットなしに完全に表示されるように修正
- /skills ピッカーがスキル設定の保存時に --config-dir を正しく尊重するように修正
- 環境に PS0 または PROMPT_COMMAND が設定されている場合に Bash シェルセッションがハングしなくなるように修正
