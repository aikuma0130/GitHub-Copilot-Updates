# GitHub Copilot CLI v1.0.55 - 2026-05-28 アップデート情報

## リリース日

2026-05-28

## GitHub Copilot CLI バージョン

1.0.55

## 概要

Claude Opus 4.8 のサポート追加、/autopilot コマンドの導入、セルベースターミナルレンダラーの全ユーザーデフォルト有効化など、多数の新機能・改善・バグ修正を含む大型アップデートです。

## 変更点

### 新機能

- Claude Opus 4.8 のサポートを追加
- /autopilot コマンドを追加してオートパイロットモードを維持（/goal をエイリアスとして利用可能）
- `permissions.disableBypassPermissionsMode` 設定を追加し、allow-all/yolo モードの有効化を防止
- フック進行状況ストリーミングにより、長時間実行フックのリアルタイムステータスメッセージをタイムラインに表示
- session.create および session.resume RPC に pluginDirectories を追加し、SDK クライアントがセッションごとに Open Plugins 形式のディレクトリをマウント可能に
- リモートセッションをセッションピッカーから直接削除可能に

### 改善

- Free/Student プランのトークンベース課金ユーザーは Auto モデル選択に制限され、モデルピッカーに説明を表示
- Claude thinking（推論）トークンをセッション使用量サマリーに表示
- MCP サーバー設定フォームで Ctrl+S 押下時に最新の入力値を保存
- /mcp でサーバーごとのトークン使用量を表示し、/context で MCP ツールトークンを分離表示
- カスタムエージェントとスキルがサブディレクトリを再帰的に検出されるように改善
- 特定サブスクリプションプランのモデル選択動作を更新
- セルベースターミナルレンダラーを全ユーザーにデフォルトで有効化
- リモート制御セッションが組織ポリシーで無効化されている場合に警告を表示
- 拡張機能のログファイルを拡張機能ごとにキャプチャし、extensions_manage ツールで表示可能に
- .github/extensions のプロジェクト拡張機能を非 git（フォルダベース）ワークスペースでも検出
- /statusline と /theme コマンドをエージェント実行中でも実行可能に
- MCP 設定を専用画面で開き、コンテンツが表示領域を超える場合にスクロール可能なサーバー/ツールリストを表示
- owner/repo#ref 構文でのプラグインマーケットプレイス追加をサポート
- フィードバックダイアログと /skills ヘルプテキストで Copilot 統一ログパスと用語を使用
- tmux 3.6b のペーン進行状況ステートとネイティブ統合
- --plugin-dir スキルが personal-home（~/.copilot, ~/.agents）スキルより優先されるよう変更（順序: project > plugin-dir > personal > custom）
- 推論トークン数をセッショントークンサマリーに全ユーザー向けに表示
- 全カラーテーマで選択背景のコントラストを向上
- /env で読み込まれた拡張機能のステータスとソースを表示

### バグ修正

- 信頼されていないフォルダで起動時にローディングスピナーが永久にハングする問題を修正
- exit_plan_mode ツールがプランモード中のみモデルに提供されるよう修正
- ネイティブバイナリクラッシュ（SIGSEGV 等）時にサイレント終了ではなく JavaScript フォールバックに移行するよう修正
- PowerShell 7 が Microsoft Store App Execution Alias としてインストールされている場合の検出を修正
- ゼロサイズ CAPI 課金バッチのセッション再開が正しく動作するよう修正
- スケジュールマネージャーのヒントバーテキストがダイアログ境界を越えて折り返される問題を修正
- `copilot update` と `copilot version` がリリース API リクエストを認証し、共有 NAT 環境でのレート制限エラーを回避
- Diff ビューのキーボードショートカットヒントが unstaged/branch diff モード切替時に正しく表示されるよう修正
- wlr-data-control 非対応の Wayland コンポジター（GNOME/Mutter 等）でクリップボードペーストが正しく動作するよう修正
- インタラクティブシェルツールが親ターミナルのカラー設定を保持するよう修正
- オプションオブジェクト入力スキーマの Canvas ツールがバリデーションエラーなしで正しく開くよう修正
- 古い CLI バージョンからフォークされた拡張機能サブプロセスで "Invalid command format" エラーが発生しないよう修正
- 設定マイグレーションが古い CLI バージョンのレガシー snake_case キー存在時にユーザーデータを保持するよう修正
- ターミナルベルが設定で明示的に有効化されない限りターン完了時に鳴らないよう修正
- /resume ピッカーでメッセージ送信前に閉じたセッションの空行を非表示に修正
- セッション中止時に Task ツールエージェント実行中でも UI が Cancelling 状態でスタックしないよう修正
- vote_memory ツール呼び出しをレスポンスおよびインタラクションごとにスロットリングし、暴走的な投票バーストを防止
- マウス選択をタイムライン上部を超えてドラッグした際に上方自動スクロールが正しく動作するよう修正
- Windows で CJK および補助面 Unicode 文字のクリップボードコピーを修正
- 拡張機能がシングル実行可能アプリケーション（SEA）として実行される際に正しく起動するよう修正
