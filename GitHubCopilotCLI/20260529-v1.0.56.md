# GitHub Copilot CLI v1.0.56 - 2026-05-29 アップデート情報

## リリース日

2026-05-29

## GitHub Copilot CLI バージョン

1.0.56

## 概要

Free/Student ユーザーの Auto 以外のモデル選択が可能になり、Diff ビューの大幅なレイアウト改善、MCP ツールの structured content 対応修正など、多数の改善とバグ修正を含むアップデートです。

## 変更点

### 新機能

- `builtInAgents.rubberDuck` 設定を追加し、`copilot config` でラバーダックエージェントの有効/無効を切り替え可能に

### 改善

- Free/Student ユーザーがモデルピッカーで Auto 以外のモデルを選択可能に
- ThemePicker のサイドバイサイドレイアウトが 120 カラムターミナル内で折り返しなく収まるよう改善
- モデルピッカーで料金ティアごとの正確な合計コンテキストウィンドウサイズを表示
- Diff ビューがスティッキーファイル/ハンクヘッダー付き連続スクロールレイアウトに変更、フルターミナル幅対応、テーマ対応カラー
- web_fetch ツールが HTTP コンテンツネゴシエーションにより Markdown コンテンツを優先取得し、ドキュメントサイトからよりクリーンな結果を返すよう改善
- コードレビューエージェントが固定デフォルトではなく現在のセッションと同じモデルを使用するよう変更
- gh CLI が PATH 上にある場合、GitHub MCP サーバーが重複する gh 代替可能ツールをデフォルトで省略し、トークン使用量を削減
- 信頼フォルダ確認メッセージでパーミッションがセッション中記憶される可能性があることを明示

### バグ修正

- tmux で Kitty キーボードプロトコルが利用できない場合の拡張キーレポーティングが正しく動作するよう修正
- 設定ファイルをアトミックに書き込み、複数 CLI プロセス同時実行時のデータ損失を防止
- BYOK プロバイダー設定が ACP セッションに正しく適用されるよう修正
- MCP ツールが human-readable な `content` テキストと `structuredContent` ペイロードの両方を返す場合、両方をエージェントに表示するよう修正（MCP spec §5.2.6 に基づくリテラル JSON シリアライゼーションの場合は重複排除）
- /context の小トークン凡例フォーマットとフリースペースグリッドの丸め処理を修正
- 推論エフォートピッカーがモデル機能を尊重し、サポートされないオプションを非表示に修正
- /env 出力のファイルパスが正しいフォーマットで表示されるよう修正
- 推論テキストが会話タイムラインでアシスタント応答の上に常に表示されるよう修正
- アシスタント応答がターミナルタイムラインで単語孤立行なしにレンダリングされるよう修正
- タブ文字を含むテキスト貼り付け後のカーソル位置が正しくなるよう修正
- コンテキストウィンドウティア選択がセッションイベントに永続的に保存され、SDK のみの再開パスでもティア制限が正しく再適用されるよう修正
- リモートセッション URL がリテラル 'copilot' ではなくリポジトリの owner/name を正しく使用するよう修正
