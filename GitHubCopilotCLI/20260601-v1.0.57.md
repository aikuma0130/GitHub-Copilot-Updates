# GitHub Copilot CLI v1.0.57 - 2026-06-01 アップデート情報

## リリース日

2026-06-01

## GitHub Copilot CLI バージョン

1.0.57

## 概要

プロセスツリー全体の終了によるシェルコマンドキャンセルの改善、デフォルトネットワークトランスポートの HTTP/1.1 への変更、Diff モードでのマウスクリック選択対応など、多数の改善とバグ修正を含むアップデートです。

## 変更点

### 新機能

- `copilot update` で GitHub API レート制限に達した場合にアクション可能なエラーメッセージを表示
- Canvas プロバイダーが open 結果で file:// URL を返せるように対応（ローカルファイルプレビュー用）
- Diff モードでマウスクリックによるライン選択が可能に
- `showTipsOnStartup` 設定を追加し、起動時のヒント表示を制御可能に

### 改善

- プラグインスラッシュコマンド（/plugin install, uninstall, update, marketplace add/remove/browse）が操作中に即時フィードバックを表示
- 実行中のシェルコマンドのキャンセル（Ctrl+C）時に、孤立プロセスを残さずプロセスツリー全体を終了するよう改善
- シンボリックリンクされたディレクトリが /cwd の補完候補に表示されるよう改善
- Azure DevOps のみのリポジトリで、GitHub MCP サーバーが完全無効化ではなく web_search ツールのみを公開するよう変更
- クォータフッターが残リクエスト数を丸めたパーセンテージで表示するよう改善
- デフォルトネットワークトランスポートを HTTP/1.1 に変更し、一部のネットワークパスでの信頼性を向上（HTTP/2 は COPILOT_ENABLE_HTTP2=1 でオプトイン可能）
- ハイコントラスト diff 背景がテキスト可読性向上のため暗い色を使用するよう改善
- preToolUse フックエラーが静かに許可せずツール呼び出しを拒否するよう変更
- /diff がアンステージされた変更がない場合にブランチ diff をデフォルトで表示するよう変更

### バグ修正

- /lsp show、/lsp test、/lsp reload がサブディレクトリから CLI 起動時にプロジェクト LSP 設定を正しく検出するよう修正
- MCP サーバーのタイムアウト設定がツールリスト変更後も保持されるよう修正
- /skills add と /skills remove がクォートで囲まれたパス（例：Windows エクスプローラーの「パスのコピー」）を正しく処理するよう修正
- 引用符なしの複数単語プロンプトで `copilot` を実行した際に commander エラーではなく「プロンプトを引用してください」のヒントを表示するよう修正
- リポジトリ設定から自動インストールされたプラグインがユーザーグローバル設定に漏出しないよう修正
- Grep ツールが tsx と jsx をファイルタイプフィルターとして正しく処理するよう修正
- COPILOT_HOME がサーバー検出レジストリディレクトリに正しく適用されるよう修正
- tmux 内で Ctrl+C やその他の修飾キーが正しく動作するよう修正
- @メンションのファイル検索がクエリの大文字小文字に関係なくファイルにマッチするよう修正
- `copilot plugin marketplace list` がリポジトリレベルの `extraKnownMarketplaces` 設定を正しく適用するよう修正
- フッターのキュー済みプロンプトが1行に制限され、セッションメッセージが画面外に押し出されないよう修正
- npx --registry で設定された MCP サーバーがポリシーにより不正にブロックされないよう修正
- 内部イベント処理中のエラー後にセッションが無限にハングしないよう修正
- インストール済みプラグインがプラグインソースリポジトリの .git ディレクトリを含まないよう修正
- ツール呼び出し後の新しい推論がタイムライン上部ではなく下部に表示されるよう修正
- ブラウザ、エディタ、ターミナルからコピーしたテキストの貼り付けで空行、壊れた罫線、カーソル位置のずれが発生しないよう修正
- クラッシュにより部分的なデータがセッションログに残った後のセッション再開が正しく動作するよう修正
- SDK auth-token 検証失敗時に誤解を招くメッセージではなく根本原因（例：GitHub API レート制限）を表示するよう修正
