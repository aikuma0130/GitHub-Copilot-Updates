# GitHub Copilot CLI v1.0.58 - 2026-06-02 アップデート情報

## リリース日

2026-06-02

## GitHub Copilot CLI バージョン

1.0.58

## 概要

Rubber Duck エージェントとリモート JSON RPC がデフォルトで有効化され、スケジュールプロンプトや新しい GitHub テーマ、Issue・PR・Gist への簡易アクセスを含む新 UI が実験的機能として追加されました。

## 変更点

### 新機能

- Rubber Duck エージェントがデフォルトで有効化
- リモート JSON RPC がデフォルトで有効化
- `/experimental` スケジュールプロンプト機能を追加（`/every` および `/after` コマンド）
- `/experimental` 新しい GitHub TUI テーマを追加
- `/experimental` Issue、Pull Request、Gist への簡易アクセスを備えた新 UI を追加
