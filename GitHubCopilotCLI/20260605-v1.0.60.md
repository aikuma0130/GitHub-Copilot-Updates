# GitHub Copilot CLI v1.0.60 - 2026-06-05 アップデート情報

## リリース日

2026-06-05

## GitHub Copilot CLI バージョン

1.0.60

## 概要

Anthropic モデルの最大推論エフォートレベルの追加、Rubber Duck エージェントの自動呼び出し設定、vim スタイルの diff ナビゲーション、PR からの git worktree 直接作成など、多数の新機能と改善を含む大型アップデートです。

## 変更点

### 新機能

- Anthropic モデルに最大推論エフォートレベルを追加し、すべてのエフォートレベルをすべてのプランで利用可能に
- `builtInAgents.rubberDuckAutoInvoke` 設定を追加し、Rubber Duck エージェントの自動呼び出しを制御可能に（デフォルトは無効）
- /usage 表示でキャッシュ読み取りトークンと並んでキャッシュ書き込みトークンを表示
- ctrl+s を現在のプロンプトのスタッシュ＆ポップに再割り当て（Claude Code と同等の操作性）
- /context でカスタムインストラクションをシステムプロンプトから分離し、サーバーごとの MCP ツールトークンコストを /mcp と相互参照
- `billing` ヘルプトピックを追加し、AI クレジット使用機能の概要を提供
- /diff ビューに vim スタイルのナビゲーションキー（g, G, Ctrl+D, Ctrl+U）を追加
- /session info ビューで Mission Control の共有ステータスを表示
- --resume の短縮形として -r を追加
- LSP サーバー設定で `bash`、`powershell`、`cwd` キーを受け付けるよう拡張。`cwd` 展開で `PLUGIN_ROOT` 等のプラグイン変数をサポート
- Rewind ピッカーで各チェックポイントの作業ツリー diff 統計（+追加 −削除）を表示
- プルリクエスト画面から直接 git worktree を作成可能に
- /session info テーブルでリモートエクスポートまたはステアリング時のセッションリンクを表示
- /env コマンドでアクティブなフックのカウントとソース出所を表示
- /help コンテンツに不足していたキーボードショートカット（?, ctrl+q, ctrl+r, ctrl+z, ctrl+y, shift+enter）を追加
- ベアな #number イシュー・PR 参照を現在の git リポジトリに自動リンク
- 動的ローダーや git-config 環境変数（例：LD_PRELOAD, GIT_EXTERNAL_DIFF）を設定するコマンド実行前に承認プロンプトを表示

### 改善

- スラッシュコマンドのパス引数で `..` の親ディレクトリ移動をタブ補完可能に（タブ切り替えではなく）
- ステータスラインが狭いターミナル幅で要素を認識不能に切り詰めるのではなく行をまたいで段階的にスタック
- IDE ピッカーがエディタ接続が切れたエントリを非表示にし、同じエディタとフォルダを共有するエントリにプロセス ID を付加
- モデルピッカーが小さなターミナルウィンドウに収まるようになり、マウススクロールが動作
- スキルおよびスラッシュコマンドピッカーの行が複数行の説明を1行として正しく表示
- 残りリクエストのパーセンテージが超過ユーザーに対して負の値を表示しないよう改善
- 拡張機能の権限プロンプトが起動時に --yolo と事前承認済みの場所を尊重
- カスタムエージェントの指示が各ターンで重複しなくなり、コンテキストウィンドウの使用量を削減
- セッション完了シグナル（ターミナルビープ、オートパイロット継続）がバックグラウンドシェルコマンドの完了を待機
- macOS プロンプト入力で Cmd+Backspace がカーソル前の行を削除
- web_fetch がループバック、プライベート、クラウドメタデータアドレスをブロックし、リダイレクトを黙って追跡しないよう変更
- MCP ツールがサーバーによりターン中に追加・削除された場合、同じターン内で即時利用可能に
- /tasks 詳細ビューでバックグラウンドエージェントへのフォローアッププロンプト送信後に最新のプロンプトを表示
- バイパスパーミッションポリシーを --allow-all-tools、--allow-all-paths、--allow-all-urls フラグに適用
- プロンプトで "help" と入力するとチャットメッセージとして送信する代わりにクイックヘルプオーバーレイを開く
- --cloud のエラーメッセージで experimental モードが無効な場合に /experimental の有効化方法を説明

### バグ修正

- ターミナルマルチプレクサ内でスリープから復帰後に画面が空白のままになる問題を修正
- ハイライトされたフレーム内の入力フィールドの背景色が正しくレンダリングされるよう修正
- プラン承認およびレビューフィードバックプロンプトでカーソルが正しい位置にレンダリングされるよう修正
- PR ブランチにスラッシュが含まれる場合に worktree ディレクトリがフラット名を使用するよう修正（例：`cli/foo` → `.worktrees/cli-foo`）
- kitty キーボードプロトコルがアクティブな場合にキューヒントが ctrl+q ではなく ctrl+enter を正しく表示するよう修正
- X11 でのクリップボード操作がターミナル表示を破損しないよう修正
- Windows で bare name（例：`git`）で呼び出す際にワーキングディレクトリから実行ファイルが発見されないよう修正
- 大量の出力を生成する対話型シェルコマンドがハングしないよう修正
- /context レジェンドの MCP ツールグリフが正しいサイズで表示されるよう修正
- allowedHosts または blockedHosts が設定されている場合に Linux サンドボックスが失敗しないよう修正
- 実験割り当てが同時キャッシュされる際に信頼フォルダやその他の設定キーが失われないよう修正
- Rewind が前のスナップショットへのロールバック時に無視ファイルを削除しないよう修正
- ACP allow_all 設定オプションがツール、パス、URL に対して無制限のアクセス許可を正しく適用するよう修正
- --available-tools、--excluded-tools、--reasoning-effort フラグが ACP モードで正しく適用されるよう修正
- LSP workspace/configuration レスポンスが正しいエントリ数を返し、ty のような厳密なサーバーがパニックしないよう修正
- ディレクトリシンボリックリンク経由でリンクされた拡張機能が正しく検出・ロードされるよう修正
- ワイド文字（例：CJK）がターミナル diff ビューで視覚的な破損なしに正しくレンダリングされるよう修正
- フォルダの信頼が再プロンプトなしで git worktree 間で持続するよう修正
- マーケットプレイスの強制削除後にプラグインが次回起動時に再インストールされないよう修正
- MCP OAuth 再認証がログインが既に進行中の場合にアドレス使用中エラーで失敗しないよう修正
- リポジトリプラグインのオーバーライドがグローバルに有効なプラグイン設定を変更しないよう修正
- MCP allowlist が先頭の @ を省略したレジストリエントリを持つ npm スコープサーバーに正しくマッチするよう修正
- Azure API Center 経由で登録された MCP サーバーが allowlist により不正にブロックされないよう修正
- シリアライズされたトークンブローカー（例：M365）を共有するローカル MCP サーバーが断続的に失敗せず確実に起動するよう修正
- BYOK ファイル添付の 5 MiB を超えるファイルが OpenAI Responses プロバイダー経由で正常に送信されるよう修正
- git リポジトリ外で実行している場合に /init の提案が表示されないよう修正
