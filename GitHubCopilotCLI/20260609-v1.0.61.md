# GitHub Copilot CLI v1.0.61 - 2026-06-09 アップデート情報

## リリース日

2026-06-09

## GitHub Copilot CLI バージョン

1.0.61

## 概要

/settings 対話ダイアログ、/worktree コマンド、Claude Fable 5 モデルのサポート追加、MCP サーバーのワークスペース自動読み込みなど、多数の新機能と改善を含むアップデートです。

## 変更点

### 新機能

- /settings 対話ダイアログを追加し、すべてのユーザー設定を一箇所で閲覧・編集可能に
- /after および /every コマンドが /experimental スラッシュコマンドリストに表示されるよう追加
- .github/mcp.json ワークスペース設定ファイルから MCP サーバーを自動ロード
- Claude Fable 5 モデルのサポートを追加
- /agent ピッカーで '/' を押してエージェント名でフィルタリング可能に
- settings.json の `tabs` 設定でホームタブバーの表示・順序・非表示タブを設定可能に
- /every および /after で自然言語を使用して cron 式、カレンダー時刻、相対期間でタスクをスケジュール可能に
- mTLS およびプライベート CA による OTLP テレメトリの HTTPS エクスポートサポートを追加
- フルスクリーンスクロールバーを追加
- http/protobuf OTLP HTTP エクスポートを標準 OTel プロトコル環境変数経由で追加
- /worktree コマンド（エイリアス /move）を追加。新しい git worktree を作成して切り替え、コミットされていない変更を一緒に移動
- `beepOnSchedule` 設定を追加し、スケジュールされた `/every` および `/after` 実行の完了ビープを無効化可能に

### 改善

- /agents ピッカーと「新しいエージェントを作成」ウィザードの境界線、ヘッダー、入力スタイルを統一
- /env 出力が内部フックを非表示にし、フックソースの完全なファイルパスを表示
- ピッカー（例：/agent）の番号キー選択が10以上のアイテムで動作するよう改善
- シンボリックリンクされたディレクトリが @-file ピッカーの候補に表示されるよう改善
- フック進捗ステータス行が一時的とマークされた場合、会話タイムラインに蓄積するのではなくその場で折りたたまれるよう改善
- /fork がフォーク作成中に「Creating fork...」進捗通知を表示
- /mcp search が外部レジストリで正しく動作するよう改善
- ライトテーマのセカンダリ背景色が正しくレンダリングされるよう改善
- 検索バーのマッチ数がプロンプトフレーム内に収まるよう改善
- GitHub テーマがライトターミナルに本格的な GitHub Primer ライトカラーパレットで適応
- 大規模モノレポでの grep 検索がインデックスエンジンを使用し大幅に高速化
- `/sessions` がオーバーレイを開く代わりに Sessions タブへナビゲート
- プロンプトモードが stderr にモデルロードエラーを表示（サイレント終了ではなく）
- プラグインインストールがネットワークエラーで設定を取得できない場合でもマネージドマーケットプレイスポリシーを適用
- `/help` が `$HOME/.copilot/instructions/**/*.instructions.md` を他のユーザーレベルのインストラクションの場所と並べて表示
- WSL および tmux セッションで色が劣化パレットにフォールバックせず正しくレンダリング
- 空のプロンプトで Esc または Ctrl+C を押してシェルモードを終了可能に（Backspace に加えて）

### バグ修正

- セッション再開時に画面が空白になるバグを修正
- メモリ無効でローカルセッションを再開した際に UI が空白画面にクラッシュする問題を修正
- Gemini モデルが nullable スキーマタイプを使用する MCP ツールで正しく動作するよう修正
- 既存リンク内の GitHub イシュー・PR 参照が壊れたネストされた自動リンクを作成しないよう修正
- Bash ツールがコマンド入力でマルチバイト UTF-8 文字（em ダッシュ、カーリークォートなど）を正しく処理するよう修正
- MCP OAuth 再認証がリモートサーバーの保存された OAuth クライアント ID を正しく使用するよう修正
- 権限ダイアログを閉じた後に貼り付けた画像がメインプロンプトにリークしないよう修正
- grep および glob ツールが単一パス引数を正しく処理し、検索結果の見落としを防止するよう修正
- シェルコマンドバリデーションで文字列リテラルやヒアドキュメント内の「kill」のような単語を含む無害なコマンドをブロックする誤検知を修正
