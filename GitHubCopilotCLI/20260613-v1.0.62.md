# GitHub Copilot CLI v1.0.62 - 2026-06-13 アップデート情報

## リリース日

2026-06-13

## GitHub Copilot CLI バージョン

1.0.62

## 概要

エリシテーションダイアログのスクロール改善、プラグインによるエクステンション配布、diff ビューのコンテンツ検索・ファイルツリー追加、企業プロキシの Kerberos 自動認証など、多数の新機能と改善を含むアップデートです。

## 変更点

### 新機能

- プラグインがエクステンションを配布できるようになり、プラグインマーケットプレイス経由でインストール可能に
- diff ビューにコンテンツ検索、マッチハイライト、n/N ナビゲーションを追加
- /app スラッシュコマンドを追加。GitHub アプリまたはブラウザフォールバックを開く
- ユーザー設定または /subagents（/agents）ピッカーでサブエージェントのモデル、推論エフォート、コンテキストティアを設定可能に
- 'YOLO'（すべて許可）インジケーターをフッターに表示し、カスタム statusLine.command に allow-all 状態を追加
- Issues または Pull Requests タブで `/` を押して GitHub のサーバーサイドフィルタリングで検索可能に
- セッションスコープのエクステンションとキャンバスを追加
- SDK クライアントが session.create および session.resume を通じてセッションメモリを設定可能に
- 企業フォワードプロキシを Kerberos/Negotiate（SPNEGO）で自動認証
- /diff ビューにファイルツリーサイドバーとインラインコメントエディタを追加
- W キーを押して展開されたイシューまたはプルリクエストの詳細ビューから worktree を作成可能に
- /every および /after でスラッシュコマンドをスケジュール可能に（例：/every 1d /chronicle standup）

### 改善

- MCP サーバー設定フォームをピッカーベースのフローで使いやすく改善
- エリシテーションダイアログがタイムラインと一緒にスクロールするようになり、高さのあるダイアログがエージェントの出力を隠さなくなった
- 推論サマリーセクション間の空行を保持
- 検索チップにユーザー入力のコロンタームを表示
- BYOK Responses プロバイダーで max_output_tokens を尊重
- シンボリンクされたディレクトリ外の設定ルートからスキルをロード可能に
- モデルピッカーが現在選択されているモデルを含むタブで開くよう改善
- シェルコマンドが疑似ターミナルではなく軽量プロセススポーンで実行されるよう変更
- GitHub テーマの色コントラストを WCAG AA アクセシビリティ基準に準拠するよう改善
- ACP セッション設定オプションの説明を表示
- ウォームセッションでブランチおよび HEAD 検出を高速化
- ライトテーマのセカンダリ背景色を正しくレンダリング

### バグ修正

- PowerShell リダイレクトパスがコンテンツ除外の拒否をトリガーしなくなった
- WebSocket トランスポートが Tokio ランタイム外でクリーンに閉じるよう修正
- シェルツールエラーがシェル ID が停止、完了、または回収されたことを説明するよう修正
- 音声ランタイムダウンロードダイアログがインストール失敗後にループで再表示されなくなった
- MCP サーバー名のドットとスラッシュが有効な Responses API 名前空間にマッピングされるよう修正
- `code-insiders --wait` などのエディタコマンドが Windows で正しく起動するよう修正
- オーバーサイズのインライン画像から正常に回復し、ターンが失敗しなくなった
- ビジョンが無効またはモデルが非対応の場合の画像添付がセッションの残りを汚染しなくなった
- /tasks からバックグラウンドに昇格されたシェルがターン終了後も実行を継続するよう修正
- バックグラウンドヘルパーエージェントから内部の無効化ツールメッセージを非表示に
- サンドボックスツールが mxc-sdk がホスト環境で提供される場合に正しくロードされるよう修正
- ネストされた .github/agents および .claude/agents ディレクトリのカスタムエージェントがサブディレクトリからセッション開始時に検出されるよう修正
- ツール権限プロンプトの承認が同じツールコールに対して2回目のプロンプトを引き起こさなくなった
- View ツールプロンプトが 50KB ではなく 20KB の切り捨て制限を正しく表示するよう修正
- ワークスペース MCP サーバーがループで再起動しなくなった
- BYOK プロバイダー使用時にカスタムエージェントが設定されたモデルを維持するよう修正
- 一時的なコンテンツポリシーエラーからセッションを再起動せずに回復するよう修正
- リレーセッションで Autopilot がクリーンに継続し、/plan が短いプロンプトを表示するよう修正
- Windows で Git コマンドがコンソールウィンドウをフラッシュしなくなった
- Claude 形式プラグインの `preToolUse` および `permissionRequest` フックが `Bash`、`Read`、`*` などのツールマッチャーで正しく発火するよう修正
- アクティブテーマがセッション中に変更された際にターミナルカラーがライブ更新されるよう修正
- ストリーミングされたアシスタントテキストがタイムラインで断続的に重複しなくなった
- grep が欠落した検索パスをスキップし、失敗せずに有効な結果を継続するよう修正
- リモート MCP OAuth サーバーが各サブエージェントごとに再起動せず、一致する設定ごとに1回のみ起動するよう修正
- ネストされたサブエージェントがターミナル入力をブロックせずに並行実行制限を尊重するよう修正
- マーケットプレイス ref が完全修飾タグ（例：refs/tags/v2.1.0）の場合にプラグインインストールが動作するよう修正
