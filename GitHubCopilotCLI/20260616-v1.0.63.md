# GitHub Copilot CLI v1.0.63 - 2026-06-16 アップデート情報

## リリース日

2026-06-16

## GitHub Copilot CLI バージョン

1.0.63

## 概要

画像添付ブロック時のエラーメッセージ改善、`/rewind` コマンドの Git 不要化、MCP サーバー設定の新オプション追加など、ユーザー体験とプロバイダー接続の信頼性を向上させるアップデートです。

## 変更点

### 新機能

- MCP サーバー設定に `deferTools` オプションを追加。ツール検索が有効な場合でもサーバーのツールを常に利用可能に保持
- `/diff` で `w` キーを押すと空白のみの変更を非表示にできるように
- フォークベースのプルリクエストを `/pr` およびブランチ PR バッジに表示
- `read_bash` 出力が大きすぎる場合にスピルファイルパスを表示
- 最近のローカルセッションを `/chronicle standup` に含めるように
- `/responses` WebSocket 接続を復元
- イシュー詳細ビューで Enter キーを押すとハイライトされたイシューを開くように
- 実験的機能：`/rewind` が Git を不要にし、Copilot が変更したファイルのみを復元（ユーザー自身の編集はそのまま維持）。会話のみまたは会話＋ファイルの選択肢を提供

### 改善

- 画像添付がブロックされた場合、ビジョンの有効化方法・対応モデルへの切替・別の画像の使用など具体的な対処法を表示するように改善
- `--help` 出力のオプションがアルファベット順にソート（2つのロングフラグを持つオプションも含む）
- 認証検証エラー（VPN や IP 許可リストの失敗など）がサインインバナーにネットワークアクセスの確認ガイダンスとともに表示されるように
- ローカルとリモートのリポジトリ名が大文字小文字のみ異なる場合でもリモートセッションを再開可能に
- エージェントモードがセッション単位で追跡されるようになり、セッションの作成・クリア・切替時に引き継がれなくなった
- プランレビューメニューが厳格な OpenAI 互換バックエンドで動作するように
- コマンド履歴検索中に入力フッターの逆検索表示が正しく整列するように
- `PostToolUse` フックマッチャー（例：`Edit|Write`）が正しく適用され、フォーマッタやリンターが対象ツールの後にのみ実行されるように
- OpenAI、Anthropic、Azure OpenAI リクエストの信頼性を向上

### バグ修正

- ネイティブランタイムアドオンが破損したホストプロセスヒープにロードされた際の Windows クラッシュを防止
- 読み取り不可能なネイティブドキュメント添付からファイルパスアップロードへのフォールバックで回復するように修正
- HMAC および OAuth モードでの一時的な 401 認証失敗をリトライするように修正
