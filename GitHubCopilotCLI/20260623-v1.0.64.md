# GitHub Copilot CLI v1.0.64 - 2026-06-23 アップデート情報

## リリース日

2026-06-23

## GitHub Copilot CLI バージョン

1.0.64

## 概要

GitHub テーマのデフォルト化、インライン画像レンダリングの追加、MCP サーバー認証の強化、オートパイロットモードの自動プロンプト処理など、UI の統一性とプロバイダー接続性を大幅に向上させるアップデートです。

## 変更点

### 新機能

- CLI にインライン画像レンダリングを追加
- BYOK OpenAI 互換プロバイダー向けに WebSocket レスポンスサポートを追加
- シェルコマンドのサンドボックスをバイパスするオプションを追加
- ページネーションリストでマウスクリックおよびダブルクリック選択を追加
- ユーザー設定で HTTP(S) プロキシを設定可能に
- 会話スクロールバーを非表示にする設定を追加
- スキルにアーギュメントヒントのフロントマター対応を追加
- `/diff` が非 Git フォルダで Copilot の変更のセッション差分を表示
- `/branch` を `/fork` のエイリアスとして追加（Claude Code のコマンド命名に合わせる）
- `/agent` 名のタブ補完を追加
- モデル設定に opus、sonnet、haiku、gpt、gemini などのモデルファミリーエイリアスを追加
- Windows Terminal 向けに Ctrl+Backspace バインディングを `/terminal-setup` に追加
- リモート MCP サーバー向けにホスト提供 OAuth トークンの SDK サポートを追加
- `/diagnose` コマンドを追加してセッションログを分析
- `/mcp` レジストリインストールを追加して MCP サーバーの閲覧とインストールが可能に
- `/security-review` を全ユーザーに `--experimental` なしで利用可能に
- MCP ツール向けに CSV 出力サポートを追加
- `/loop` を `/every` コマンドのエイリアスとして追加
- 実験的機能：`--worktree [name]` (`-w`) フラグを追加。`<repo>.worktrees/` 配下に Git ワークツリーを作成・再利用してセッションを開始
- 実験的機能：コンパクトタイムラインでツールコールや推論行をクリックして展開・折りたたみが可能に
- セッション再開ピッカーにフォルダ列を追加して各セッションの作業ディレクトリを表示
- インストール済みプラグインの MCP サーバーを `mcp` コマンドで表示
- インストール済みプラグインが提供する MCP サーバーを自動検出

### 改善

- GitHub テーマをデフォルトで使用し、ホームタブとプロンプトフレームを全ユーザーに有効化
- パスアクセスプロンプトで解決済みシンボリックリンクのターゲットを表示し、付与されるアクセス権を明確化
- 従量課金制の追加利用予算を起動時に表示し、制限到達時にフレンドリーメッセージを表示
- 再開セッションが元の添付ファイル参照を再現し、ファイル変更後もプロンプトキャッシュリセットを回避
- コンテンツ除外がルールサービスに到達不能な場合にすべてのファイルをブロックしなくなり、バックグラウンドでリトライ
- `/subagents` でラバーダックサブエージェントを設定可能に（補完モデル戦略を含む）
- スペースを含むセッション名でもセッション再開が可能に
- リモートホストセッションで未対応のスラッシュコマンドを非表示に
- ターミナルリサイズ後も出力の位置揃えを維持
- MCP 組織ポリシーをセッション作成・リロード時に適用
- `/settings` 検索フィールドが readline 編集キーとカーソル移動をサポート
- `--help` 出力に `--remote-export` と `--no-remote-export` を表示
- コンパクトタイムラインの展開されたシェルエントリで長いコマンドと説明を折り返して表示
- Markdown テーブル内のリンクをクリック可能に
- `/usage` でモデルごとのトークン合計を表示し、大規模履歴スキャンを高速化
- オートパイロットモードがエージェントの `task_complete` 呼び出し後にインタラクティブモードに復帰
- オートパイロットモードがエリシテーション、ask_user、サンプリング、許可プロンプトを自動処理
- `--continue` と `--resume` が現在のリポジトリの最新セッションを選択
- セッション再開が大規模履歴ロード中もレスポンシブを維持
- コード検索とワークツリー一覧表示の高速化
- 装飾絵文字の代わりにプレーンテキストラベルを使用
- タイムラインでシェルコマンドをシンタックスハイライト
- システムのライト/ダークモード変更に自動追従
- フッターダイアログがユニファイドビューでタイムラインとスクロール
- `/diff` ツリーでファイル名クリックでそのファイルの最初の変更にジャンプ
- Markdown でインラインコードをテーマ付きチップスタイルでレンダリング
- スラッシュコマンドピッカーの名前列を 25 文字から 35 文字に拡大
- `/diff` ビューで長い行を折り返して表示
- `/diff` ホットキーラベルを改善
- OpenTelemetry GenAI スパンが `gen_ai.usage.cache_read.input_tokens`、`gen_ai.usage.cache_creation.input_tokens`、`gen_ai.usage.reasoning.output_tokens` を仕様に準拠して出力
- OpenTelemetry GenAI チャットスパンが `gen_ai.request.reasoning.level` を出力

### 削除

- 旧来のインテント報告ツールを CLI から削除
- ターミナル報告カラースキームサポートを削除
- `/terminal-setup` で作成された不正な Ctrl+Enter VS Code キーバインドを削除

### バグ修正

- コロンを含むフリーテキスト検索語（例：`CLI:`）がイシューおよびプルリクエスト検索で正しい結果を返すように修正
- PowerShell コマンドレット（Select-String、Where-Object、ForEach-Object）が不要なディレクトリアクセスプロンプトをトリガーしなくなった
- 非インタラクティブプロンプト出力が列 1 に正しく留まるように修正
- ビジョン無効時にキュー済みツール画像をクリア
- モデル変更時に新しいモデルが適用されるまで待機するように修正
- `2>/dev/null` リダイレクトをシェル安全性プロンプトで読み取り専用として処理
- 外部エディタでプロンプトを開く際に編集テキストを LF に正規化
- フルallow-allセッションでコンピュータ使用同意プロンプトをスキップ
- `/clear` 後もリモートエクスポートが継続し、`/session info` がタスク URL を保持
- セッションセレクタで削除後にカーソルが隣接セッションに留まるように修正
- musl ホストで SEA パッケージの正しい Linux libc ターゲットを使用
- `minItems` 未設定時に必須マルチセレクトプロンプトで空の選択を送信可能に修正
- ホームセッションタイムラインがアタッチ・復元後も表示されるように修正
- CLI 終了後にターミナルでマウスホイールスクロールが壊れる問題を修正（ターミナルモードを逆順で解除）
- `/rewind` ファイル復元確認ダイアログがスクロールされたタイムラインの上に開いた際に下部が切れる問題を修正
- 完了したバックグラウンドコマンドの出力が後から利用不可になる問題を修正
- ツールワイルドカード `*` を使用するカスタムエージェントが `deferredToolLoading` オプトインスイッチを尊重するように修正
- WSL セッションで tmux カラー検出を尊重
- カスタムエージェントフロントマターで設定された MCP サーバーの `deferTools` を尊重
- 補完ピッカーが開いている間に Ctrl+Q でプロンプトをエンキュー可能に修正
- セッション名変更時にセッションタブ行ラベルが即座に更新されるように修正
- nix 提供の bash が PATH の最初にある場合にシェルセッションが正しく開始されるように修正
- marketplace.json で MCP サーバーを宣言するマーケットプレイスプラグインが OAuth で正しく認証されるように修正
- コンテンツ除外がコマンド名やファントムパスでシェルコマンドをブロックしなくなった
- 孤立サロゲートがセッション再開を壊したりプロンプトを切り詰めたりしなくなった
- Windows ホームディレクトリパスをスラッシュコマンド補完で展開
- 切り詰められたツール出力プレビューが有効な UTF-8 を維持
- Alpine システムで CLI 自動アップデータが正しい musl Linux パッケージをダウンロード
- マルチブロックレスポンスを含む最後のアシスタントターン全体をコピー
- 信頼されたサーバーモードセッションでワークスペース MCP サーバーをロード
- スタックド差分がファイルツリーと同じファイル順序を使用
- `/pr` ステータスと Web 確認が PR のリポジトリにリンクするように修正
- スナップショットのないターンへの巻き戻し時に後のファイル変更を復元
- キュー済み `!` シェルコマンドをエージェントに送信せずローカルで実行
- スケジュールプロンプトマネージャダイアログがエントリに合わせて縮小
- シンボリックリンクループに当たった場合も @-file ピッカーを表示
- キャッシュ書き込み価格を省略するモデルの価格を表示
- `copilot -r` で開始されたセッションで `/update` が再起動できるように修正
- ピッカーとダイアログがコンテンツ読み込み時にシフトやクリッピングしないように修正
- Markdown でダブルチルダのみを取り消し線としてレンダリング
- リレーセッションで `/allow-all` が動作するように修正
- コンパクトタイムライン Markdown でクリック可能な PR・イシューリンクを復元
- リポジトリスコープのプラグインがプロジェクト間でグローバル設定に漏れなくなった
- サインイン後の再開セッションで `/model` が動作するように修正
- PowerShell スクリプトブロックと補間 `$()` サブ式がコンテンツ除外拒否をトリガーしなくなった
- 終了メッセージがフレンドリー名ではなくセッション ID を常に表示
- クラウドセッション開始前にリモートサンドボックスの起動を待機
- エージェントタブで新しく生成されたセッションがグループの下部に表示
- 添付画像と PDF がソースファイル変更・削除後もセッション再開で持続
- コンテンツ除外有効時に長いストリーム結果を正しくフィルタリング
- Windows スタイルの glob パターンを検索ツールが正しく処理
- kill 自己保護がクオートされたパイプや kill で終わるパスをフラグしないように修正
- Azure CLI、PowerShell、Developer CLI の認証情報が Azure 認証で再び動作
- MCP サーバー認証で静的 OAuth クライアントオーバーライド（クライアントシークレットを含む）をサポート
- CLI 読み込み中に入力されたキーストロークを保持
