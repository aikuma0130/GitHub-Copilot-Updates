# GitHub Copilot CLI v1.0.67 - 2026-06-30 アップデート情報

## リリース日

2026-06-30

## GitHub Copilot CLI バージョン

1.0.67

## 概要

サンドボックスの即時無効化、Claude Sonnet 5 モデルのサポート追加、サブエージェントのツール制限継承など、セキュリティとモデル対応を強化するアップデートです。

## 変更点

### 新機能

- Claude Sonnet 5 をサポートモデルとして追加
- プロンプトモードの終了サマリーにセッン継続のための resume ヒントを表示

### 改善

- セッション中のサンドボックス無効化が即座に反映され、シェルおよび検索コマンドがターン中にバイパスの再確認を行わなくなった
- サブエージェントセッションが親のツール制限を継承
- ホストカスタムエージェントの読み込みに失敗した場合に警告とエラーを表示
- セッション制限を最低30 AIクレジットに要求
- フックがタイムアウトした際もツール呼び出しを継続可能に
- Ctrl+Q がハイライトされたスラッシュコマンド引数補完をエンキュー

### バグ修正

- Microsoft Entra サーバー（テナントバニティドメイン、例：Copilot Studio）に対する MCP OAuth のリフレッシュまたは再認証が失敗しなくなった（AADSTS9010010 / AADSTS90023）
