# GitHub Copilot CLI v1.0.68 - 2026-07-01 アップデート情報

## リリース日

2026-07-01

## GitHub Copilot CLI バージョン

1.0.68

## 概要

kimi-k2.7-code モデルのサポート追加、サンドボックスポリシーの強化、IDE 一時切断時のツール可用性維持など、モデル対応とアクセシビリティを改善するアップデートです。

## 変更点

### 新機能

- kimi-k2.7-code モデルのサポートを追加
- エージェント画面からセッションの閲覧、再開、切り替えが可能に
- プランバジェットの詳細をステータスラインおよび /usage で表示（対応プラン向け）

### 改善

- /mcp 設定フォームでフォーカス中のフィールドに「❯」シェブロンを表示し、色だけに依存しないアクセシビリティ対応
- IDE の一時切断中もツールを利用可能に保ち、切断中は明確なエラーを返し、再接続時に自動復帰
- タブ補完でスラッシュコマンドのエイリアスをインライン表示（例：/pr automerge|agentmerge）
- スラッシュコマンド入力選択肢に説明を表示
- サンドボックスファイルシステムポリシー内でファイル編集とパッチを維持
- デバイス管理設定がサーバー管理設定をオーバーライド可能に
- 埋め込み /skill トークンをスキルプロンプトコンテキストに維持
- サンドボックス内のリポジトリサブディレクトリから git コマンドを実行可能に
- ワーキングディレクトリがターン間で変更された際にエージェントに通知し、新しいディレクトリを使用
- 対話セッションでデフォルトのカーソルを安定した縦棒に変更
- 起動時にカーソル形状を変えずにカーソル点滅を無効化
- カスタムエージェントがネストされたサブエージェントでツールフィルターを維持
- シンボリックリンクされたサンドボックスパスを1行に折りたたみ
- コードレビューが変更収集時の一時的な git 失敗をリトライ
- 不正なプラグインマニフェストをスキップし、有効なプラグインの読み込みを継続
- OSC 8 ハイパーリンク ID のラッピングを保持

### バグ修正

- セッションのワーキングディレクトリまたは git ワークツリーが削除された際にフックがエラーを起こし全ツールを拒否する問題を修正
- フッターのモデルステータスが推論努力度やコンテキストティアの変更と同期しない問題を修正
- シンボリックリンクされたスキャンルートからの重複スキル・コマンドパースエラーを回避
- /cd および /worktree 後にセッションサイドバーのブランチを更新
- Windows の Win32 拡張長パス配下のエージェントと指示ファイルを検出可能に
- macOS でタイムラインテキストをコピーする際のターミナル破損を防止
- タイ語およびデーヴァナーガリー文字のクリップ出力での正しいレンダリング
- PowerShell 変数参照がコンテンツポリシー拒否をトリガーしなくなった
- 他のリストのシンボリックリンクと競合するサンドボックスパス編集を拒否
- スラッシュコマンドピッカー表示時にステータスラインを表示し続ける
- /diagnose セッションログ抜粋でマルチバイト文字（絵文字、アクセント、非ラテン文字）を保持
