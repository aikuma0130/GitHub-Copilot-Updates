# GitHub Copilot CLI v1.0.70 - 2026-07-09 アップデート情報

## リリース日

2026-07-09

## GitHub Copilot CLI バージョン

1.0.70

## 概要

GPT-5.6 モデルサポートの追加、サンドボックスのセッション単位制御フラグ、/refine コマンドによるプロンプト書き直し機能、リポジトリ単位でのモデル・努力度・コンテキスト層のピン留め機能など、多数の新機能と改善を含むアップデートです。

## 変更点

### 新機能

- GPT-5.6 モデルサポートを追加
- --sandbox および --no-sandbox フラグを追加し、保存済み設定を変更せずに現在のセッションのみで OS レベルシェルサンドボックスのオン/オフを切り替え可能に（-p と組み合わせて使用可能）
- /refine コマンドを追加し、ラフなストリームオブコンシャスネスのプロンプトを明確なものに書き直し可能に
- /settings および /model に --repo と --local フラグを追加
- タイムラインタイムスタンプの表示/非表示設定を追加
- 信頼されたリポジトリが .github/copilot/settings.json でモデル、努力度、コンテキスト層のピン留めおよび URL/MCP/skill 拒否リストの拡張が可能に
- MCP サーバーの SDK API を公開し、実行中セッションでライブ管理が可能に
- MCP サーバーリソースのページネーション付き session.mcp.resources read/list/listTemplates RPC を追加
- preToolUse フックが終了コード 2 でツールコールを拒否可能に
- Forge が明確なワークフローパターンを発見した際にドラフトスキルを作成
- プラグインをプラグインソース設定の `sha` フィールドで正確なコミット SHA にピン留め可能に
- /model でコンテキストウィンドウを Tab キーで切り替え可能に
- 長時間実行セッションでエンタープライズ管理設定を毎時リフレッシュ
- ローカルで起動した MCP サーバーが `/mcp list` でサンドボックス内実行を表示（例: `connected (sandboxed)`）

### 改善

- MCP およびスキルコマンドの失敗時に単一の Error プレフィックスを表示
- --agent でフォーマット不正のカスタムエージェントを選択した際に実際のパースエラーを表示
- web_fetch が必須 HTTPS プロキシを経由して動作
- Gists タブで / 検索を非表示に
- 超過されたサブエージェント実行を失敗ではなくキャンセルとして処理
- リモートターミナルで GitHub App インストールのナッジを非表示に
- /user switch 後にアクティブユーザーのモデルを表示
- 拡張機能の権限プロンプトを拒否してもセッション残りのツール承認が無効にならないよう改善
- ブロッキング read_agent が結果を返した後の冗長なバックグラウンドエージェント通知を回避
- 起動時の認証エラーが実際の `copilot login` コマンドを推奨するよう改善
- /settings でマージセマンティクス設定を編集可能に維持
- マネージドプラグインのキャッシュが見つからないか空の場合に再同期
- コマンドエコー後もラストアシスタントレスポンスをコピー可能に
- ログイン時にラストログインユーザーを永続化し、再起動したランタイムクライアントが認証を維持
- 選択対象がない場合に /agent ピッカーのナビゲーションヒントを非表示に
- 任意のモードで Ctrl+Y でプランファイルやリサーチレポートを開く
- SSH およびリモートシェルでターミナルカラースキーム変更を同期維持
- /chronicle 検索をプリフィルしてクエリを受け付け可能に
- ノーカラーパス（--no-color、非カラーターミナル）で /model ピッカーに別のスクロールバーサムグリフを表示
- リモートターミナルでブラウザ起動をスキップ
- GPT-5.6 のツール駆動型進捗更新に対するコメンタリーガイダンスを改善
- サイドバートグルヒントを入力フッターでハイライト
- タイムラインとツール出力内の Markdown リンクおよびベア URL をクリック可能に
- ホームタブバー下の空行を回収し、プロンプトピン留め時にタイムラインをタブ直下にフラッシュ配置
- プロンプトピン留め時のみ1行の余白を維持

### バグ修正

- /search および逆検索の矢印キーがタブ切り替えではなく検索内に留まるよう修正
- ツールイベントの順序を復元し、権限プロンプトがツール開始後に表示されるよう修正
- ストリーミング中断時にキャンセルメッセージを1つだけ表示するよう修正
- コンパクトタイムラインビューで /pr テーブルの整列を維持
- 空または非 ASCII のスキル名・コマンド名に対する明確なバリデーションエラーを表示
- セッションバー表示時のフッター選択ハイライトの整列を維持
- マーケットプレイスプラグイン git 認証にターミナルプロンプトが必要な場合にフェイルファスト
- サンドボックス無効化後に他の保留中の読み取り・フェッチバイパスプロンプトを閉じる
- Windows でデスクトップトースト通知がトリガーするクラッシュを修正
