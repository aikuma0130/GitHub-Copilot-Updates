# GitHub Copilot CLI v1.0.73 - 2026-07-20 アップデート情報

## 概要

カスタムエージェントの安定性向上に関するリリース。Anthropicサブエージェントの追加ディレクトリ設定時の動作修正と、カスタムエージェント指示ファイル内の相対リンク解決が改善されました。

## 変更点

### バグ修正
- Anthropicサブエージェントが追加ディレクトリが設定されている場合でも正常に動作するよう修正
- カスタムエージェント指示ファイル内の相対リンクが、エージェントファイルの場所を基準に正しく解決されるよう修正
