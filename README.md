# GitHub Copilot アップデート情報

GitHub Copilot や GitHub Copilot CLI、VS Code の GitHub Copilot に関連する最新アップデートを追跡するリサーチリポジトリです。

## 概要

このリポジトリでは、[GitHub Changelog](https://github.blog/changelog/) や公式ブログで発表される GitHub Copilot 関連のアップデート、GitHub Copilot CLI のリリースノート、VS Code の GitHub Copilot 拡張機能のアップデートを日本語で記録しています。各アップデートにはカテゴリ分類、概要説明、公式リンクが含まれています。

## リポジトリ構成

```
├── Updates/                        # GitHub Copilot アップデート情報
│   ├── YYYYMM/                     # 月別ディレクトリ
│   │   ├── YYYYMMDD-updates.md     # 月別サマリー
│   │   ├── YYYYMMDD-topic-name.md  # 個別アップデートの詳細記事
│   │   └── ...
│   └── ...
├── GitHubCopilotCLI/               # GitHub Copilot CLI アップデート情報
│   ├── YYYYMMDD-vX.Y.Z.md          # バージョン別ディレクトリ(リリース日とバージョン番号)
│   └── ...
└── VSCode/                         # VS Code の GitHub Copilot 関連機能アップデート情報
    ├── YYYYMMDD-vX.Y.Z.md          # バージョン別ディレクトリ(リリース日とバージョン番号)
    └── ...


```
