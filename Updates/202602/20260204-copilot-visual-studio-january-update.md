# GitHub Copilot in Visual Studio — 1月アップデート

## 日付

2026-02-04

## ステータス

Release

## 概要

Visual Studio の1月アップデートで、カラー付きコード補完、部分的な提案受け入れなどの生産性向上機能を追加。

## 詳細

Visual Studio の1月アップデートでは、以下の改善が含まれています：

- **カラー付きコード補完**: シンタックスハイライトにより読みやすさが向上
- **部分的な提案受け入れ**: クリックで提案の一部のみを受け入れ可能
- **高速スクロール**: スクロール操作の改善
- **HTML リッチコピー＆ペースト**: HTML コンテンツの改善されたコピー＆ペースト体験
