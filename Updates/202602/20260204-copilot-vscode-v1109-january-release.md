# GitHub Copilot in VS Code v1.109 — 1月リリース

## 日付

2026-02-04

## ステータス

Release

## 概要

VS Code v1.109 の1月リリースで、エージェント拡張性、セッション管理の改善、ターミナルコマンドサンドボックスなどを追加。

## 詳細

VS Code v1.109（1月リリース）では、以下の改善が含まれています：

- **エージェント拡張性**: Claude などの AI モデル、ツール、カスタムワークフローの統合をサポート
- **セッション管理の改善**: Copilot、Claude、ローカル/バックグラウンド/クラウドエージェント間の作業委任が容易に
- **エージェントスキルとカスタマイズ**: 組織/リポジトリ全体のエージェントオーケストレーション
- **ターミナルコマンドサンドボックス**: エージェントが提案するターミナルコマンドのより安全な実行
- **ターミナル体験の改善**: よりスムーズな実行とマルチタスク処理
