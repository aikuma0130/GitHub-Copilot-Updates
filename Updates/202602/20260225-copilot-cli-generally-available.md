# GitHub Copilot CLI が一般提供開始

## 日付

2026-02-25

## ステータス

GA

## 概要

ターミナルネイティブのコーディングエージェント Copilot CLI が GA。プランモード、自律ワークフロー、バックグラウンド委任、マルチモデル選択に対応。

## 詳細

GitHub Copilot CLI が一般提供開始されました。ターミナルネイティブのコーディングエージェントとして、以下の機能を提供します：

- **プランモード**: 構造化された実装計画の作成
- **オートパイロットモード**: 完全な自律ワークフロー
- **バックグラウンド委任**: シンプルなプロンプトでクラウドに作業を委任
- **モデル選択**: Anthropic、OpenAI、Google のモデルを動的に選択
- **拡張性**: カスタムサーバー、プラグイン、スキルのサポート
