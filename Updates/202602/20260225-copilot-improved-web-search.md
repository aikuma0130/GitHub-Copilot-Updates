# github.com の Copilot で Web 検索が改善

## 日付

2026-02-25

## ステータス

Public Preview

## 概要

github.com の Copilot Chat でモデルネイティブ Web 検索が有効化。時事情報や時間に敏感な質問により高速・正確に回答。

## 詳細

github.com の Copilot Chat で一部のモデルにモデルネイティブ Web 検索が有効化されました。GPT-5.1、GPT-5.1-Codex、GPT-5.1-Codex-Mini、GPT-5.1-Codex-Max、GPT-5.2-Codex で利用可能。他のモデルは引き続き Bing 検索を使用します。Copilot 設定からオプトアウト可能。
