# Copilot 使用メトリクスにエンタープライズレベルの CLI アクティビティが追加

## 日付

2026-02-27

## ステータス

Release

## 概要

エンタープライズ使用メトリクスに Copilot CLI テレメトリが追加。日次アクティブ CLI ユーザー、セッション数、トークン使用量を追跡可能。

## 詳細

エンタープライズ使用メトリクスに Copilot CLI テレメトリが追加されました。以下のデータを追跡可能：

- 日次アクティブ CLI ユーザー数
- CLI リクエストおよびセッション数
- 平均リクエストあたりトークン数を含むトークン使用量合計

エンタープライズ全体での CLI 導入状況の追跡や、使用パターンの比較、トークン使用量の消費傾向の把握に活用できます。
