# Copilot メトリクスが一般提供開始

## 日付

2026-02-27

## ステータス

GA

## 概要

Copilot 使用メトリクスが GA。ダッシュボードと API でエンタープライズ/組織/ユーザーレベルの使用状況を追跡可能。

## 詳細

GitHub Copilot 使用メトリクスが一般提供開始されました。チームの Copilot 導入・使用状況を一元的に確認できます。

**含まれる機能：**
- Copilot 使用ダッシュボード（コード補完、IDE 使用、モデル/言語内訳）
- コード生成ダッシュボード（提案/追加/削除行数の定量化）
- エンタープライズ/組織/ユーザーレベルの API
- 細粒度アクセスコントロール
- データレジデンシーサポート
