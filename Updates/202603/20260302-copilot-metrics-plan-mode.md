# Copilot メトリクスにプランモードが追加

## 日付

2026-03-02

## ステータス

Release

## 概要

Copilot 使用メトリクスにプランモードのテレメトリが追加。エンタープライズでプランモードの導入状況を IDE メトリクスと並行して追跡可能に。

## 詳細

Copilot 使用メトリクスにプランモードのテレメトリが追加されました。エンタープライズはプランモードの導入状況やエンゲージメントのトレンドを、既存の Copilot メトリクスと並行して追跡できます。API レスポンスでは `chat_panel_plan_mode` として表示されます。JetBrains、Eclipse、Xcode、VS Code Insiders でサポートされています。
