# Copilot Memory が Pro/Pro+ ユーザーでデフォルト有効に（パブリックプレビュー）

## 日付

2026-03-04

## ステータス

Public Preview

## 概要

Copilot Memory が Pro/Pro+ ユーザーでデフォルト有効に。リポジトリレベルの理解を保持し、Coding Agent、Code Review、CLI 間で知識を共有。

## 詳細

Copilot Memory が GitHub Copilot Pro および Pro+ ユーザーでデフォルト有効になりました。Copilot Memory は、リポジトリレベルのコーディング規約やアーキテクチャパターンなどの知識を保持・再利用します。メモリは28日後に自動期限切れし、現在のコードベースに対して検証されてから適用されます。

Copilot Memory は以下で動作します：
- **Copilot Coding Agent**: タスク実装時にリポジトリ固有のコンテキストを適用
- **Copilot Code Review**: 学習したパターンを基にレビューフィードバックを提供
- **Copilot CLI**: ターミナルワークフローにリポジトリの認識を提供
