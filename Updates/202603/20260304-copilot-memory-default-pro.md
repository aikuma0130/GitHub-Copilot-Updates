# Copilot Memory が Pro/Pro+ ユーザーでデフォルト有効に（パブリックプレビュー）

## 日付

2026-03-04

## ステータス

Public Preview

## 概要

Copilot Memory 機能が Pro および Pro+ ユーザーでパブリックプレビューとしてデフォルト有効になりました。会話のコンテキストがセッション間で保持されます。

## 詳細

Copilot Memory がパブリックプレビューとして Pro および Pro+ ユーザーでデフォルト有効になりました。

この機能により、Copilot が過去の会話やプロジェクトの好みなどを記憶し、よりパーソナライズされた支援を提供します。
