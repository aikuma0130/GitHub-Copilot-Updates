# GitHub Copilot in Visual Studio — 2月アップデート

## 日付

2026-03-04

## ステータス

Release

## 概要

Visual Studio 2026 の2月アップデートで、エージェント機能の拡張、コード補完の改善、Copilot Memory、.NETテスト生成などの新機能を追加。

## 詳細

Visual Studio 2026 の2月アップデートでは、以下の主要な改善が含まれています：

- **クリックでコード補完を部分的に受け入れ**: 提案の必要な部分だけをクリックで受け入れ可能
- **カラー付きコード補完**: シンタックスハイライト付きで提案を表示
- **Copilot Memory**: コーディング規約や好みを学習し、セッション間で再利用
- **カスタムプロンプト（スラッシュコマンド）**: `/generateInstructions` や `/savePrompt` で効率化
- **スマート .NET テスト生成**: `@Test` で xUnit、NUnit、MSTest に対応
- **WinForms エキスパートエージェント**: WinForms 開発に特化したガイダンス
- **C++ ビルドパフォーマンスエージェント**: `@BuildPerfCpp` でビルド最適化を提案
- **コールスタック分析**: デバッグ中に Copilot でコールスタックを即座に分析
