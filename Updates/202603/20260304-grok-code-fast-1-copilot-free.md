# Grok Code Fast 1 が Copilot Free の自動モデル選択で利用可能に

## 日付

2026-03-04

## ステータス

GA

## 概要

Grok Code Fast 1 が Copilot Free プランの自動モデル選択で利用可能になりました。

## 詳細

Grok Code Fast 1 が Copilot Free プランの自動モデル選択に追加されました。

自動モデル選択を有効にしている Copilot Free ユーザーは、タスクに応じて Grok Code Fast 1 が自動的に選択される場合があります。
