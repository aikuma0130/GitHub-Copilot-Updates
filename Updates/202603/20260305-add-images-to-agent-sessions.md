# エージェントセッションに画像を追加可能に

## 日付

2026-03-05

## ステータス

Improvement

## 概要

github.com のエージェントセッションに画像の貼り付け・ドラッグ＆ドロップが可能に。画像からタスクを開始できるように。

## 詳細

github.com 上のエージェントセッション（リポジトリの Agents タブなど）で、画像を貼り付け、ドラッグ、またはアイコンクリックで追加できるようになりました。画像からタスクを開始し、Copilot Coding Agent に視覚的なコンテキストを提供できます。
