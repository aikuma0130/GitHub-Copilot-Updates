# 新しいセッションフィルターでエージェントアクティビティを検出・管理

## 日付

2026-03-05

## ステータス

Improvement

## 概要

Enterprise AI Controls にステータス、リポジトリ、ユーザーによるセッションフィルターが追加。エージェントアクティビティの管理が容易に。

## 詳細

GitHub Enterprise AI Controls のエージェントコントロールプレーンに、以下の新しいセッションフィルターが追加されました：

- **ステータス**: キューイング中、進行中、完了、失敗、ユーザー待ち、タイムアウト、キャンセルなど
- **リポジトリ**: エージェントセッションが発生したリポジトリで検索
- **ユーザー**: セッションを開始したユーザーで検索
