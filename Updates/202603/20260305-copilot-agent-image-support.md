# エージェントセッションに画像を追加可能に

## 日付

2026-03-05

## ステータス

GA

## 概要

Copilot coding agent のセッションに画像を添付できるようになり、UI のスクリーンショットやデザインモックアップを参照しながら作業を依頼できます。

## 詳細

Copilot coding agent のセッションに画像を追加できるようになりました。

UI のスクリーンショット、デザインモックアップ、エラー画面などの画像を添付して、より具体的なコンテキストをエージェントに提供できます。
