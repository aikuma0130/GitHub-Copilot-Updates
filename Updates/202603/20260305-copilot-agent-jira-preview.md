# GitHub Copilot coding agent for Jira がパブリックプレビューに

## 日付

2026-03-05

## ステータス

Public Preview

## 概要

GitHub Copilot coding agent for Jira がパブリックプレビューとして公開されました。Jira チケットから直接 Copilot にコーディングタスクを依頼できます。

## 詳細

GitHub Copilot coding agent for Jira がパブリックプレビューとして利用可能になりました。

Jira チケットから直接 `@GitHub Copilot` をメンションすることで、コーディングタスクを Copilot coding agent に依頼できます。エージェントは自動的にプルリクエストを作成します。
