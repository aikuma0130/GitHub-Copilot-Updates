# Copilot Code Review がエージェンティックアーキテクチャで動作開始（GA）

## 日付

2026-03-05

## ステータス

GA

## 概要

Copilot Code Review がエージェンティックなツールコーリングアーキテクチャで一般提供開始。リポジトリ全体のコンテキストを考慮した高品質なレビューが可能に。

## 詳細

Copilot Code Review がエージェンティックなツールコーリングアーキテクチャに移行し、一般提供（GA）されました。関連するコード、ディレクトリ構造、参照などのリポジトリコンテキストを必要に応じて収集し、変更がより大きなアーキテクチャにどのようにフィットするかを反映したフィードバックを提供します。

主な改善点：
- 正確性とアーキテクチャの整合性を優先した高品質な指摘
- ノイズの低減
- より実用的なガイダンス

GitHub Actions 上で動作し、セルフホストランナーを使用している場合は一回限りのセットアップが必要です。
