# GitHub Copilot Coding Agent for Jira がパブリックプレビューで提供開始

## 日付

2026-03-05

## ステータス

Public Preview

## 概要

Jira の課題を Copilot Coding Agent にアサインし、AI によるドラフト PR の自動生成が可能に。Jira Cloud + Rovo 環境で利用可能。

## 詳細

Jira の課題を GitHub Copilot Coding Agent にアサインできるようになりました。アサインされると、Copilot は Jira の課題説明とコメントを分析し、独立して変更を実装してドラフト PR を作成します。Jira でステータスの更新や質問も行います。

要件：
- Jira Cloud（Rovo 有効化済み）
- Copilot Coding Agent が有効
- GitHub リポジトリとの接続

Atlassian Marketplace からアプリをインストールして開始できます。
