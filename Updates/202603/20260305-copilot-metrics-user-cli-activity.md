# Copilot 利用メトリクスにユーザーレベルの CLI アクティビティを追加

## 日付

2026-03-05

## ステータス

GA

## 概要

Copilot 利用メトリクスにユーザーレベルの GitHub Copilot CLI アクティビティデータが追加されました。

## 詳細

Copilot 利用メトリクスにユーザーレベルの GitHub Copilot CLI アクティビティが含まれるようになりました。

管理者は個々のユーザーの CLI ツール利用状況を把握し、採用率の分析が可能になります。
