# Copilot 使用メトリクスにユーザーレベルの CLI アクティビティが追加

## 日付

2026-03-05

## ステータス

Release

## 概要

エンタープライズレベルに続き、ユーザーレベルの Copilot CLI テレメトリがメトリクスに追加。個々のユーザーの CLI 利用状況を把握可能に。

## 詳細

エンタープライズレベルの CLI テレメトリに続き、ユーザーレベルでも Copilot CLI アクティビティがメトリクスに追加されました。これにより、どのユーザーがアクティブな CLI ユーザーかを特定し、CLI リクエスト数やセッション数、トークン使用量を確認できるようになりました。
