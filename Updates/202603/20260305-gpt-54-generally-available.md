# GPT-5.4 が GitHub Copilot で一般提供開始

## 日付

2026-03-05

## ステータス

GA

## 概要

OpenAI の最新エージェンティックコーディングモデル GPT-5.4 が Copilot で GA。マルチステップの論理的推論とタスク実行能力が大幅に向上。

## 詳細

OpenAI の最新エージェンティックコーディングモデル GPT-5.4 が GitHub Copilot で一般提供開始されました。実世界のエージェンティックなソフトウェア開発タスクで高い成功率を記録し、複雑なマルチステップ・ツール依存プロセスでの論理的推論とタスク実行能力が向上しています。

Copilot Pro、Pro+、Business、Enterprise ユーザーが利用可能で、VS Code、Visual Studio、JetBrains、Xcode、Eclipse、github.com、GitHub Mobile、GitHub CLI、Copilot Coding Agent のモデルピッカーから選択できます。
