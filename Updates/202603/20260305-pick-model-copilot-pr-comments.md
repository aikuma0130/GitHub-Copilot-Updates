# PR コメントで @copilot に使用するモデルを選択可能に

## 日付

2026-03-05

## ステータス

Improvement

## 概要

PR コメントで `@copilot` をメンションした際にモデルピッカーが表示され、使用する AI モデルを選択可能に。

## 詳細

PR コメントで `@copilot` をメンションすると、コメントボックスの左下にモデルピッカーが表示されるようになりました。Copilot Coding Agent がサポートするモデルから選択できます。現時点では github.com の PR コメントのみ対応。GitHub Mobile やレビューコメントへの対応は近日予定。
