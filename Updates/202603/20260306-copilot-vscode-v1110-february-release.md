# GitHub Copilot in VS Code v1.110 — 2月リリース

## 日付

2026-03-06

## ステータス

Release

## 概要

VS Code v1.110 の2月リリースで、エージェントフック、会話の分岐、自動承認、エージェントプラグインなど長時間タスク向けの改善を多数追加。

## 詳細

VS Code v1.110（2月リリース）では、エージェントをより複雑で長時間のタスクに対応させる改善が多数追加されました：

**エージェントのプログラミング：**
- フック: エージェントライフサイクルイベントでカスタムコードを実行
- 会話の分岐: チェックポイントから別のパスを探索
- `/autoApprove` / `/yolo`: チャットからグローバル自動承認を切り替え
- キューイング: エージェント動作中にフォローアップメッセージを送信

**エージェントの拡張：**
- エージェントプラグイン: スキル、ツール、フック、MCP サーバーのパッケージをインストール
- ブラウザツール: エージェントが統合ブラウザを操作
- `/create-*` コマンド: 会話からプロンプトやスキルを生成

**コンテキスト管理：**
- エージェントメモリ共有: Coding Agent、CLI、Code Review 間で知識を共有
- `/compact` コマンド: 手動でコンテキスト圧縮
- Explore サブエージェント: 軽量モデルでコードベースリサーチを並列実行
