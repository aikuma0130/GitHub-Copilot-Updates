# Figma MCP サーバーが VS Code からデザインレイヤーを生成可能に

## 日付

2026-03-06

## ステータス

Release

## 概要

Figma MCP サーバーを使って、VS Code から Figma へ UI を送信し、編集可能なフレームとして生成可能に。デザインとコードの双方向ワークフローを実現。

## 詳細

GitHub Copilot ユーザーが Figma MCP サーバーに接続して、デザインコンテキストをコードに取り込むだけでなく、レンダリングされた UI を Figma に編集可能なフレームとして送信できるようになりました。デザインからコードを生成し、UI を Figma に送って反復し、更新をコードベースに戻すという双方向ワークフローをサポートします。VS Code で利用可能で、Copilot CLI への対応も近日予定。
