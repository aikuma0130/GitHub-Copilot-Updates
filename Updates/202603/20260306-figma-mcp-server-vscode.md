# Figma MCP サーバーが VS Code からデザインレイヤーを生成可能に

## 日付

2026-03-06

## ステータス

GA

## 概要

Figma MCP サーバーにより、VS Code から直接デザインレイヤーを生成できるようになりました。デザインとコードのワークフローが統合されます。

## 詳細

Figma MCP サーバーが VS Code からデザインレイヤーを生成できるようになりました。

これにより、Copilot のエージェントモードで Figma のデザインデータにアクセスし、デザインに基づいたコード生成やUI実装を効率化できます。
