# GitHub CLI から Copilot コードレビューをリクエスト可能に

## 日付

2026-03-11

## ステータス

GA

## 概要

GitHub CLI から Copilot コードレビューをリクエストできるようになり、ターミナルベースのワークフローからレビューを依頼可能になりました。

## 詳細

GitHub CLI から直接 Copilot コードレビューをリクエストできるようになりました。

ターミナルベースの開発ワークフローを好む開発者が、ブラウザを開かずにコードレビューを依頼できます。
