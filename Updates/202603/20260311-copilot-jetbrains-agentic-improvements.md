# JetBrains IDEs で GitHub Copilot のエージェンティック機能が大幅改善

## 日付

2026-03-11

## ステータス

GA

## 概要

JetBrains IDEs でカスタムエージェント、サブエージェント、プランエージェントが GA。エージェントフック（プレビュー）、MCP 自動承認、AGENTS.md サポートも追加。

## 詳細

JetBrains IDEs の GitHub Copilot で以下の改善が行われました：

**GA（一般提供）：**
- カスタムエージェント、サブエージェント、プランエージェント
- Auto モデル選択

**パブリックプレビュー：**
- エージェントフック: `userPromptSubmitted`、`preToolUse`、`postToolUse`、`errorOccurred`

**新機能：**
- MCP 自動承認（サーバー/ツールレベル）
- `AGENTS.md` / `CLAUDE.md` インストラクションファイルのサポート
- `/memory` スラッシュコマンド
- 拡張推論モデル用シンキングパネル
- コンテキストウィンドウ使用量インジケーター
- 編集モードの廃止予告
