# Web 上の Copilot でリポジトリのファイル構造を探索可能に

## 日付

2026-03-11

## ステータス

Public Preview

## 概要

github.com の Copilot Chat でリポジトリのファイルツリーをブラウズし、ファイルを参照として自動追加可能に（パブリックプレビュー）。

## 詳細

github.com 上の Copilot Chat でリポジトリのファイル構造をより簡単に探索できるようになりました。Copilot にファイルを表示するよう依頼すると、ファイルツリーとファイル内容を並べて閲覧できます。ファイルエクスプローラでナビゲートすると、選択したファイルが現在のチャットに一時参照として自動追加されます。パブリックプレビューとして提供中。
