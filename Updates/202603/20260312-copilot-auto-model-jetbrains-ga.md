# Copilot Auto モデル選択が JetBrains IDEs で GA

## 日付

2026-03-12

## ステータス

GA

## 概要

JetBrains IDEs で Auto モデル選択が GA。リアルタイムのモデル可用性とパフォーマンスに基づいて最適なモデルを自動選択。有料プラン10%割引付き。

## 詳細

GitHub Copilot Auto モデル選択が JetBrains IDEs で全 Copilot プラン向けに一般提供開始されました。Auto はリアルタイムのモデル可用性とパフォーマンスに基づき、GPT-5.4、GPT-5.3-Codex、Sonnet 4.6、Haiku 4.5 などのモデルにルーティングします。

- ホバーで使用モデルを確認可能
- Auto と手動選択をいつでも切り替え可能
- 管理者のモデル設定を尊重
- 有料プランで Auto 使用時にモデル乗数10%割引
