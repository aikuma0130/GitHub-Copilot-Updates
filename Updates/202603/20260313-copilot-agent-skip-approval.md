# Copilot coding agent の Actions ワークフロー承認をスキップ可能に

## 日付

2026-03-13

## ステータス

GA

## 概要

Copilot coding agent が実行する GitHub Actions ワークフローの承認をオプションでスキップできるようになり、自動化されたワークフローがよりスムーズになりました。

## 詳細

Copilot coding agent が実行する GitHub Actions ワークフローの承認プロセスをオプションでスキップできる設定が追加されました。

これにより、信頼できるリポジトリでのエージェントの自動化ワークフローがよりスムーズに実行されます。
