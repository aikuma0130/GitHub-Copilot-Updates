# GitHub Copilot for Students の更新

## 日付

2026-03-13

## ステータス

Release

## 概要

GitHub Education 特典を持つ学生が新しい Copilot Student プランに移行。利用可能なモデルラインナップが更新。

## 詳細

GitHub Education 特典を持つ学生が新しい GitHub Copilot Student プランに移行しました。この移行に伴い、利用可能なモデルラインナップが更新されました。新しい Student プランは、学生向けに長期的で持続可能な Copilot 体験の構築と、AI ネイティブな学習ツールへの継続的な投資を可能にします。
