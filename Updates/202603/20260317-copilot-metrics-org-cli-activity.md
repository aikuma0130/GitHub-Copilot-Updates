# Copilot 利用メトリクスに Organization レベルの CLI アクティビティを追加

## 日付

2026-03-17

## ステータス

GA

## 概要

Copilot 利用メトリクスに Organization レベルの GitHub Copilot CLI アクティビティが追加されました。

## 詳細

Copilot 利用メトリクスに Organization レベルの GitHub Copilot CLI アクティビティデータが含まれるようになりました。

管理者は、組織全体での CLI ツールの利用状況を把握し、採用率の分析が可能になります。
