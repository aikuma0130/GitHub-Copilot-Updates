# Copilot 使用メトリクスに組織レベルの CLI アクティビティが追加

## 日付

2026-03-17

## ステータス

Improvement

## 概要

エンタープライズ・ユーザーレベルに続き、組織レベルの Copilot CLI テレメトリがメトリクスに追加。日次アクティブ CLI ユーザー数等を確認可能。

## 詳細

エンタープライズレベル、ユーザーレベルに続き、組織レベルでも Copilot CLI アクティビティがメトリクスに追加されました。組織管理者は1日および28日レポートで以下を確認できます：

- 日次アクティブ CLI ユーザー数
- CLI セッションおよびリクエスト数
- 平均リクエストあたりトークン数を含むトークン使用量合計
