# GPT-5.4 mini が GitHub Copilot で一般提供開始

## 日付

2026-03-17

## ステータス

GA

## 概要

OpenAI の GPT-5.4 mini が Copilot で GA。最速のファーストトークン時間と grep スタイルツール使用時の強化された性能を提供。0.33x プレミアムリクエスト乗数。

## 詳細

OpenAI の GPT-5.4 の高速バージョンである GPT-5.4 mini が GitHub Copilot で一般提供開始されました。OpenAI の最高性能 mini モデルで、最速のファーストトークン時間を実現し、コードベース探索と grep スタイルツールでの使用に特に効果的です。

- 0.33x プレミアムリクエスト乗数（暫定価格、変更の可能性あり）
- Copilot Pro、Pro+、Business、Enterprise で利用可能
- VS Code、Visual Studio、JetBrains、Xcode、Eclipse、github.com、GitHub Mobile、GitHub CLI で選択可能
