# Copilot coding agent の検証ツールを設定可能に

## 日付

2026-03-18

## ステータス

GA

## 概要

Copilot coding agent が使用する検証ツール（テスト、リンターなど）をカスタマイズ設定できるようになりました。

## 詳細

Copilot coding agent が変更を検証する際に使用するツール（テスト、リンターなど）を設定でカスタマイズできるようになりました。

これにより、プロジェクト固有の品質基準に合わせてエージェントの検証プロセスを調整できます。
