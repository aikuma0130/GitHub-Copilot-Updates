GPT-5.3-Codex が GitHub Copilot で長期サポート（LTS）モデルとして提供されるようになりました。

LTS モデルは安定性を重視する開発ワークフローに適しており、一定期間のサポートが保証されます。頻繁なモデル変更を避けたいエンタープライズ環境での利用に特に有用です。
