# GPT-5.3-Codex の長期サポート（LTS）を GitHub Copilot で提供開始

## 日付

2026-03-18

## ステータス

Release

## 概要

GPT-5.3-Codex が初の LTS モデルとして提供開始（2027年2月まで）。Business/Enterprise の新しいベースモデルとして GPT-4.1 を置き換え。

## 詳細

エンタープライズ向けの安定性ニーズに応え、GPT-5.3-Codex が初の LTS（長期サポート）モデルとして提供されます。2026年2月5日のリリースから12ヶ月間（2027年2月4日まで）Copilot Business および Enterprise で利用可能です。

**主要日程：**
- 2026年3月18日: LTS とベースモデル変更を発表
- 2026年5月17日: 全 Business/Enterprise 組織のベースモデルに
- 2027年2月4日: LTS 提供期間終了

GPT-5.3-Codex は1x プレミアムリクエスト乗数。GPT-4.1 は0x 乗数で引き続き利用可能ですが、最新モデルへの移行を推奨。
