# Copilot coding agent のコミットからセッションログへのトレース

## 日付

2026-03-20

## ステータス

GA

## 概要

Copilot coding agent が作成したコミットからセッションログにトレースできるようになり、エージェントの思考プロセスと変更理由の確認が容易になりました。

## 詳細

Copilot coding agent が作成した任意のコミットから、そのセッションログへトレースバックできるようになりました。

これにより、エージェントがどのような思考プロセスで変更を行ったのか、その理由を確認することが容易になります。デバッグやレビューの効率が向上します。
