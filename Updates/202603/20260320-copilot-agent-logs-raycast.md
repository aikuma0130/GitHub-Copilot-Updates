# Raycast で Copilot Coding Agent のログをライブモニタリング

## 日付

2026-03-20

## ステータス

Improvement

## 概要

Raycast の GitHub Copilot 拡張機能で Coding Agent のログをライブ監視可能に。GitHub を開かずにタスクの進捗を追跡。

## 詳細

Raycast（macOS/Windows 向けランチャー）の GitHub Copilot 拡張機能が強化され、Copilot Coding Agent のログをライブで監視できるようになりました。GitHub を開くことなく、Raycast から直接 Copilot の作業状況をリアルタイムで追跡できます。

使用方法：拡張機能をインストール → Raycast を開く → **View Tasks** コマンドを選択 → リストからセッションを選択
