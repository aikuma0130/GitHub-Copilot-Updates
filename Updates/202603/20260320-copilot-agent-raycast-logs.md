# Raycast で Copilot coding agent のログをリアルタイム監視

## 日付

2026-03-20

## ステータス

GA

## 概要

Raycast から Copilot coding agent のセッションログをリアルタイムで監視できるようになりました。

## 詳細

Raycast を使って Copilot coding agent のセッションログをリアルタイムで監視できるようになりました。

これにより、エージェントの作業進捗を IDE やブラウザに切り替えることなく確認でき、開発ワークフローの効率が向上します。
