# Copilot 使用メトリクスが Auto モデル選択を実際のモデルに解決

## 日付

2026-03-20

## ステータス

Improvement

## 概要

Auto モデル選択時の使用が「Auto」ラベルではなく実際のモデル名で表示されるように。管理者はモデル別使用状況を正確に把握可能に。

## 詳細

Copilot 使用メトリクスで、Auto モデル選択が有効な場合のモデル使用状況が完全に明確になりました。以前は「Auto」というラベルで表示されていた使用状況が、実際のモデル名に解決されるようになり、管理者はチームが使用しているモデルの完全かつ正確なビューを得られます。

REST API と ダッシュボードの両方に適用され、コンプライアンスや監査ワークフローをサポートします。
