# Gemini 3.1 Pro が JetBrains IDE、Xcode、Eclipse で利用可能に

## 日付

2026-03-23

## ステータス

Public Preview

## 概要

Gemini 3.1 Pro がパブリックプレビューとして、github.com、GitHub Mobile、VS Code、Visual Studio、JetBrains IDE、Xcode、Eclipse の Copilot Chat で利用可能になりました。

## 詳細

Gemini 3.1 Pro が Copilot Enterprise、Business、Pro、Pro+ ユーザー向けにパブリックプレビューとして利用可能になりました。

github.com、GitHub Mobile、Visual Studio Code、Visual Studio、JetBrains IDE、Xcode、Eclipse の Copilot Chat（Agent、Ask、Edit モード）のモデルピッカーからアクセスできます。

Copilot Business および Enterprise の管理者は、Copilot 設定で Gemini 3.1 Pro ポリシーを有効にする必要があります。
