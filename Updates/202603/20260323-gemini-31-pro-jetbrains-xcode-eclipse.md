# Gemini 3.1 Pro が JetBrains IDEs、Xcode、Eclipse で利用可能に

## 日付

2026-03-23

## ステータス

Public Preview

## 概要

Gemini 3.1 Pro が全主要 IDE と github.com でパブリックプレビューとして利用可能に。Business/Enterprise は管理者のポリシー有効化が必要。

## 詳細

Gemini 3.1 Pro が Copilot Enterprise、Business、Pro、Pro+ ユーザー向けにパブリックプレビューで提供されました。github.com、GitHub Mobile、Visual Studio Code、Visual Studio、JetBrains IDEs、Xcode、Eclipse のチャットモデルピッカー（agent、ask、edit モード）から利用可能です。

Copilot Business/Enterprise 管理者は Copilot 設定で Gemini 3.1 Pro ポリシーを有効化する必要があります。
