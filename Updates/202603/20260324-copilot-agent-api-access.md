# Copilot coding agent のリポジトリアクセスを API で管理

## 日付

2026-03-24

## ステータス

Public Preview

## 概要

新しい Copilot coding agent 管理 REST API（パブリックプレビュー）により、Organization オーナーが CCA のリポジトリアクセスをプログラムで管理できるようになりました。

## 詳細

Organization オーナーが Copilot coding agent のアクセスを大規模に管理できる、新しい Copilot coding agent 管理 REST API がパブリックプレビューとして利用可能になりました。

この API では以下が可能です：
- Organization が所有するリポジトリに対して、CCA を無効、全リポジトリで有効、または選択したリポジトリのみで有効に設定
- 「選択したリポジトリのみ」設定時に、許可するリポジトリの追加・削除

UI からも Organization 設定 > Copilot > Coding agent から管理可能です。
