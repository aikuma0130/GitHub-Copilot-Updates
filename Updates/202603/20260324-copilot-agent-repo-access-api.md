# API で Copilot Coding Agent のリポジトリアクセスを管理可能に

## 日付

2026-03-24

## ステータス

Public Preview

## 概要

新しい REST API で Copilot Coding Agent のリポジトリアクセスをプログラマティックに管理可能に（パブリックプレビュー）。

## 詳細

組織オーナーが Copilot Coding Agent のアクセスを大規模に管理するための新しい REST API がパブリックプレビューで提供されました。

API で可能な操作：
- Copilot Coding Agent の利用対象リポジトリを「なし」「全リポジトリ」「選択したリポジトリ」から設定
- 「選択したリポジトリ」の場合に許可リポジトリの追加/削除

UI からも組織設定の **Copilot → Coding agent** で同様の管理が可能です。
