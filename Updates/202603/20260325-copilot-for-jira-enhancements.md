# GitHub Copilot for Jira — パブリックプレビューの機能強化

## 日付

2026-03-25

## ステータス

Public Preview

## 概要

GitHub Copilot coding agent for Jira のパブリックプレビューに、オンボーディング改善、モデル選択機能、Jira チケットの PR 参照、Confluence コンテキスト（MCP 経由）が追加されました。

## 詳細

GitHub Copilot coding agent for Jira のパブリックプレビューに以下の改善が追加されました。

### オンボーディングとセットアップガイダンスの改善
エラーメッセージの明確化と一般的な設定問題の解決手順の改善。ドキュメントも拡充されました。

### モデル選択
Jira から直接 Copilot coding agent が使用する AI モデルを選択できるようになりました。

### Jira チケット参照の PR への組み込み
PR タイトルとブランチ名に Jira チケット番号を含め、元の Jira チケットへのリンクも追加されます。

### Confluence コンテキスト（MCP 経由）
Atlassian MCP サーバーを設定することで、Confluence ページへのアクセスを Copilot coding agent に提供できます。
