# GitHub Copilot for Jira — パブリックプレビューの機能強化

## 日付

2026-03-25

## ステータス

Public Preview

## 概要

Jira 統合のパブリックプレビューが強化。モデル選択、Jira チケット参照の PR 自動追加、Confluence コンテキスト（MCP 経由）に対応。

## 詳細

GitHub Copilot Coding Agent for Jira のパブリックプレビューが機能強化されました：

- **オンボーディングとセットアップガイダンスの改善**: エラーメッセージの明確化とドキュメントの拡充
- **モデル選択**: Jira コメントから直接使用する AI モデルを指定可能
- **Jira チケット参照**: PR タイトルとブランチ名に Jira チケット番号を自動追加、元の Jira チケットへのリンクも含む
- **Confluence コンテキスト（MCP 経由）**: Atlassian MCP サーバーを設定して Confluence ページへのアクセスを提供可能
