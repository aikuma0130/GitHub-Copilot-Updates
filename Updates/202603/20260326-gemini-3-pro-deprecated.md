# Gemini 3 Pro の廃止

## 日付

2026-03-26

## ステータス

Deprecated

## 概要

Gemini 3 Pro が GitHub Copilot の全エクスペリエンス（Chat、インライン編集、Ask/Agent モード、コード補完）で廃止されました。代替として Gemini 3.1 Pro への移行が推奨されます。

## 詳細

Gemini 3 Pro が2026年3月26日をもって GitHub Copilot の全エクスペリエンス（Copilot Chat、インライン編集、Ask モード、Agent モード、コード補完）で廃止されました。

代替モデルとして Gemini 3.1 Pro が推奨されています。Copilot Enterprise の管理者は、Copilot 設定のモデルポリシーで代替モデルへのアクセスを有効にする必要がある場合があります。

廃止されたモデルの削除に関してユーザー側の操作は不要です。
