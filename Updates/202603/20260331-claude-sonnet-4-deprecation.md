# Claude Sonnet 4 の廃止予告

## 日付

2026-03-31

## ステータス

Deprecated（予定: 2026-05-01）

## 概要

Claude Sonnet 4 が 2026年5月1日に GitHub Copilot の全エクスペリエンス（Chat、インライン編集、Ask/Agent モード、コード補完）で廃止予定です。代替として Claude Sonnet 4.6 が推奨されています。

## 詳細

GitHub は Claude Sonnet 4 モデルを 2026年5月1日に廃止することを発表しました。対象となるのは Copilot Chat、インライン編集、Ask モードおよび Agent モード、コード補完を含むすべての GitHub Copilot エクスペリエンスです。

### 廃止スケジュール

| モデル | 廃止日 | 推奨代替モデル |
|--------|--------|--------------|
| Claude Sonnet 4 | 2026-05-01 | Claude Sonnet 4.6 |

### 必要なアクション

- 廃止日までにワークフローと統合を対応モデルに移行してください
- Copilot Enterprise 管理者は、Copilot 設定のモデルポリシーから代替モデルへのアクセスを有効にする必要がある場合があります
- 廃止済みモデルの削除に関して、ユーザー側のアクションは不要です
