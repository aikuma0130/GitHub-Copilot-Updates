# Copilot cloud agent でリサーチ、計画、コーディングが可能に

## 日付

2026-04-01

## ステータス

GA

## 概要

Copilot coding agent が「Copilot cloud agent」にリブランドされ、PR ワークフローに限定されない、より幅広い活用が可能になりました。ブランチ上での作業、実装計画の生成、コードベースの深いリサーチ機能が追加されています。

## 詳細

Copilot cloud agent（旧 Copilot coding agent）が大幅に機能拡張され、プルリクエストワークフローに限定されずに利用できるようになりました。

### PR を開くタイミングの柔軟な制御

- Copilot がプルリクエストを作成せずにブランチ上でコードを生成
- **Diff** ボタンで完全な差分をレビューしてから PR を作成するか判断可能
- PR の準備ができたら **Create pull request** をクリック
- プロンプトで最初から PR 作成を指示することも可能

### 実装計画の生成

- プロンプトで計画を依頼すると、コードを書く前に実装計画を生成
- 提案されたアプローチをレビューし、承認またはフィードバックを提供
- 承認後、計画に基づいて実装を実行

### コードベースの深いリサーチ

- リサーチセッションを開始して、徹底的な調査と包括的な回答が必要な質問に対応
- リポジトリコンテキストに基づいた回答を取得
- Copilot Chat の会話からもリサーチセッションを開始可能

### 利用条件

- リポジトリの **Agents** タブおよび Copilot Chat から利用可能
- すべての有料 Copilot プランで利用可能
- Copilot Business / Enterprise ユーザーは管理者による有効化が必要
