# GitHub Mobile で Issue からのエージェント割り当てがより高速・柔軟に

## 日付

2026-04-01

## ステータス

GA

## 概要

GitHub Mobile の Issue メニューに新しい「Assign an Agent」オプションが追加され、カスタム指示の追加や別リポジトリの選択が可能になり、エージェントへの作業委任がより高速かつ柔軟になりました。

## 詳細

GitHub Mobile での Issue からのエージェント割り当てが改善されました。

### 主な改善点

- **Assign an Agent オプション**: Issue のオーバーフローメニューに新しい「Assign an Agent」オプションが追加され、より迅速にエージェントを割り当て可能
- **カスタム指示**: エージェント割り当て時にカスタム指示を追加可能
- **リポジトリ選択**: 作業を委任する際に別のリポジトリを選択可能
- **Issue 作成時の統合**: Issue 新規作成時にも同じオプションが利用可能

### 利用方法

iOS および Android の GitHub Mobile 最新プロダクションビルドで利用可能です。
