# GitHub Mobile の Copilot タブがリニューアル、ネイティブセッションログに対応

## 日付

2026-04-01

## ステータス

GA

## 概要

GitHub Mobile で Copilot タブが刷新され、エージェントワークフローの管理がモバイルからより簡単になりました。ネイティブセッションログの閲覧、PR の作成・レビュー、実行中セッションの停止などがアプリ内で可能です。

## 詳細

GitHub Mobile に大幅なアップデートが行われ、Copilot のエージェントワークフローをモバイルからより効率的に管理できるようになりました。

### 主な変更点

- **Copilot タブの移動（Android）**: Copilot がナビゲーションバーに移動し、セッションやチャット履歴へのアクセスが高速化
- **新しいホーム画面**: エージェントセッションとチャット履歴の概要を一目で把握可能
- **セッションリストビュー**: タスク状態によるフィルタリングとリアルタイム更新に対応

### エージェントセッションの新しいアプリ内操作

- セッションログのネイティブ表示
- 完了したセッションからプルリクエストを作成
- プルリクエストの詳細レビュー
- 実行中のセッションを停止

### 利用方法

iOS および Android の GitHub Mobile 最新プロダクションビルドで利用可能です。
