# GPT-5.4 mini が Copilot Student の自動モデル選択で利用可能に

## 日付

2026-04-01

## ステータス

GA

## 概要

GPT-5.4 mini が Copilot Student プランの自動モデル選択で一般提供（GA）になりました。VS Code、Visual Studio、JetBrains IDE、Xcode、Eclipse の Copilot Chat で利用可能です。

## 詳細

GPT-5.4 mini が Copilot Student プランの Copilot 自動モデル選択を通じて一般提供されました。

### 対応プラットフォーム

このモデルは以下のプラットフォームの GitHub Copilot Chat で Auto モデル選択の一部として利用可能です：

- Visual Studio Code
- Visual Studio
- JetBrains IDE
- Xcode
- Eclipse
