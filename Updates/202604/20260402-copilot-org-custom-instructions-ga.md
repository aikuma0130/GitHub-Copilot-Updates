# Copilot Organization カスタムインストラクションが一般提供開始

## 日付

2026-04-02

## ステータス

GA

## 概要

GitHub Copilot の Organization カスタムインストラクションが一般提供（GA）になりました。Copilot Business および Enterprise の管理者が、Organization 全体で Copilot の動作をガイドするデフォルトの指示を設定できます。

## 詳細

2025年4月に導入された Organization カスタムインストラクション機能が一般提供となりました。

### 適用範囲

Organization カスタムインストラクションは以下の機能で適用されます：

- github.com 上の Copilot Chat
- Copilot コードレビュー
- Copilot cloud agent

### 設定方法

1. Organization の設定に移動
2. **Copilot** を選択
3. **Custom instructions** をクリック
4. インストラクションを記述して保存

管理者が設定した指示は、Organization 内のすべてのリポジトリで Copilot の動作に反映されます。
