# GPT-5.1 Codex、GPT-5.1-Codex-Max、GPT-5.1-Codex-Mini が非推奨に

## 日付

2026-04-03

## ステータス

Deprecated

## 概要

GPT-5.1-Codex、GPT-5.1-Codex-Max、GPT-5.1-Codex-Mini が 2026年4月1日付けですべての GitHub Copilot エクスペリエンスで非推奨となりました。代替モデルとして GPT-5.3-Codex が推奨されています。

## 詳細

以下のモデルが Copilot Chat、インライン編集、Ask/Agent モード、コード補完を含むすべての GitHub Copilot エクスペリエンスで非推奨になりました。

### 非推奨モデル一覧

| モデル | 非推奨日 | 推奨代替モデル |
|--------|----------|----------------|
| GPT-5.1-Codex | 2026-04-01 | GPT-5.3-Codex |
| GPT-5.1-Codex-Mini | 2026-04-01 | GPT-5.3-Codex |
| GPT-5.1-Codex-Max | 2026-04-01 | GPT-5.3-Codex |

### 必要なアクション

- ワークフローおよびインテグレーションをサポート対象モデルに更新すること
- Copilot Enterprise 管理者はモデルポリシー設定で代替モデルへのアクセスを有効化する必要がある場合あり
- 非推奨モデルの削除に特別な操作は不要
