# GitHub Copilot in Visual Studio Code — 3月リリースまとめ

## 日付

2026-04-08

## ステータス

Release

## 概要

VS Code の週次安定リリース（v1.111〜v1.115）をまとめた3月〜4月初旬のリリースノートが公開されました。Autopilot による完全自律エージェントセッション、統合ブラウザデバッグ、画像・動画対応などが主なハイライトです。

## 詳細

VS Code が週次安定リリースに移行し、2026年3月〜4月初旬の v1.111 から v1.115 までのリリースをカバーするまとめが公開されました。

### エージェントの実行制御

- **エージェント権限レベル**: セッションごとに「Default」「Bypass Approvals」「Autopilot」の3段階の権限レベルを選択可能。ローカルおよび Copilot CLI セッションの両方に適用
- **Autopilot（パブリックプレビュー）**: エージェントが自律的にアクションを承認し、エラー時に自動リトライ。タスク完了まで手動承認不要で動作
- **思考の深さの設定**: Claude Sonnet 4.6 や GPT-5.4 などの推論モデルの思考深度をモデルピッカーから直接設定可能。設定は会話間で保持

### エージェント機能の拡張

- **統合ブラウザデバッグ**: VS Code を離れずにブレークポイント設定、ステップ実行、変数検査が可能。新しい `editor-browser` デバッグタイプは既存の `chrome` / `msedge` 設定と互換
- **ネストされたサブエージェント**: サブエージェントが他のサブエージェントを呼び出し可能になり（`chat.subagents.allowInvocationsFromSubagents`）、複雑なマルチステップワークフローに対応
- **画像・動画対応**: チャットメッセージにスクリーンショットや動画を添付可能。エージェントは変更の画像やレコーディングを返却でき、カルーセルでプレビュー可能
- **CLI / Claude エージェント連携改善**: VS Code で設定した MCP サーバーが Copilot CLI および Claude エージェントセッションでも利用可能。セッションのフォークによる代替アプローチの探索も可能

### カスタマイズとデバッグの簡素化

- **チャットカスタマイズエディタ（プレビュー）**: インストラクション、カスタムエージェント、スキル、プラグインを一元管理する新しいインターフェース
- **高速ワークスペース検索**: `#codebase` ツールが単一の自動管理インデックスに対する純粋なセマンティック検索に移行
- **サンドボックス MCP サーバー**: ローカル MCP サーバーを制限付きサンドボックスで実行可能（macOS / Linux）
- **`/troubleshoot` コマンド**: エージェントのデバッグログをチャット内で直接分析し、予期しない動作の原因を特定可能
- **モノレポ対応カスタマイズ**: リポジトリルートまでの親フォルダからインストラクション、エージェント、スキル、フックを自動検出
- **エージェントスコープフック（プレビュー）**: `.agent.md` ファイルの YAML フロントマターでカスタムエージェント固有の前後処理ロジックを定義可能

### VS Code 生産性向上

- 新しいデフォルトテーマ「VS Code Light」「VS Code Dark」
- TypeScript 6.0 サポート
- 統合ブラウザでの自己署名証明書サポート
- コードシンボルをチャットに貼り付け時の自動シンボル参照
- **Copy Final Response** コマンド（思考ステップとツール呼び出しをスキップして最終回答のみコピー）
