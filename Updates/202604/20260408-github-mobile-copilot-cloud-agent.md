# GitHub Mobile で Copilot cloud agent がフル活用可能に

## 日付

2026-04-08

## ステータス

Improvement

## 概要

GitHub Mobile で Copilot cloud agent が PR ワークフロー以外にも対応し、コードベースのリサーチ、実装計画の生成、ブランチ上でのコード変更がモバイルデバイスから可能になりました。

## 詳細

Copilot cloud agent が GitHub Mobile でより幅広いワークフローに対応しました。従来の PR ベースのワークフローに加えて、以下の操作がモバイルデバイスから可能になります：

### 新しいワークフロー

- **コードベースのリサーチ**: Copilot にコードベースの調査を依頼し、実装前の情報収集が可能
- **実装計画の生成**: コーディング前に実装計画を生成し、レビューした上でコード変更に進められる
- **ブランチ上での作業**: PR を即座に作成せずにブランチ上でコード変更を実行可能
- **差分の確認と反復**: 変更内容の差分を確認し、必要に応じて反復・修正が可能
- **PR の作成**: 準備ができた時点で PR を作成可能

### 自動 PR 作成

プロンプトで PR の作成を指示すれば、セッション完了時に Copilot が自動的に PR を作成します。

### 入手方法

Apple App Store または Google Play Store から GitHub Mobile をダウンロード・アップデートしてください。
