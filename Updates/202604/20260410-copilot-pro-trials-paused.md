# GitHub Copilot Pro の新規トライアルを一時停止

## 日付

2026-04-10

## ステータス

Retired

## 概要

無料トライアルシステムの悪用増加に対応するため、GitHub Copilot Pro の新規トライアルが一時的に停止されました。既存のトライアルと有料サブスクリプションには影響ありません。

## 詳細

GitHub Copilot の成長に伴い、無料トライアルシステムの悪用が大幅に増加していることを受けた措置です。

### 影響範囲

- **新規ユーザー**: 現時点では Copilot Pro の無料トライアルを開始できません
- **既存トライアル**: 影響なし（継続利用可能）
- **Copilot Free**: 引き続き利用可能
- **有料 Copilot Pro サブスクリプション**: 引き続き利用可能

### 今後の対応

これは一時的な措置であり、トライアルシステムの悪用を防止する改善されたセーフガードを積極的に開発中です。保護策が整い次第、トライアルを再開する予定です。

Copilot を今すぐ利用したい開発者は、Copilot Free または有料の Copilot サブスクリプションに直接登録できます。
