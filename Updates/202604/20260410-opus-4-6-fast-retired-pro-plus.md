# Opus 4.6 Fast が Copilot Pro+ から廃止、新しい利用制限の導入

## 日付

2026-04-10

## ステータス

Retired

## 概要

Copilot Pro+ ユーザー向けの Opus 4.6 Fast モデルが廃止されました。また、サービスの安定性確保のため、全体的なサービス信頼性制限とモデル別容量制限の2種類の新しい利用制限が導入されます。

## 詳細

GitHub Copilot の急速な成長に伴い、高い並行性と集中的な利用パターンが増加しています。すべてのユーザーに高速で信頼性の高い Copilot 体験を提供するため、以下の変更が実施されます。

### 新しい利用制限

今後数週間かけて展開される2種類の制限：

- **サービス信頼性制限**: この制限に達した場合、現在のセッションがリセットされるまで待つ必要があります
- **モデル別容量制限**: 特定のモデルまたはモデルファミリーの制限に達した場合、代替モデルに切り替えるか、[Auto モード](https://docs.github.com/copilot/concepts/auto-model-selection)を使用できます

### 推奨事項

- リクエストを大量に集中して送信するのではなく、時間をかけて均等に分散させることを推奨
- より高い制限が必要な場合は[プランのアップグレード](https://github.com/features/copilot/plans)が可能

### Opus 4.6 Fast の廃止

モデル提供の効率化の一環として、Copilot Pro+ ユーザー向けの Opus 4.6 Fast が本日より廃止されます。代替モデルとして同様の機能を持つ Opus 4.6 の使用が推奨されます。
