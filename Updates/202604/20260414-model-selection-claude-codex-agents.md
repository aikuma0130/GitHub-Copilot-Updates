# Claude および Codex エージェントのモデル選択が github.com で利用可能に

## 日付

2026-04-14

## ステータス

GA

## 概要

github.com 上の Claude および Codex サードパーティコーディングエージェントでモデル選択が利用可能になりました。Copilot cloud agent と同様に、タスク開始時にモデルを選択できます。

## 詳細

github.com 上のサードパーティコーディングエージェント（Claude、Codex）でモデル選択機能が利用可能になりました。Copilot cloud agent と同様に、タスク開始時に使用するモデルを選択できるようになり、最新かつ最も高性能なモデルが利用可能になり次第アクセスできます。

### 利用可能なモデル

**Claude コーディングエージェント:**
- Claude Sonnet 4.6
- Claude Opus 4.6
- Claude Sonnet 4.5
- Claude Opus 4.5

**Codex コーディングエージェント:**
- GPT-5.2-Codex
- GPT-5.3-Codex
- GPT-5.4

### アクセス方法

Claude および Codex へのアクセスは既存の Copilot サブスクリプションに含まれています。Copilot Business または Enterprise ユーザーの場合、管理者が Anthropic Claude または OpenAI Codex の該当ポリシーを有効にする必要があります。また、リポジトリのオーナーが **Settings > Copilot > Cloud agent** からエージェントを有効にする必要があります。
