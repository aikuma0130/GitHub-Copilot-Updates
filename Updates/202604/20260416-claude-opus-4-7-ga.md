# Claude Opus 4.7 が一般提供開始

## 日付

2026-04-16

## ステータス

GA

## 概要

Anthropic の最新モデル Claude Opus 4.7 が GitHub Copilot で一般提供（GA）開始されました。マルチステップタスクの性能向上とエージェント実行の信頼性向上が図られています。

## 詳細

### 主な改善点

Claude Opus 4.7 は、前モデルのコーディング戦略の強みを基盤に、以下の改善を実現しています：

- **マルチステップタスク性能の向上**: 複数のステップにまたがるタスクでより強力なパフォーマンスを発揮
- **エージェント実行の信頼性向上**: エージェントモードでの実行がより安定的に
- **長期的推論の改善**: 長いホライズンの推論や複雑なツール依存ワークフローで大幅な改善

### モデルラインナップの整理

サービス信頼性の向上を目的として、今後数週間で Copilot Pro+ のモデルピッカーから Opus 4.5 および Opus 4.6 が段階的に Opus 4.7 に置き換えられます。

### 料金

プロモーション価格として **7.5 倍のプレミアムリクエスト乗数** が 2026 年 4 月 30 日まで適用されます。

### 利用可能なプラン

- Copilot Pro+
- Copilot Business
- Copilot Enterprise

### 対応プラットフォーム

- Visual Studio Code
- Visual Studio
- Copilot CLI
- GitHub Copilot Cloud Agent
- github.com
- GitHub Mobile（iOS / Android）
- JetBrains
- Xcode
- Eclipse

### 有効化

Copilot Enterprise および Copilot Business プランの管理者は、Copilot 設定で Claude Opus 4.7 ポリシーを有効にする必要があります。
