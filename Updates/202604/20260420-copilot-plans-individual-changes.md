# GitHub Copilot 個人プランの変更

## 日付

2026-04-20

## ステータス

GA

## 概要

GitHub Copilot の個人向けプラン（Pro、Pro+、Student）の新規サインアップが一時停止され、利用制限の強化や Pro プランからの Opus モデル削除など、サービス持続性のための大幅な変更が発表されました。

## 詳細

### Pro・Pro+・Student の新規サインアップ一時停止

既存の有料ユーザーへのサービス品質を優先するため、Student、Pro、Pro+ プランの新規サインアップが一時停止されます。Copilot Free は引き続き新規サインアップ可能で、既存ユーザーはプラン間のアップグレードが可能です。

### 個人プランの利用制限強化

Pro+ プランは Pro の 5 倍以上の利用制限を提供します。より高い制限が必要な Pro ユーザーは Pro+ へのアップグレードが可能です。VS Code および Copilot CLI で制限に近づくと警告通知が表示され、利用状況の進捗トラッキング機能も近日追加予定です。

### Pro プランから Opus モデルを削除

Opus モデルが Copilot Pro では利用できなくなりました。Opus 4.7 は Pro+ で引き続き利用可能です。以前発表されたとおり、Opus 4.5 および 4.6 も Pro+ から削除される予定です。

### 既存ユーザーへの対応

これらの変更に不満がある場合、Pro または Pro+ のサブスクリプションをキャンセルすると4月分の利用料は請求されません。2026年4月20日から5月20日の間に GitHub サポートに連絡すれば返金が可能です。
