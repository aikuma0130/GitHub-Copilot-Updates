# Issue とプロジェクトからエージェントセッションを表示・管理

## 日付

2026-04-23

## ステータス

GA

## 概要

Issue とプロジェクトから直接 Copilot cloud agent のセッションを表示・操作できるようになり、ワークフローを離れることなくエージェントのアクティビティを把握できます。

## 詳細

### 新機能

#### Issue でのセッション管理

- **セッションピル**: Issue ヘッダーに新しいピルが表示され、アクティブおよび完了済みのエージェントセッションを一覧で確認可能
- **セッションサイドパネル**: ピルまたは担当者エリアからセッションをクリックすると、サイドバーで進捗確認・ログ閲覧・エージェントへの指示が可能

#### プロジェクトでのセッション管理

- **デフォルト有効化**: 「Show agent sessions」が新規・既存のプロジェクトビューでデフォルト有効に
- **セッションサイドパネル**: プロジェクトボードからエージェントセッションをクリックしてサイドバーで直接操作可能

### その他の改善

- **スマートラベル提案**: ラベルピッカーが頻繁に使用するラベルを記憶し、上部に表示
