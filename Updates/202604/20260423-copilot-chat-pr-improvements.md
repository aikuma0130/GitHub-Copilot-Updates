# Copilot Chat のプルリクエスト対応が強化

## 日付

2026-04-23

## ステータス

Public Preview

## 概要

GitHub Copilot Chat がプルリクエストや差分に関するより豊富なコンテキストと新機能を提供するようになりました。PR の理解・レビュー・要約の3つの新機能が追加されています。

## 詳細

### 新機能

github.com/copilot またはグローバル Copilot ナビゲーションからプルリクエストに関する質問が可能になり、以下の3つの新しい機能が追加されました。

- **プルリクエストの理解**: PR をコンテキストとして与えると、コメント、ファイル変更、コミット、レビューなどのデータを含めて回答します
- **プルリクエストのレビュー**: Copilot Chat に PR のレビューを依頼すると、構造化されたレビューを提供します
- **プルリクエストの要約**: PR の要約を依頼すると、変更内容の簡潔な概要を提供します

### アクセス方法

- github.com/copilot またはグローバル Copilot ナビゲーションから利用可能
- パブリックプレビュー対象ユーザーは、差分ページの Copilot ボタンからも質問可能
- 「Help review this pull request」などのサジェストプロンプトが更新済み
