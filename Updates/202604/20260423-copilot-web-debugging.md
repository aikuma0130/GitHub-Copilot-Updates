# GitHub Copilot Web でスタックトレースのデバッグが強化

## 日付

2026-04-23

## ステータス

GA

## 概要

github.com の Copilot Chat にスタックトレースを貼り付けると、リポジトリのコードコンテキストを活用した構造化された根本原因分析が提供されるようになりました。

## 詳細

### 改善内容

スタックトレースの認識精度が向上し、Copilot Chat が以下の構造化された分析を提供します：

- **何が、どこで失敗したか**
- **なぜ失敗したか、どの前提が破られたか**
- **最も可能性の高い根本原因と、無効な状態が導入された箇所**
- **関連するコード参照からの証拠**
- **信頼度レベルと推奨される修正**
- **検証が必要な場合の次のチェック項目**

### 利用方法

github.com の Copilot Chat にスタックトレースを貼り付け、関連するリポジトリまたはファイルコンテキストを含めます。再現手順やエラーを引き起こす入力を追加すると、より迅速な根本原因分析が可能です。
