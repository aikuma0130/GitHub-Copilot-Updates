# GPT-5.5 が GitHub Copilot で一般提供開始

## 日付

2026-04-24

## ステータス

GA

## 概要

OpenAI の最新 GPT モデル GPT-5.5 が GitHub Copilot で一般提供開始されました。複雑なマルチステップのエージェントコーディングタスクにおいて最高のパフォーマンスを発揮し、従来の GPT モデルでは解決できなかった実世界のコーディング課題を解決します。

## 詳細

### 利用可能なプラン

- Copilot Pro+
- Copilot Business
- Copilot Enterprise

### 利用可能な環境

- Visual Studio Code
- Visual Studio
- Copilot CLI
- GitHub Copilot cloud agent
- github.com
- GitHub Mobile（iOS / Android）
- JetBrains
- Xcode
- Eclipse

### プレミアムリクエスト倍率

プロモーション価格として **7.5倍** のプレミアムリクエスト倍率が適用されます。

### 有効化

Copilot Enterprise および Copilot Business のプラン管理者は、Copilot 設定で GPT-5.5 ポリシーを有効にする必要があります。

### ロールアウト

段階的にロールアウトされます。まだ表示されない場合は、しばらくお待ちください。
