# GPT-5.2 および GPT-5.2-Codex の非推奨化予告

## 日付

2026-05-01

## ステータス

予告（2026年6月1日に非推奨化）

## 概要

GPT-5.2 および GPT-5.2-Codex が 2026年6月1日に全 Copilot エクスペリエンスで非推奨化されます（Copilot Code Review での GPT-5.2-Codex を除く）。代替モデルへの移行が推奨されています。

## 詳細

### 非推奨化スケジュール

| モデル | 非推奨化日 | 推奨代替モデル |
|---|---|---|
| GPT-5.2 | 2026-06-01 | GPT-5.5 |
| GPT-5.2-Codex | 2026-06-01 | GPT-5.3-Codex |

### 影響範囲

以下の全 Copilot エクスペリエンスが対象です：

- Copilot Chat
- インライン編集
- Ask モードおよび Agent モード
- コード補完

**例外：** Copilot Code Review における GPT-5.2-Codex は引き続き利用可能です。

### 必要な対応

- ワークフローと統合を非推奨化日までにサポート対象モデルに更新してください
- Copilot Enterprise 管理者は、Copilot 設定のモデルポリシーで代替モデルへのアクセスを有効化する必要がある場合があります
- 非推奨化後のモデル削除については追加の対応は不要です
