# 従量課金に向けた4月分レポートが利用可能に

## 日付

2026-05-12

## ステータス

GA

## 概要

従量課金（AI クレジット）への移行に向けて、4月分の GitHub Copilot 使用量レポートがダウンロード可能になりました。6月1日の正式移行前にコスト構造を把握し、予算計画に役立てることができます。

## 詳細

### 概要

2026年6月1日より、GitHub Copilot は AI クレジットを課金単位とする従量課金制に移行します。本日より、4月分の Copilot 使用量を AI クレジット換算で確認できるレポートをダウンロードできるようになりました。

### 対象ユーザー

- **Copilot Business / Enterprise 管理者**: エンタープライズ向けレポートをダウンロード可能
- **Copilot Pro / Pro+ ユーザー**: 個人使用量のレポートをダウンロード可能

### レポートの活用方法

- トップ消費者の特定
- どのモデルやサーフェスが最も多くの消費を生んでいるかの確認
- 6月1日以前の月次 AI クレジット範囲の見積もり

### 注意事項

- **4月1日〜24日の 0x モデル使用量はレポートに含まれません**。有料プラン全体で約2%程度の影響です。
- **4月24日〜30日のデータに重複エントリが含まれる場合があります**（データバックフィルのギャップによる）。
- **一部の Code Review エントリで AI クレジット推定値が欠落**しています。ライセンスなしユーザーからの自動化経由の Code Review は 0 AI クレジットと表示されます。

レポートは正確な請求書ではなく、コスト構造の方向性を示す参考値です。5月分のパターンを監視し、予算の更新を計画してください。
