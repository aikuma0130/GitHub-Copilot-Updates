# GitHub Copilot アプリがテクニカルプレビューとして提供開始

## 日付

2026-05-14

## ステータス

Technical Preview

## 概要

GitHub ネイティブのデスクトップアプリ「GitHub Copilot app」がテクニカルプレビューとして提供開始。Issue や PR からエージェント型開発セッションを開始し、コーディングからレビュー・マージまでを一つのアプリで完結できる。

## 詳細

### 概要

GitHub Copilot app は、GitHub のコンテキスト（Issue、PR、プロンプト、過去のセッション）から直接エージェント型開発を開始できるデスクトップネイティブアプリケーション。エディタ拡張ではなく、スタンドアロンの開発プラットフォームとして提供される。

### 主な機能

#### GitHub コンテキストからの開始

- Issue、PR、プロンプト、または過去のセッションから作業を開始
- 受信トレイで接続リポジトリ全体の Issue や PR を確認
- Issue の詳細、リポジトリの状態、レビューコメント、チェック結果がセッションに自動接続

#### 分離されたフォーカスセッション

- 各セッションが独自のブランチ、ファイル、会話、タスク状態を保持
- セッションの一時停止・再開が可能
- 複数プロジェクトにまたがる作業を分離して管理
- スキルやプロンプトを使ったトリアージ、依存関係更新、リリースノート作成などの自動化ワークフロー

#### ステアリング・検証・出荷の統合

- プランと差分のレビュー、フィードバックの提供、次のイテレーションへの誘導
- 統合ターミナルとブラウザからのコマンド実行・プレビュー確認・テスト
- セッションから直接 PR を作成
- Agent Merge によるレビューコメント対応、失敗チェックの修正、条件充足時の自動マージ

### 利用可能なプラン

- **Copilot Pro / Pro+**: 早期アクセスへのサインアップが可能
- **Copilot Business / Enterprise**: 週内にロールアウト予定。組織またはエンタープライズの管理者がプレビューと Copilot CLI を有効にする必要あり
