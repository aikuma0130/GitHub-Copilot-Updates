# Copilot Memory がユーザーレベルのプリファレンスに対応（Pro / Pro+ 向け早期アクセス）

## 日付

2026-05-15

## ステータス

Early Access

## 概要

Copilot Memory がリポジトリレベルの情報に加え、ユーザー個人のプリファレンス（好み・設定）を記憶できるようになった。Copilot Pro および Pro+ ユーザー向けに早期アクセスとして提供開始。

## 詳細

### 変更内容

従来、Copilot Memory はリポジトリレベルの情報のみを保存していたが、今回のアップデートにより、ユーザー個人のプリファレンスもキャプチャできるようになった。これにより、ユーザーの好みがリポジトリを横断して適用され、同じリポジトリの他のユーザーには影響しない。

### 記憶できるプリファレンスの例

- コミットスタイルの好み
- プルリクエストの構成方法
- コミュニケーションやトーンの好み

### プリファレンスの管理

ユーザーは GitHub の [Copilot Memory 設定](https://github.com/settings/copilot/memory) から、保存されたユーザーレベルのプリファレンスを確認・削除できる。

### 利用方法

Copilot Pro / Pro+ ユーザーは、[Copilot 設定](https://github.com/settings/copilot) から **Copilot Memory** を有効にすることで利用可能。今後、対応プランの拡大が予定されている。
