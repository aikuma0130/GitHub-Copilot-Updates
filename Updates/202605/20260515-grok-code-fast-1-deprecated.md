# Grok Code Fast 1 の非推奨化（実施完了）

## 日付

2026-05-15

## ステータス

Deprecated

## 概要

Grok Code Fast 1 が 2026年5月15日をもって全 GitHub Copilot エクスペリエンスで非推奨化された。代替として GPT-5 mini / Claude Haiku 4.5 が推奨されている。

## 詳細

### 非推奨化の詳細

| モデル | 非推奨化日 | 推奨代替モデル |
|---|---|---|
| Grok Code Fast 1 | 2026-05-15 | GPT-5 mini / Claude Haiku 4.5 |

### 影響範囲

以下の全 Copilot エクスペリエンスが対象：

- Copilot Chat
- インライン編集
- Ask モードおよび Agent モード
- コード補完

### 必要な対応

- ワークフローと統合をサポート対象モデル（GPT-5 mini / Claude Haiku 4.5）に更新してください
- Copilot Enterprise 管理者は、Copilot 設定のモデルポリシーで代替モデルへのアクセスを有効化する必要がある場合があります
- 非推奨化されたモデルの削除について追加の対応は不要です
