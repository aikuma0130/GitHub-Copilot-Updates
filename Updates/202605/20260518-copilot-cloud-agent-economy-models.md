# Copilot Cloud Agent: シンプルなタスク向けの高速・低コストモデル

## 日付

2026-05-18

## ステータス

GA

## 概要

Copilot Cloud Agent で利用可能なモデルに、より高速でコスト効率の高い選択肢として Claude Haiku 4.5 と GPT-5.4-mini（いずれも 0.33x 乗数）が追加された。タスクの複雑さに応じて最適なモデルを選択可能に。

## 詳細

Copilot Cloud Agent にタスクを委任する際、使用するモデルを選択できるようになっている。今回のアップデートでは、サポートモデルのリストに高速かつコスト効率の高いオプションが追加された。

### 新たに利用可能なモデル

| モデル | 乗数 |
|--------|------|
| Claude Haiku 4.5 | 0.33x |
| GPT-5.4-mini | 0.33x |

これにより、単純な変更にはより小さく高速なモデルを、複雑な作業にはより高性能なモデルを選ぶことで、コストとパフォーマンスのバランスを最適化できる。
