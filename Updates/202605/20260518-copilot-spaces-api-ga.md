# Copilot Spaces API が正式リリース（GA）

## 日付

2026-05-18

## ステータス

GA

## 概要

GitHub Copilot Spaces API が GA となり、Spaces の作成・取得・更新・削除をプログラムから実行可能に。大規模なコンテキスト管理を行うチームの手動作業を削減する。

## 詳細

### 変更内容

Copilot Spaces API が正式に一般提供（GA）された。自社アプリケーションから直接 Spaces を管理でき、GitHub UI での手動操作に依存する必要がなくなった。

### API で可能な操作

- 新しい Spaces のプログラム的作成
- 既存 Spaces の詳細取得
- Spaces の設定更新
- 不要な Spaces の削除
- Spaces のコラボレーターとリソースの管理

### ユースケース

複数の Spaces を管理する必要があるエンタープライズにとって特に有用。CI/CD パイプラインや自動化ワークフローとの統合が容易になる。
