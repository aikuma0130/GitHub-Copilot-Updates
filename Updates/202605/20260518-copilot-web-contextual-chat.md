# Copilot Web チャットがページ内コンテキスト対応に

## 日付

2026-05-18

## ステータス

GA

## 概要

GitHub.com 上の Copilot チャットが、閲覧中のページ上でパネルとして開くようになり、コンテキストスイッチなしに質問可能に。ナビゲーション時に参照が自動追加される機能も搭載。

## 詳細

### 変更内容

Copilot アイコンをクリックした際の動作が変更された。従来は github.com/copilot に遷移していたが、現在は閲覧中のページ上にパネルが開く。

### コンテキストの自動追加

プルリクエストや Issue などの GitHub サーフェスでチャットを開くと、そのページが自動的にチャットセッションのコンテキストとして付与される。GitHub 内をナビゲーションするたびに参照が追加され、複数の PR・Issue・リポジトリにまたがる質問が可能。

### イマーシブチャットへの切り替え

- Copilot アイコン横の「More」メニューから「In immersive chat」を選択
- コンテキストパネルの矢印アイコンをクリック

### エージェントセッションへの昇格

チャット内で PR の作成やディープリサーチの質問を行うことで、会話をエージェントセッションに変換可能。

### 対象プラン

全 GitHub Copilot プランで利用可能。
