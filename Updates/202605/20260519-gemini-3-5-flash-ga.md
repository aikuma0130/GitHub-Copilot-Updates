# Gemini 3.5 Flash が GitHub Copilot で GA

## 日付

2026-05-19

## ステータス

GA

## 概要

Google の最新 Flash ティアモデル「Gemini 3.5 Flash」が GitHub Copilot で一般提供開始。Flash ティアの速度・コスト効率で Pro に迫るコーディング品質を実現し、高速な反復型エージェントワークフローに最適。

## 詳細

### 変更内容

Gemini 3.5 Flash が GitHub Copilot のモデルピッカーに追加された。強力なツール使用能力、高速レスポンス、高いキャッシュ効率を特徴とし、高速で反復的なエージェント型コーディングワークフローに適している。

### プレミアムリクエスト乗数

ローンチ時点では 14x プレミアムリクエスト乗数が設定されているが、価格は暫定的で変更される可能性がある。

### 対象プラン

- Copilot Pro
- Copilot Pro+
- Copilot Business
- Copilot Enterprise

### 対応エディタ

- Visual Studio Code（v1.115.0 以降）
- Visual Studio（v17.14.22 または v18.1.0 以降）
- JetBrains
- Xcode
- Eclipse

### 管理者の操作

Copilot Enterprise / Business のプラン管理者は、Copilot 設定で Gemini 3.5 Flash ポリシーを有効にする必要がある。

### 段階的ロールアウト

ロールアウトは段階的に行われる。まだ表示されない場合は後日確認を推奨。
