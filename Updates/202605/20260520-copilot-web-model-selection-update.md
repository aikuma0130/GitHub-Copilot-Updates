# Copilot Web チャットで利用可能なモデルの更新

## 日付

2026-05-20

## ステータス

GA

## 概要

Copilot Chat on Web で利用可能なモデルの選択肢が絞り込まれ、全 Gemini モデルおよび一部モデル（GPT-5.2 Codex、GPT-5.4 nano 等）が削除された。より一貫した応答品質を提供するための変更。

## 詳細

### 変更内容

- すべての Gemini モデルが Copilot Chat on Web から削除
- GPT-5.2 Codex および GPT-5.4 nano 等の一部モデルが削除
- OpenAI モデルおよび Claude モデルは引き続き全 Copilot プランで利用可能

### 主なメリット

- 推奨モデルに絞ったシンプルな体験
- すべてのインタラクションでより一貫した応答品質

### 今後の方針

Web チャットでは今後、最適なパフォーマンスを確保するため、新モデルの追加はより限定的に行われる予定。

### 確認方法

利用可能なモデルの最新リストは [github.com/copilot](https://github.com/copilot) のモデルピッカー、または[公式ドキュメント](https://docs.github.com/copilot/reference/ai-models/supported-models#supported-ai-models-per-client)で確認可能。
