# Copilot Chat でセマンティック Issue 検索が利用可能に

## 日付

2026-05-20

## ステータス

GA

## 概要

GitHub Copilot Chat（Web）で自然言語による Issue のセマンティック検索が一般提供開始。正確なキーワードを覚えていなくても、意図に基づいて関連 Issue を発見・グルーピング・分析できる。

## 詳細

### 新機能

Copilot Chat に新しいセマンティック Issue インデックスを活用した検索機能が追加された。従来の完全一致検索や手動フィルターに加え、クエリの意図を理解し、異なる表現で書かれた関連 Issue も表示される。

### ユースケース

- Issue のタイトルやキーワードを正確に覚えていない場合の自然言語検索
- 特定のプラットフォームや環境に関連する Issue の素早いフィルタリング
- プランニング・トリアージ・ディスカバリーの効率化

### 利用方法

[GitHub Copilot Chat](https://github.com/copilot/) にアクセスしてセマンティック Issue 検索を利用可能。

### 対象プラン

全 Copilot プラン（Free / Pro / Pro+ / Business / Enterprise）で利用可能。
