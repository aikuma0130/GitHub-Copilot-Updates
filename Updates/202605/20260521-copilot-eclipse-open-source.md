# GitHub Copilot for Eclipse がオープンソース化

## 日付

2026-05-21

## ステータス

GA

## 概要

GitHub Copilot for Eclipse プラグインが MIT ライセンスでオープンソース化。コミュニティによる貢献とAI開発ツールの透明性向上を促進する。

## 詳細

GitHub Copilot for Eclipse のソースコードが [GitHub リポジトリ](https://github.com/microsoft/copilot-for-eclipse) で MIT ライセンスのもと公開された。

### オープンソース化の目的

- コミュニティ主導のイノベーションの促進
- AI ツールの透明性向上
- Eclipse エコシステムのオープンな精神に沿った開発

### 公開されたコード内容

- **コード補完**: インラインコード補完の生成とレンダリング
- **Next Edit Suggestions (NES)**: 次の編集候補の表示
- **チャット**: チャットビュー、会話フロー、ツール呼び出しの実装
- **Agent モード**: マルチステップのエージェント型ワークフロー
- **スキルとプロンプトファイル**: スキル・プロンプトファイルの検出・読み込み・呼び出し
- **BYOK (Bring Your Own Key)**: 独自APIキーの統合
- **高度なエージェント機能**: カスタムエージェント、分離サブエージェント、プランエージェント、MCP 統合

### コミュニティ貢献

オープンソース化以降、すでにコミュニティからの貢献が始まっている。Issue 報告やプルリクエストを通じたフィードバックを歓迎。
