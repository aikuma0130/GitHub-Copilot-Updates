# 組織ごとに Copilot モデルを指定できるモデルルールの導入

## 日付

2026-05-26

## ステータス

Public Preview

## 概要

エンタープライズオーナーが組織ごとに利用可能な Copilot モデルをきめ細かく制御できる「ターゲットモデルルール」がパブリックプレビューとして導入された。

## 詳細

### ターゲットモデルルール

エンタープライズ全体の一律設定ではなく、特定の組織に特定の Copilot モデルを許可するルールを作成可能に。これにより、セキュリティ要件やコンプライアンス要件が異なる部門・組織ごとに、利用可能なモデルを個別に管理できる。

### デフォルトモデル管理の UI 刷新

デフォルトモデルの可用性管理画面がリフレッシュされ、単一ページから以下を実行可能に:

- エンタープライズ内の組織に対してデフォルトで利用可能な Copilot モデルを選択
- 各モデルの可用性を「Enabled」（全組織で自動的に有効）または「Optional」（組織が個別に有効化を決定）に設定

### 対象プラン

Copilot Business および Copilot Enterprise プランの顧客が利用可能。
