# Claude Opus 4.8 が GitHub Copilot で一般提供開始

## 日付

2026-05-28

## ステータス

GA

## 概要

Anthropic の最新モデル Claude Opus 4.8 が GitHub Copilot で一般提供開始。コード理解・生成、複雑な問題解決、大規模コードベースのナビゲーションにおいて前バージョンから顕著な改善を実現。

## 詳細

### 利用可能なプラン

- Copilot Pro+
- Copilot Business
- Copilot Enterprise

### 利用可能な環境

- Visual Studio Code（チャット、Ask、編集、エージェントの全モード）
- Visual Studio
- Copilot CLI
- GitHub Copilot Cloud Agent
- GitHub Copilot App
- github.com
- GitHub Mobile（iOS / Android）
- JetBrains
- Xcode
- Eclipse

### プレミアムリクエスト乗数

2026年6月1日の従量課金（Usage Based Billing）開始まで **15倍** の乗数が適用される。

### アクセスの有効化

Copilot Enterprise および Business プランの管理者は、Copilot 設定で Claude Opus 4.8 ポリシーを有効化する必要がある。

### ロールアウト

段階的にロールアウトされるため、モデルピッカーに表示されるまで時間がかかる場合がある。
