# GitHub Copilot Code Review for Azure Repos がテクニカルプレビュー

## 日付

2026-06-02

## ステータス

Technical Preview

## 概要

GitHub Copilot Code Review が Azure Repos でテクニカルプレビューとして利用可能に。Azure DevOps のワークフロー内で直接 PR レビューを実行でき、Copilot ライセンス不要。

## 詳細

### 仕組み

組織とリポジトリレベルで有効化すると、Azure Repos のプルリクエストから直接 Copilot コードレビューをリクエスト可能。Copilot はコード変更にインラインでレビューコメントを追加し、改善提案を行い、開発プロセスの早い段階で問題を特定する。

### 利用対象

テクニカルプレビューにサインアップしたすべての Azure DevOps ユーザーが利用可能。GitHub Copilot ライセンスは不要。

### 課金

- GitHub AI Credits で課金
- 既存の GitHub Copilot プランの含有 AI Credits からは差し引かれない
- 2026年6月2日から課金開始
- プレビュー課金であり、GA 時に価格変更の可能性あり

### 主な利点

- Azure Repos 内でのオンデマンド PR レビューで問題を早期発見
- 既存の Azure DevOps ワークフローを変更不要
- Copilot ライセンスのプロビジョニングなしで利用開始可能
- GitHub AI Credits による従量課金
