# Copilot Memory がユーザープリファレンスを Business / Enterprise で対応

## 日付

2026-06-02

## ステータス

Public Preview

## 概要

Copilot Memory のユーザーレベルプリファレンスが Copilot Business および Enterprise でパブリックプレビュー開始。コミュニケーションスタイルやツールスタック等の個人設定をリポジトリ横断で適用可能に。

## 詳細

Pro / Pro+ ユーザーへのロールアウトに続き、Copilot Memory のユーザーレベルプリファレンスが Business および Enterprise プランでも利用可能になった。

### 主な機能

- **ユーザーレベルプリファレンス**: コミュニケーションスタイル、ツールスタック、Git 規約などを記憶し、リポジトリ横断で適用
- **管理者ポリシー**: 既存の Copilot Memory ポリシーがユーザーレベルプリファレンスにも適用（デフォルト無効）
- **監査用エクスポート**: 管理者がユーザーレベルプリファレンスをエクスポート可能
- **一括削除**: 管理者がプリファレンスを一括削除可能
- **ユーザーオプトアウト**: 個人設定からいつでもオプトアウト可能
- **課金エンティティ分離**: 生成されたデータは課金エンティティ内にスコープ

### 利用可能な環境

Copilot Coding Agent および Copilot CLI でパブリックプレビューとして提供。
