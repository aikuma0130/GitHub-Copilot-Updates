# エージェントアプリで GitHub を拡張

## 日付

2026-06-02

## ステータス

GA

## 概要

GitHub パートナーの AI エージェントを GitHub Marketplace からインストールし、GitHub ワークフローに直接統合できる「エージェントアプリ」が提供開始。

## 詳細

### エージェントアプリとは

エージェントアプリは GitHub パートナーが提供する AI エージェントで、GitHub Marketplace からインストール可能。通常の GitHub App と同様にインストールし、管理者が有効化すると GitHub ワークフロー内で利用可能になる。

### 利用方法

3つのエントリーポイントから利用：

- イシューをエージェントにアサイン
- PR コメントでエージェントを @メンション
- Agents UI でエージェントを選択してプロンプトを入力

### 初期パートナー

以下のパートナーからエージェントアプリが提供開始：

- Amplitude
- Bright Security
- Endor Labs
- LaunchDarkly
- Miro
- Sonar
- PagerDuty
- Packfiles
- Octopus Deploy

### 今後の展開

今後数か月で、誰でもエージェントアプリを構築できるようアクセスを拡大予定。パートナー向けのウェイトリストも公開中。
