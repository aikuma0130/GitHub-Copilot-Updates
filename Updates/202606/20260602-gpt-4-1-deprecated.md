# GPT-4.1 が廃止

## 日付

2026-06-02

## ステータス

Deprecated

## 概要

GPT-4.1 がすべての GitHub Copilot エクスペリエンスで廃止された。推奨される代替モデルは GPT-5.5。

## 詳細

### 廃止内容

2026年6月1日付で、GPT-4.1 が GitHub Copilot のすべてのエクスペリエンスから廃止された。対象は以下を含む：

- Copilot Chat
- インライン編集
- Ask モードおよび Agent モード
- コード補完

### 推奨代替モデル

| モデル | 廃止日 | 推奨代替 |
|--------|--------|----------|
| GPT-4.1 | 2026-06-01 | GPT-5.5 |

### 管理者への注意事項

- Copilot Enterprise 管理者は、Copilot 設定のモデルポリシーから代替モデルへのアクセスを有効にする必要がある場合がある
- 廃止されたモデルの削除に対するアクションは不要
- 質問や懸念がある場合はアカウントマネージャーに連絡可能
