# MAI-Code-1-Flash が GitHub Copilot で利用可能に

## 日付

2026-06-02

## ステータス

GA（段階的ロールアウト中）

## 概要

Microsoft の最新小型コーディングモデル MAI-Code-1-Flash が GitHub Copilot で利用可能になった。VS Code から段階的にロールアウト中。

## 詳細

### モデルの特徴

MAI-Code-1-Flash は Microsoft が開発した小型コーディングモデルで、以下の特徴を持つ：

- そのサイズクラスで最高品質のコーディング性能を実現
- 初期テストで他の小型モデルを上回る性能
- 軽量なコーディングワークフローに最適化
- GitHub Copilot 向けに特別に設計・チューニングされた初のモデル

### 利用可能なプラン

以下のプランで段階的にロールアウト中：

- Copilot Free
- Copilot Pro
- Copilot Pro+
- Copilot Max

最初は限定的なユーザーに提供され、今後数週間で拡大予定。

### 利用方法

Visual Studio Code のモデルピッカーからモデルを選択可能。
