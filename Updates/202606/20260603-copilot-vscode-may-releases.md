# GitHub Copilot in Visual Studio Code — 2026年5月リリースまとめ

## 日付

2026-06-03

## ステータス

GA / Public Preview

## 概要

VS Code v1.120〜v1.123 のリリースをカバー。エージェントウィンドウが Stable でプレビュー提供開始、BYOK モデルのエアギャップ環境対応、ターミナルの安全性向上など多数の改善が含まれる。

## 詳細

### エージェントウィンドウ

- **エージェントウィンドウ（Stable プレビュー）**: タスク完了に特化したエージェントファーストの専用画面が VS Code Stable で利用可能に。複数プロジェクトの横断的なナビゲーションと変更レビューを高速化。
- **リモートエージェント（プレビュー）**: SSH や Dev Tunnels を介してリモートマシン上でセッションを実行。クライアント切断後もセッションが継続。
- **Agent Host Protocol（AHP）**: 複数クライアント間でエージェントセッション状態を同期するオープンプロトコルへの継続的な投資。
- **セッション設定の引き継ぎ**: 新規セッションでもエージェントハーネスや分離モードなどの直近の選択が維持される。
- **セッションと Git フロー改善**: ベースブランチの最新変更を自動プル、Git 状態の自動リフレッシュ、リモートマシンでのタスクトリガーが可能に。
- **セッション同期**: チャットセッションが GitHub アカウントに自動同期され、マシンやワークスペースをまたいで検索可能な作業履歴を提供。
- **Chronicle**: `/chronicle` コマンドで過去のセッション検索、スタンドアップレポート生成、パーソナライズされた生産性ヒントの取得が可能。
- **複数セッションの並行表示**: エージェントウィンドウで複数のセッションを同時に開き、ドラッグや Alt+クリックで並べて比較・レビュー可能。
- **サンドボックスでのネットワークコマンド自動リトライ**: ネットワークアクセスが必要なターミナルコマンドが、ファイルシステム保護を維持しながら自動的にリトライされる。

### 言語モデルと BYOK

- **エアギャップ BYOK**: GitHub 認証なしで、隔離された環境で独自のモデルを実行可能に。
- **カスタムエンドポイントプロバイダー**: チャット完了・レスポンス・メッセージ互換のエンドポイントを1つのプロバイダーフローから追加可能。
- **プロバイダー別モデルピッカー**: マルチプロバイダー環境でモデルの検索・切替が容易に。
- **BYOK トークン可視化**: コンテキストウィンドウで BYOK モデルの実際のトークン使用量を表示。
- **推論努力度コントロール**: モデルピッカーから直接思考量を設定し、品質・レイテンシ・コストのバランスを調整。
- **ユーティリティモデルの設定**: タイトル、サマリー、リネーム提案、コミットメッセージ、インテント検出に使用するモデルを選択可能に。

### ターミナルの安全性と効率

- **ターミナル出力圧縮の拡大**: テスト、ビルド、リンター、Docker、パッケージマネージャーの冗長出力をモデルに送る前に圧縮し、トークン使用量とコストを最適化。
- **コマンドリスク評価（実験的）**: ターミナル確認時に AI 生成のリスクレベルと安全性の説明を表示。
- **機密プロンプトのターミナル内処理**: パスワード、パスフレーズ、PIN、認証コードはターミナル内で直接入力され、LLM に共有されない。
- **バックグラウンドコマンド UX 改善**: チャット内の実行状態インジケータが明確化され、完了したバックグラウンドエージェントターミナルが自動クリーンアップ。
- **エージェント対応ターミナルコマンド**: `VSCODE_AGENT` 環境変数により CLI がエージェント起動コマンドに適応可能。

### その他の新機能

- **統合ブラウザ**: デバイスエミュレーション、スクリーンショット機能、お気に入りページの保存機能を追加。
- **HTML ファイルプレビュー**: 統合ブラウザでローカル HTML ファイルを直接プレビュー可能。
- **変更ファイルのみの検索**: 検索パネルでローカル変更済み・未コミットファイルのみにスコープを絞るトグルを追加。
- **Markdown プレビュー改善**: Mermaid ダイアグラムと YAML フロントマターの表示がビルトイン化。
- **クイック提案のデフォルト調整**: インライン補完が利用可能な場合のノイズを軽減。
