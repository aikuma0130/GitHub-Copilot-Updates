# Agent tasks REST API が Copilot Pro、Pro+、Max で利用可能に

## 日付

2026-06-04

## ステータス

Public Preview

## 概要

Copilot クラウドエージェントのタスクをプログラムから開始・追跡できる REST API がパブリックプレビューとして提供開始。カスタム自動化への統合が容易に。

## 詳細

Copilot Pro、Pro+、Max ユーザーは、Agent tasks REST API を使用して Copilot クラウドエージェントのタスクをプログラムで開始・追跡できるようになった。

Copilot クラウドエージェントは独自の開発環境でバックグラウンド動作し、コード変更の作成・検証を行った後、プルリクエストを作成する。

### 主なユースケース

- 多数のリポジトリにわたるリファクタリングやマイグレーションをスクリプトから一括実行
- 社内開発者ポータルからワンクリックで新規リポジトリをセットアップ
- 毎週のリリースノートを含む新リリースを自動準備

### 認証

- パーソナルアクセストークン（Classic および Fine-grained）
- OAuth トークン
