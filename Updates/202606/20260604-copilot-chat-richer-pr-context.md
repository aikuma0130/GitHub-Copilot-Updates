# Copilot Chat がプルリクエストでより豊かなコンテキストを提供

## 日付

2026-06-04

## ステータス

GA

## 概要

GitHub Copilot Chat が github.com 上の差分やプルリクエストで、より豊かなコンテキストと新しい機能を提供開始。パブリックプレビューを経て一般提供に。

## 詳細

### コードとチャットの並行表示

プルリクエストの差分を見ながら、Copilot との会話を直接横に表示可能。チャットウィンドウとプルリクエストを切り替えることなく、コメントやインライン編集が可能に。

### コンテキストの高速取得

プルリクエストの理解・レビュー・サマリーの新機能により、差分やプルリクエストについて質問するたびに関連するプルリクエストとリポジトリのコンテキストが自動的に追加される。パフォーマンスも改善され、より迅速に質の高い回答を取得可能。

### 利用方法

各差分の上部にある「Ask about this diff」ボタン、トップナビゲーションバーの Copilot ボタン、またはコード行をハイライトしてドロップダウンから「Copilot」を選択してアクセス可能。
