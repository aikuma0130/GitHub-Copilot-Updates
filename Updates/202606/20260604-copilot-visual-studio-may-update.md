# GitHub Copilot in Visual Studio — 2026年5月アップデート

## 日付

2026-06-04

## ステータス

GA

## 概要

Visual Studio 2026 向けの5月リリース。新しい Plan エージェント、Skills パネル、マルチファイル差分サマリー、コンテキストウィンドウ使用量管理など、計画・レビュー・管理ツールが大幅に強化された。

## 詳細

### Plan エージェント

エージェントピッカーで「Plan」を選択すると、コード変更前に Copilot と実装計画を共同作成できる。Copilot は読み取り専用ツールでコードベースを探索し、明確化のための質問を行い、`.copilot/plans/plan-{title}.md` に詳細な計画を保存する。準備ができたら「Implement plan」をクリックして Agent モードに引き継ぎ可能。

### Skills パネル

チャットウィンドウに新しい Skills パネルが追加され、ワークスペースとユーザープロファイルから検出されたすべてのエージェントスキルを一覧表示。スキルの編集、ファイル場所の参照、名前やキーワードによる検索が可能。

### マルチファイルサマリー差分

Copilot が複数ファイルを編集した後、ワーキングセットの「Open change summary view」をクリックすると、すべての変更を1つのタブで確認可能。全ファイル・ファイル単位・チャンク単位での受け入れ・元に戻し操作に対応。

### コンテキストウィンドウ使用量管理

Copilot Chat プロンプトの右上に新しいリングアイコンが追加され、コンテキストウィンドウの使用量を視覚的に表示。クリックで詳細な内訳を確認でき、「Summarize conversation」で以前のターンを圧縮してスペースを確保可能。

### コミットを Copilot Chat に追加

Git History、File History、Annotate（Blame）ビューからコミットを右クリックし、Copilot Chat のコンテキストとして直接添付可能。複数選択にも対応。

### コミットメッセージ指示の移動

コミットメッセージのカスタム指示が、リポジトリの Copilot カスタム指示ファイルに移動。すべての Copilot ガイダンスを一箇所で管理可能に。

### C++ ビルド最適化分析の改善

@BuildPerfCpp がフルリビルド分析でリグレッションを検出した場合、同等のインクリメンタルビルドを再実行するようになった。プリコンパイル済みヘッダーやヘッダーリファクタリングなどの日常的な最適化効果をより正確に反映。
