# GitHub Copilot の大規模コンテキストウィンドウと構成可能な推論レベル

## 日付

2026-06-04

## ステータス

GA

## 概要

GitHub Copilot が100万トークンのコンテキストウィンドウと構成可能な推論レベルをサポート開始。より大規模で複雑なコードベースの理解と、タスクに応じた速度・深度のバランス調整が可能になった。

## 詳細

### 100万トークンコンテキストウィンドウ

より大きなコードベース、長いドキュメント、複雑なマルチファイルプロジェクトにわたってコンテキストを失うことなく作業可能に。VS Code、Copilot CLI、GitHub Copilot アプリで利用可能で、今後さらに多くのサーフェスに拡大予定。

### 構成可能な推論レベル

速度と深度の適切なバランスを調整でき、最も困難なアーキテクチャやデバッグの課題に対して拡張思考（extended thinking）を活用可能。VS Code、Copilot CLI、GitHub Copilot アプリで利用可能。

### AI クレジット消費に関する注意

より大きなコンテキストウィンドウや高い推論レベルを選択すると、インタラクションあたりの AI クレジット消費量が増加する。日常的なタスクにはデフォルト設定を使用し、複雑なマルチファイル問題に取り組む際に拡張コンテキストや高い推論レベルの活用が推奨される。
