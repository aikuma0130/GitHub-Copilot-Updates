# GPT-5.2 および GPT-5.2-Codex が Copilot で廃止

## 日付

2026-06-05

## ステータス

Deprecated

## 概要

GPT-5.2 と GPT-5.2-Codex が GitHub Copilot の主要エクスペリエンス（Chat、インライン編集、エージェントモード、コード補完）で廃止された。GPT-5.2 は Code Review では引き続き利用可能。

## 詳細

2026年6月5日付けで、以下のモデルが廃止された。

| モデル | 廃止日 | 推奨代替モデル |
|--------|--------|---------------|
| GPT-5.2 | 2026-06-05 | GPT-5.5 |
| GPT-5.2-Codex | 2026-06-05 | GPT-5.3-Codex |

### 影響範囲

- Copilot Chat
- インライン編集
- Ask / Agent モード
- コード補完

### 注意事項

- GPT-5.2 は Copilot Code Review では引き続き利用可能
- Enterprise 管理者はモデルポリシー設定で代替モデルへのアクセスを有効化する必要がある場合がある
- 廃止モデルの削除にユーザー操作は不要
