# Copilot Chat がエージェントセッションを表示可能に

## 日付

2026-06-10

## ステータス

GA

## 概要

Copilot Chat からクラウドエージェントセッションのステータス確認・検索・フォローアップが可能になった。エージェントの作業内容を Chat 上で直接把握・継続できる。

## 詳細

Web 上の Copilot Chat と Copilot クラウドエージェント間のハンドオフ体験が改善され、過去のエージェントセッションを Chat 内で検索・照会できる新機能が追加された。

### 主な変更点

- Chat からエージェントセッションを開始すると、進行中のセッションのステータスが Chat に反映される
- セッション完了後、Chat 内でフォローアップの質問や新たなセッションの開始が可能
- 2つの新しいツールが有効化:
  - **Get agent logs**: プルリクエストに対するクラウドエージェントのセッションログを取得し、変更内容・検証結果・理由を会話内で確認可能
  - **Session search**: トピック・タイトル・最近の履歴でエージェントセッションを検索・要約し、過去の作業を容易に再開可能
