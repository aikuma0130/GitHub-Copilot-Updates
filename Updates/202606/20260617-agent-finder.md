# GitHub Copilot の Agent Finder が利用可能に

## 日付

2026-06-17

## ステータス

GA

## 概要

Agent Finder により、Copilot がタスクに最適な AI エージェント・ツール・スキルを自動的に発見・推薦する機能が一般提供開始。全 Copilot プランで利用可能。

## 詳細

Agent Finder は、MCP サーバー、スキル、キャンバス、エージェント、ツールを手動で選択する代わりに、タスクの説明に基づいて最適なリソースを自動的に発見・推薦する。

### 主な特徴

- **自動発見**: タスクを自然言語で記述するだけで、利用可能な AI リソースのインデックスから最適なものをランク付けして推薦
- **リソース効率**: ジョブに必要なエージェントだけをロードし、オーバーヘッドを削減
- **オープン標準**: Google、GoDaddy、Hugging Face、Microsoft と共同開発した Agentic Resource Discovery（ARD）仕様を実装
- **カスタムレジストリ**: GitHub のパブリックカタログだけでなく、組織のプライベートレジストリも指定可能
- **ガバナンス**: 自動インストールや暗黙の接続はなく、ユーザーが完全にコントロール
- **エンタープライズ管理**: Copilot の既存管理コントロールが Agent Finder にも適用
