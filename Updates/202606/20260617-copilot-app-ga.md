# GitHub Copilot アプリが一般提供開始

## 日付

2026-06-17

## ステータス

GA

## 概要

GitHub Copilot アプリが macOS、Windows、Linux で一般提供開始。エージェントネイティブなデスクトップ開発体験を提供する。

## 詳細

Copilot アプリは、エージェント駆動の開発に特化したネイティブデスクトップアプリケーション。Issue、プルリクエスト、プロンプトからセッションを開始し、リポジトリ横断で並列セッションを実行できる。

### 主な機能

- **セッション管理**: Issue、PR、プロンプトからセッションを開始し、各セッションは独自のブランチとワークツリーで動作
- **並列作業**: リポジトリ横断で複数のセッションを同時に実行可能
- **差分レビュー**: 統合ターミナルとブラウザで変更を検証し、チームの既存チェックとマージ要件を使用して PR を作成
- **キャンバス**: エージェントとの双方向コラボレーションサーフェスで、計画・PR・ターミナル・ブラウザセッションの進捗をリアルタイムに追跡
- **クラウドオートメーション**: ローカルマシンに依存せず、クラウドで定期的なエージェント作業をスケジュール
- **カスタマイズ**: セッションごとに AI モデルを選択し、MCP サーバーを通じて外部ツールを接続可能

### 利用要件

Copilot Business / Enterprise プランのユーザーは、組織またはエンタープライズ管理者がポリシー設定で Copilot CLI を有効にする必要がある。
