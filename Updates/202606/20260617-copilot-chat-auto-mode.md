# Copilot Chat の Auto モードが全ユーザーに提供開始

## 日付

2026-06-17

## ステータス

GA

## 概要

Copilot Chat の Auto モードが全プランのユーザーに一般提供開始。リクエストの複雑さやモデルの可用性に基づき、最適な AI モデルを自動的に選択する。

## 詳細

Auto モードは、ユーザーのリクエストを評価し、Claude Sonnet 4.6、GPT-5.4 mini、GPT-5.4、Haiku 4.5 などの利用可能なモデルから最適なものにルーティングする。

### 主な特徴

- **動的モデル選択**: リクエストの複雑さとリアルタイムのモデル可用性に基づいて最適なモデルを自動選択
- **透明性**: チャット上でモデルの応答にホバーすると、使用されたモデルを確認可能
- **柔軟な制御**: Auto モードと手動モデル選択をいつでも切り替え可能
- **ポリシー準拠**: ユーザーおよび管理者のモデル設定・制限を尊重
- **トークン割引**: 有料 Copilot サブスクライバーは Auto モード使用時に 10% のトークン割引を受けられる

GitHub.com および GitHub モバイルアプリで利用可能。
