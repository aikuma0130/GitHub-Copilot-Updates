# MAI-Code-1-Flash がさらに多くの Copilot サーフェスで利用可能に

## 日付

2026-06-18

## ステータス

GA

## 概要

Microsoft のコーディング特化小型モデル MAI-Code-1-Flash が、Copilot CLI・GitHub Copilot アプリ・Visual Studio・JetBrains・Eclipse・Xcode・GitHub Mobile など追加のサーフェスで利用可能になった。

## 詳細

MAI-Code-1-Flash は、Microsoft がコーディング用途に特化して開発した小型モデル。そのサイズクラスで最高レベルの品質を実現し、GitHub Copilot 向けに特別にチューニングされている。

### 新たに対応したサーフェス

- Copilot CLI
- GitHub Copilot アプリ
- Copilot Chat on GitHub
- Visual Studio
- GitHub Mobile
- JetBrains IDEs
- Eclipse
- Xcode

### 利用可能なプラン

Copilot Free、Student、Pro、Pro+、Max プランで利用可能。限定ユーザーから段階的にロールアウトされ、今後数週間で拡大予定。Copilot Business / Enterprise への対応も近日予定。
