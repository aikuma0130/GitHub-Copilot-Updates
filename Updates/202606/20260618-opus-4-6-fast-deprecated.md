# Opus 4.6 (fast) の廃止予定

## 日付

2026-06-18

## ステータス

Deprecated（2026-06-29 に廃止予定）

## 概要

Opus 4.6 (fast) がすべての GitHub Copilot エクスペリエンスで 2026年6月29日に廃止予定。推奨代替モデルは Opus 4.8 (fast)。

## 詳細

### 対象範囲

Opus 4.6 (fast) は以下のすべての Copilot エクスペリエンスで廃止される：

- Copilot Chat
- インライン編集
- Ask モード / Agent モード
- コード補完

### 移行ガイダンス

- 廃止後、Opus 4.6 (fast) はモデルセレクターに表示されなくなる
- 推奨代替モデル: **Opus 4.8 (fast)**
- エンタープライズ管理者は Copilot 設定のモデルポリシーで Opus 4.8 (fast) へのアクセスを事前に有効化すること
- 自動化やワークフローで Opus 4.6 (fast) を指定している場合は、廃止日までに更新が必要

### タイムライン

- 発表日: 2026年6月18日
- 廃止日: 2026年6月29日
