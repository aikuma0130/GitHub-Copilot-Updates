# GitHub Copilot for Jira が一般提供開始

## 日付

2026-06-25

## ステータス

GA

## 概要

GitHub Copilot for Jira が一般提供開始。エージェント進捗のリアルタイムストリーミング、セッション後のフォローアップ指示、簡素化されたオンボーディングが新たに追加。

## 詳細

2026年3月のパブリックプレビュー開始以降、ユーザーフィードバックに基づき多くの機能強化が行われ、今回の GA リリースに至った。

### GA で追加された新機能

#### エージェント進捗の Jira へのストリーミング

コーディングエージェントの作業進捗を Jira イシュー内でリアルタイムに確認可能。GitHub に切り替えることなくエージェントの状況を把握できる。

#### セッション後のステアリング

エージェントがドラフト PR を作成した後、Jira のチャットパネルから直接フォローアップの指示を送信可能。新しい PR を作成せず、同じ PR 上で作業を継続するため、レビューが効率化される。

#### 簡素化されたオンボーディング

GitHub 組織とリポジトリの接続に必要な設定ステップが削減され、セットアップが高速化。

### パブリックプレビュー期間中に提供された機能

- Jira 内からのモデル選択
- PR タイトルへの Jira チケット参照
- MCP 経由の Confluence コンテキスト統合
- カスタムエージェントとカスタムフィールド
- スペースレベルのカスタムガイダンス
- Jira でのレビューリクエスト通知
- オンボーディングガイダンスとエラーメッセージの改善

### 利用開始方法

Atlassian Marketplace から GitHub Copilot for Jira アプリをインストール（または最新版に更新）。
