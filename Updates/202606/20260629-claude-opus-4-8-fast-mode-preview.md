# Claude Opus 4.8 (fast mode) が GitHub Copilot でプレビュー提供開始

## 日付

2026-06-29

## ステータス

Public Preview

## 概要

Claude Opus 4.8 のファストモードが GitHub Copilot でプレビューとして利用可能に。標準モードと同等の知性を維持しつつ、大幅に高速な出力トークン速度を実現し、インタラクティブなコーディングやエージェントワークフローに最適。

## 詳細

### 特徴

- Claude Opus 4.8 と同等の知性を維持しながら、出力トークン速度が大幅に向上
- インタラクティブなコーディングやエージェントワークフローなど、レスポンスの速さが重要なシナリオに最適
- 以前のファストモードと比較してコストが削減されているが、標準の Claude Opus 4.8 よりは高コスト
- 使用量ベースの課金（Usage Based Billing）でプロバイダーリスト価格にて課金

### 対象プラン

- Copilot Pro+
- Copilot Max
- Copilot Business
- Copilot Enterprise

### 利用可能なプラットフォーム

- Visual Studio Code
- Visual Studio
- Copilot CLI
- GitHub Copilot cloud agent
- GitHub Copilot app
- github.com
- GitHub Mobile（iOS / Android）
- JetBrains
- Xcode
- Eclipse

### 有効化

Copilot Enterprise および Business プランの管理者は、Copilot 設定で Claude Opus 4.8 のファストモードポリシーを明示的に有効化する必要がある（デフォルトではオフ）。

### 注意事項

- ロールアウトは段階的に行われる
- Opus 4.6 (fast) は同日（2026年6月29日）に廃止済み
