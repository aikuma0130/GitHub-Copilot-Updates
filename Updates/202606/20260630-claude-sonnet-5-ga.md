# Claude Sonnet 5 が GitHub Copilot で一般提供開始

## 日付

2026-06-30

## ステータス

GA

## 概要

Anthropic の最新 Sonnet クラスモデル「Claude Sonnet 5」が GitHub Copilot で一般提供開始。CLI スタイルのタスクで特に優れた性能を発揮し、高速かつ高品質なコーディング支援を提供する。

## 詳細

Claude Sonnet 5 は Anthropic の最新 Sonnet クラスモデルであり、日常的な開発作業からエージェントワークフローまで幅広いコーディングシナリオで強力な性能を発揮する。

### 主な特徴

- CLI スタイルのタスクで特に優れたパフォーマンス
- 優れたプロンプトキャッシュ活用率
- 低い推論レベルでの競争力のあるレイテンシ
- Zero Data Retention（ZDR）に対応

### 利用可能なプラン

- Copilot Pro
- Copilot Pro+
- Copilot Max
- Copilot Business
- Copilot Enterprise

### 利用可能なサーフェス

- Visual Studio Code
- Visual Studio
- Copilot CLI
- GitHub Copilot クラウドエージェント
- GitHub Copilot アプリ
- github.com
- GitHub Mobile（iOS / Android）
- JetBrains
- Xcode
- Eclipse

### 課金

プロバイダーリスト価格に基づく従量課金（Usage Based Billing）で提供。

### 管理者向け

Copilot Enterprise および Business プラン管理者は、Copilot のモデルポリシー設定から Claude Sonnet 5 を有効化可能。
