# Copilot Agent が JetBrains AI Assistant で利用可能に

## 日付

2026-06-30

## ステータス

GA

## 概要

GitHub Copilot が JetBrains AI Assistant のエージェントピッカーにファーストクラスオプションとして統合された。モデル選択や推論深度の調整を含むマルチステップのコーディングタスクを JetBrains IDE 内で直接実行可能に。

## 詳細

JetBrains と GitHub による統合強化により、GitHub Copilot が JetBrains AI Assistant のエージェントピッカーでネイティブに選択可能になった。既存の Copilot プラグインや ACP（Agent Client Protocol）経由のアクセスに加え、より直感的なワークフロー選択が可能になる。

### 新機能

- **GitHub Copilot をネイティブエージェントピッカーで選択**: AI チャットのエージェントピッカーから GitHub Copilot を直接選択し、アクティブエージェントとして会話に利用可能
- **モデル選択と推論深度の調整**: サポートされている Copilot モデルを選択し、速度・深度・コストのバランスを調整可能
- **マルチステップコーディングタスク**: プロジェクトを分析し、変更を提案し、コマンドを実行し、開発者と対話的にイテレーション

### 今後の予定

- **NES（Next Edit Suggestions）サポート**: マルチステップのコード変更で次の編集を提案
- **スキル**: 再利用可能な特化機能を迅速に呼び出し可能に
- **ツール間のより深いオーケストレーション**: 複雑な開発タスクの計画・実行・イテレーションをさらに改善
