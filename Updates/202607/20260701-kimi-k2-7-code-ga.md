# Kimi K2.7 Code が GitHub Copilot で一般提供開始

## 日付

2026-07-01

## ステータス

GA

## 概要

初のオープンウェイトモデルとして Kimi K2.7 Code が Copilot のモデルピッカーで選択可能に。GitHub が Microsoft Azure 上でホストし、低コストなコーディングワークフローの選択肢を提供。

## 詳細

### 提供プラン

Copilot Pro、Pro+、Max プランで段階的にロールアウト中。Copilot Business / Enterprise への拡大は今後数週間で予定。

### 利用可能な環境

- Visual Studio Code（バージョン 1.127.0 以降）
- Visual Studio（バージョン 17.14.6 以降）
- Copilot CLI
- GitHub Copilot Cloud Agent
- GitHub Copilot App
- github.com
- GitHub Mobile（iOS / Android）
- JetBrains（バージョン 1.9.1-251 以降）
- Xcode
- Eclipse

### Business / Enterprise での有効化

Kimi K2.7 Code は Business / Enterprise プランでは**デフォルト無効**。プラン管理者が Copilot 設定で Kimi K2.7 Code ポリシーを明示的に有効にする必要がある。

管理者はオープンウェイトモデルを有効にする前に、自組織のセキュリティ・コンプライアンス・データガバナンス要件を確認することを推奨。

### 課金

使用量ベース課金のプロバイダー公式価格で課金される。
