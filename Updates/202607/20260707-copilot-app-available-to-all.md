# GitHub Copilot アプリが全プランで利用可能に

## 日付

2026-07-07

## ステータス

GA

## 概要

GitHub Copilot デスクトップアプリが全 Copilot プラン（Free、Education 含む）で利用可能になった。macOS、Windows、Linux に対応し、BYOK（自分のモデルキーを持ち込む）でサブスクリプション不要での利用も可能。

## 詳細

### 主な特徴

- **全プラン対応**: Copilot Free および GitHub Education ユーザーを含む全プランで利用可能
- **BYOK サポート**: Copilot サブスクリプションなしでも、自分のモデルプロバイダーキーを使用してセッションを実行可能
- **マルチプラットフォーム**: macOS、Windows、Linux で利用可能

### Business / Enterprise プランでの利用

Copilot Business または Enterprise プランでアプリを利用するには、組織またはエンタープライズの管理者がポリシー設定で Copilot CLI を有効にする必要がある。

### 始め方

[GitHub Copilot アプリをダウンロード](https://gh.io/app)して、GitHub アカウントでサインインするだけで数クリックで最初のセッションを開始できる。
