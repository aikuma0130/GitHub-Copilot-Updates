# Copilot Billing Preview アプリが 8月3日に廃止予定

## 日付

2026-07-07

## ステータス

Deprecated

## 概要

Copilot Billing Preview アプリが 2026年8月3日に廃止予定。GitHub の組み込み課金機能がアプリの機能を上回ったため、ユーザーは組み込みの課金設定への移行が必要。

## 詳細

### 廃止の背景

Copilot Billing Preview アプリは、使用量ベース課金（AI クレジット）への移行期間中にユーザーが課金状況を把握するために構築されたもの。GitHub の組み込み課金機能がより詳細な可視性を提供するようになったため廃止が決定された。

### 移行先の機能

GitHub の組み込み課金設定で以下が可能:

- 「AI usage」ページで AI クレジットデータのグループ化、フィルタリング、エクスポート
- ユーザーレベルでの予算設定（組織の場合）
- 使用レポートまたは Billing API を通じた生の使用データへのアクセス
- コストセンターによる使用量管理

### 対応が必要な事項

2026年8月3日までに、Copilot Billing Preview アプリに依存しているワークフローを GitHub の組み込みツールに移行する必要がある。

### 影響範囲

この変更はプレビュー視覚化アプリの廃止のみに影響。Copilot のメインエクスペリエンスやサブスクリプションには影響なし。
