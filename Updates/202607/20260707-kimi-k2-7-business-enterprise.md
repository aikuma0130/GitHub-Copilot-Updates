# Kimi K2.7 Code が Copilot Business / Enterprise で利用可能に

## 日付

2026-07-07

## ステータス

GA

## 概要

7月1日に Pro / Pro+ / Max プラン向けに提供開始された Kimi K2.7 Code が、Copilot Business および Enterprise プランでも利用可能になった。管理者によるポリシー有効化が必要。

## 詳細

### 背景

2026年7月1日に Copilot Pro、Pro+、Max プラン向けに一般提供が開始されたオープンウェイトモデル Kimi K2.7 Code が、Business および Enterprise プランにも拡大された。

### 有効化手順

Kimi K2.7 Code は Copilot Business / Enterprise プランでは**デフォルト無効**。利用するにはプラン管理者が Copilot 設定で Kimi K2.7 Code ポリシーを明示的に有効にする必要がある。ポリシーが無効のままの場合、組織内のユーザーはモデルピッカーでこのモデルを選択できない。

### 推奨事項

管理者はオープンウェイトモデルを有効にする前に、自組織のセキュリティ・コンプライアンス・データガバナンス要件を確認することが推奨される。

### 課金

使用量ベース課金のプロバイダー公式価格で課金される。詳細は GitHub Copilot のモデル・リクエスト価格表を参照。
