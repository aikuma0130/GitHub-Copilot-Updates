# GitHub Copilot in Visual Studio Code — 2026年6月リリースまとめ

## 日付

2026-07-08

## ステータス

GA

## 概要

VS Code v1.123〜v1.127 で提供された Copilot 関連アップデートのまとめ。統合ブラウザの GA、並列セッション管理、コスト可視性の向上、Marketplace からのモデルプロバイダー検出、Autopilot の改善などが含まれる。

## 詳細

### 統合ブラウザの更新

- エージェント用ブラウザツールが GA。ページ操作、スクリーンショット取得、Web アプリ検証が VS Code 内で可能に。
- お気に入り、履歴、Web 検索機能の追加。カメラ・位置情報・マイクなどのパーミッション対応。
- スクリーンショットやエリアスクリーンショットをチャットに追加する新しい方法。
- リモートワークスペースでの HTTP(S) プロキシ経由ブラウジング（パブリックプレビュー）。

### 並列セッションとチャット

- 複数のエージェントセッションを並べて表示・実行可能に。
- 1つのセッション内で複数チャットを使い、実装・レビュー・テスト・ドキュメントを分離可能に。
- セッションのグループ化やドラッグ＆ドロップによる整理機能。

### コスト可視性の向上

- セッション全体のクレジット使用量を確認可能に。
- GitHub Copilot ステータスダッシュボードから追加使用量を確認。
- サブエージェントごとのクレジット使用量の確認機能。

### Marketplace からのモデルプロバイダー

- Language Models エディタからモデルプロバイダー拡張を検出可能に。
- Marketplace からフィルタリングされた結果で拡張をインストール。
- コンテキストサイズや推論エフォートの統一ピッカーからのカスタマイズ。

### Autopilot の改善

- タスク完了の判定がより正確に。
- 手動介入なしでステップを進行する能力が向上。

### その他のエージェントワークフロー改善

- セッション同期と Chronicle：GitHub アカウントへのチャットセッション同期、マシン間のコーディング履歴検索。
- ガターフィードバック：エディタのガターからエージェントの変更に直接コメント。
- PR 作成の改善：セッションコンテキストからタイトルと説明を自動生成。
- 100万トークンコンテキストウィンドウ：対応する Anthropic/OpenAI モデルで大規模コードベースに対応。
- モデルホバーカード、公式 Ollama 拡張、マネージド Copilot 設定、MCP OAuth 資格情報、拡張自動更新遅延、Workspace Trust の改善。
