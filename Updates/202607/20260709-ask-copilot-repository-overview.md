# Copilot にリポジトリの概要を聞く機能

## 日付

2026-07-09

## ステータス

GA

## 概要

github.com で初めて訪問するリポジトリに対し、Copilot がリポジトリの目的・使用技術・コントリビューションガイドラインの概要を自動生成する機能が追加。全 Copilot プランで利用可能。

## 詳細

### 機能概要

リポジトリのホームページを訪問した際に、Copilot が「Give me a high-level overview」ボタンを提供する。選択すると、リポジトリのコンテキストを収集し以下の情報をまとめて返す：

- リポジトリの目的
- 使用されている技術
- コントリビューションガイドライン

### README がないリポジトリ

README が存在しないリポジトリでは、Copilot が README を自動生成してリポジトリの内容を把握できるようにする。

### アクセス方法

- リポジトリ訪問時に表示されるショートカットから利用
- github.com ナビゲーションバーの Copilot アイコンから利用
- Copilot Chat でリポジトリ概要の生成を依頼

### 対象プラン

全 GitHub Copilot プランで利用可能。
