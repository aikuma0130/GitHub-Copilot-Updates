# OpenAI GPT-5.6 Sol・Terra・Luna が GitHub Copilot で利用可能に

## 日付

2026-07-09

## ステータス

GA

## 概要

OpenAI の GPT-5.6 ファミリー（Sol、Terra、Luna）が GitHub Copilot で利用可能に。用途に応じて高推論・汎用・軽量の3バリアントから選択でき、VS Code、CLI、クラウドエージェントなど全サーフェスで段階的にロールアウト。

## 詳細

### モデルバリアント

- **GPT-5.6 Sol**: ファミリー内で最高の推論能力。大規模コードベースでの複雑な推論や長時間のエージェント作業に最適。
- **GPT-5.6 Terra**: バランスの取れたデフォルト。日常的なインタラクティブおよびエージェントコーディングに最適な汎用モデル。
- **GPT-5.6 Luna**: 軽量でコスト効率の高いバリアント。小規模で高速なタスク向けで、ファミリー内で最もコストが低い。

### 利用可能なプラン

- GPT-5.6 Sol: Pro+、Max、Business、Enterprise
- GPT-5.6 Terra / Luna: Pro、Pro+、Max、Business、Enterprise

### 対応サーフェス

- Visual Studio Code
- Visual Studio
- Copilot CLI
- GitHub Copilot クラウドエージェント
- GitHub Copilot アプリ
- github.com
- GitHub Mobile（iOS / Android）
- JetBrains
- Xcode
- Eclipse

### 有効化

Business / Enterprise プランの管理者は Copilot 設定で GPT-5.6 モデルのポリシーを有効にする必要がある（デフォルトはオフ）。

### 課金

使用量ベース課金（Usage Based Billing）でプロバイダーリスト価格が適用される。
