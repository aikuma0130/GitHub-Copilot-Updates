# GitHub Mobile で Copilot セッションのフィルター・ソート機能が改善

## 日付

2026-07-10

## ステータス

GA

## 概要

GitHub Mobile で Copilot セッション一覧のフィルターおよびソート機能が改善され、セッション数が増えても目的のセッションを素早く見つけられるようになった。

## 詳細

### 新しいフィルター

モバイル向けに設計された軽量なフィルターでセッション一覧を絞り込み可能:

- **アクティブ**（非アーカイブ）
- **ステータス**
- **リポジトリ**
- **タイプ**
- **エージェント**
- **ソート**

### ソートオプション

セッションを以下の順序で並べ替え可能:

- 最新順
- 古い順
- アクティブ優先
- 注意が必要なもの優先

ソート順を変更してもフィルターコンテキストが保持されるため、最初からやり直すことなくリストを絞り込める。

### 対応プラットフォーム

- iOS（最新プロダクションビルド）
- Android（最新プロダクションビルド）
