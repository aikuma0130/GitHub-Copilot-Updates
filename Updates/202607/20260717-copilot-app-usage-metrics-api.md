# GitHub Copilot アプリが使用メトリクス API で利用可能に

## 日付

2026-07-17

## ステータス

GA

## 概要

Copilot 使用メトリクス API が GitHub Copilot アプリの使用状況を追跡・レポート可能に。エンタープライズ・組織レベルの1日・28日レポートでアプリ利用データが確認可能。

## 詳細

### 変更内容

使用メトリクス API のレポートに GitHub Copilot アプリ（デスクトップアプリ）がサーフェスとして追加された。これにより、管理者はアプリ経由の Copilot 利用パターンを他のサーフェス（IDE、CLI 等）と同様に把握可能になった。

### レポートスコープ

- エンタープライズレベル（1日・28日サマリー）
- 組織レベル（1日・28日サマリー）

### メリット

- AI 採用パターンの追跡精度が向上
- 使用量ベース課金環境での利用状況の可視性が向上
- Copilot アプリの導入・活用状況を定量的に把握可能
