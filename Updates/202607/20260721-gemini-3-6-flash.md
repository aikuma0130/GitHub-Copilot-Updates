# Gemini 3.6 Flash が GitHub Copilot で利用可能に

## 日付

2026-07-21

## ステータス

GA

## 概要

Google の最新 Flash モデル「Gemini 3.6 Flash」が GitHub Copilot で利用可能に。Web/アプリ開発やエージェント型タスクに最適化されており、Gemini 3.5 Flash と比較して平均17%少ない出力トークンでタスクを完了し、コスト効率と性能が向上。

## 詳細

### 対応環境

Gemini 3.6 Flash は以下のすべての Copilot サーフェスのモデルピッカーから選択可能:

- Visual Studio Code
- Visual Studio
- Copilot CLI
- GitHub Copilot クラウドエージェント
- GitHub Copilot アプリ
- JetBrains
- Xcode
- Eclipse

### 対象プラン

- Copilot Pro
- Copilot Pro+
- Copilot Max
- Copilot Business（管理者による有効化が必要）
- Copilot Enterprise（管理者による有効化が必要）

### 主な特徴

- **トークン効率**: 出力トークンが平均17%削減（特定のコーディングタスクでは最大65%削減）
- **価格**: 出力トークンは100万トークンあたり$7.50（$9から値下げ）、入力トークンは$1.50/100万トークン
- **設計思想**: Web/アプリ開発、長期間のエージェント型タスク向けに設計
- **機能**: 設定可能な推論エフォート、並列ツール使用をサポート
- **パフォーマンス**: Gemini 3.5 Flash と比較してタスク完了率が向上

### 有効化方法

- Pro / Pro+ / Max ユーザー: モデルピッカーから直接選択可能
- Business / Enterprise: 管理者が Copilot 設定で「Gemini 3.6 Flash Preview」ポリシーを有効化する必要あり

### 課金

プロバイダーリスト価格での従量課金制。詳細は GitHub Copilot のモデルと価格のドキュメントを参照。
