# Visual Studio Code v1.123.0 (June 2026) — GitHub Copilot 関連アップデート

## リリース日

2026-06-03

## VS Code バージョン

1.123.0

## 概要

エージェントセッションの同期・検索機能やリサーチエージェント、統合ブラウザのスクリーンショット強化など、GitHub Copilot のエージェント体験を大幅に拡張するリリース。

---

## GitHub Copilot 関連の変更点

### セッション同期と Chronicle

**設定**: `chat.sessionSync.enabled`（組織レベルで管理）

チャットセッションが GitHub アカウントに自動同期されるようになり、マシンやワークスペースをまたいで個人的な作業履歴を検索できるようになりました。

各セッションには、会話内容、操作したファイル、リポジトリコンテキスト（リポジトリ、ブランチ、タイムスタンプ）、参照されたプルリクエスト・Issue・コミットが記録されます。

新しい Chronicle コマンド（`/chronicle`）により、以下が可能になります：

- 過去のセッションに対する自然言語での質問
- スタンドアップレポートの生成
- パーソナライズされた生産性のヒント
- トピック、ファイル、PR による作業履歴の検索

---

### サンドボックスでのネットワーク依存コマンドのリトライ

**設定**: `chat.agent.sandbox.retryWithAllowNetworkRequests`

ローカルエージェントが実行するターミナルコマンドが、許可ドメインとして設定されていないドメインへのアクセスを必要とする場合、ネットワークアクセス制限なしのサンドボックス内で自動的にリトライされるようになりました。それでも失敗する場合は、サンドボックス外での実行にフォールバックします。これにより `git fetch` などのネットワーク依存操作が、ファイルシステム保護を維持したまま完了できるようになります。

---

### Agents ウィンドウ（プレビュー）

Agents ウィンドウで複数のエージェントセッションを同時に開けるようになりました。アクティブなセッションに加えて、別のセッションを横に並べて表示できます。

複数セッションの表示方法：

- セッションリストのコンテキストメニューから「Open to the Side」を選択
- セッションリストからセッションビューエリアへドラッグ＆ドロップ
- Alt キーを押しながらセッションリストのセッションを選択

セッションビューをピン留めすることで、他のセッションを選択しても置き換えられなくなります。また、最大化アクションで単一セッションのフォーカスビューに切り替えることもできます。

---

### リサーチエージェント（プレビュー）

> **注意**: リサーチエージェントは現在プレビュー段階で、Insiders の Copilot CLI（ローカル）セッションでのみ利用可能です。

不慣れなコードの理解、アプローチの比較、ライブラリや API の動作の学習など、簡単なチャット回答では不十分な場合に、リサーチエージェントがトピックに対する深い調査を実行し、コードベース、関連する GitHub リポジトリ、Web からの情報を収集・統合して、引用付きの詳細な Markdown レポートを生成します。

リサーチエージェントは速度よりも深さに最適化されており、読み取り専用アクセスのみ持つため、コードを変更せずに調査・報告を行います。使用するには、Copilot CLI（ローカル）セッションのチャット入力で `/research` に続けてトピックを入力します。

---

### 統合ブラウザのスクリーンショット機能強化

**設定**: `workbench.browser.experimentalUserTools.enabled`

前回のリリースで導入された「Add Screenshot to Chat」に加えて、2つの新しいスクリーンショット機能が追加されました：

- **Add Area Screenshot to Chat**: 選択した矩形エリアのスクリーンショットを撮影し、チャットコンテキストとして添付
- **Add Full Page Screenshot to Chat（実験的）**: 現在のビューポートを超えたページ全体のスクリーンショットを撮影し、チャットコンテキストとして添付（`workbench.browser.experimentalUserTools.enabled` 設定の有効化が必要）

これらは UI 関連のタスク（レイアウトの問題のデバッグなど）に特に有用です。

---

### 統合ブラウザのお気に入りページ

統合ブラウザのアドレスバーが刷新され、URL 入力だけでなく、ページのお気に入り登録や、お気に入り・開いているタブへの素早いアクセスが可能になりました。ブラウザ URL バーのスターアイコンでお気に入りに追加できます。
