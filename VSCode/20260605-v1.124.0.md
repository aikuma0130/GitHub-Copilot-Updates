# Visual Studio Code v1.124.0 (June 2026) — GitHub Copilot 関連アップデート

## リリース日

2026-06-05

## VS Code バージョン

1.124.0

## 概要

Agents ウィンドウの WSL 接続対応やバックグラウンド送信、セッション管理の強化など、エージェント体験の利便性を向上させるリリース。

---

## GitHub Copilot 関連の変更点

### Agents ウィンドウの WSL 接続サポート

Agents ウィンドウが WSL（Windows Subsystem for Linux）接続をサポートするようになりました。WSL 環境内でもエージェントセッションを直接利用できるようになります。

_[#307568](https://github.com/microsoft/vscode/issues/307568)_

---

### Agents ウィンドウのチャット入力履歴のセッションスコープ化

Agents ウィンドウのチャット入力欄で `Up`/`Down` キーを押した際の入力履歴が、現在のセッション内で送信したプロンプトのみに限定されるようになりました。他のセッションのプロンプトは表示されなくなり、セッション内のコンテキストに集中しやすくなります。

_[#319341](https://github.com/microsoft/vscode/issues/319341)_

---

### Agents ウィンドウのローカルセッションでのマルチチャットサポート

Agents ウィンドウのローカルセッションで複数のチャットを同時に開けるようになりました。複数の会話を並行して進めることが可能になります。

_[#319602](https://github.com/microsoft/vscode/issues/319602)_

---

### Agents ウィンドウのバックグラウンド送信

`Alt+Enter` または `Alt` キーを押しながら送信ボタンをクリックすることで、セッションに移動せずにメッセージを送信できるようになりました。送信後すぐに次のメッセージを作成できるため、複数のセッションを効率的に管理できます。

_[#319598](https://github.com/microsoft/vscode/issues/319598)_

---

### セッショングリッドのキーボードナビゲーションと一括クローズ

セッショングリッドで以下のキーボード操作が追加されました：

- `Ctrl+1`〜`Ctrl+9`（`Cmd+1`〜`Cmd+9`）: 位置指定でセッションにフォーカス
- `Ctrl+K Ctrl+W`（`Cmd+K Cmd+W`）: すべてのセッションを一括クローズ

_[#319596](https://github.com/microsoft/vscode/issues/319596)_

---

### 統合ブラウザツールバーのカスタマイズ

統合ブラウザのツールバーがカスタマイズ可能になりました。コンテキストメニューから個々のツールバーアクションの表示・非表示を切り替えられます。

_[#320016](https://github.com/microsoft/vscode/issues/320016)_ _[#318437](https://github.com/microsoft/vscode/issues/318437)_

---

### 統合ブラウザの URL 履歴

統合ブラウザのアドレスバーで `Up`/`Down` キーを押すと、以前にアクセスした URL を遡ることができるようになりました。

_[#303767](https://github.com/microsoft/vscode/issues/303767)_
