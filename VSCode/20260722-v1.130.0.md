# Visual Studio Code v1.130.0 (July 2026) — GitHub Copilot 関連アップデート

## リリース日

2026-07-22

## VS Code バージョン

1.130.0

## 概要

エージェントホストの継続的な改善、アシスト付きツール承認機能の追加、Agents ウィンドウでのレビューワークフローの効率化、Copilot Business/Enterprise 向けクレジット使用量の可視化など、エージェント体験をさらに強化するリリース。

---

## GitHub Copilot 関連の変更点

### エージェントホストの継続的改善

前回のリリースで導入されたエージェントホストの開発が継続されています。エージェントホストは [Agent Host Protocol](https://microsoft.github.io/agent-host-protocol/)（AHP）に基づく専用プロセスで、Copilot、Claude、Codex などのエージェントハーネスを実行します。

エージェントホストの Copilot エージェントは引き続き [Copilot SDK](https://www.npmjs.com/package/@github/copilot-sdk) を使用しており、Copilot CLI、スタンドアロン GitHub Copilot アプリ、その他の Copilot 製品と動作・機能が統一されています。

オプトインするには `chat.agentHost.enabled` を有効にし、ハーネスドロップダウンからエージェントホストハーネスを選択します。この設定は組織レベルで管理されるため、変更するには管理者に連絡する必要があります。

---

### アシスト付きツール承認

**設定**: `chat.assistedPermissions.enabled`

長時間実行されるエージェントタスク中に繰り返されるツール承認プロンプトを軽減する新機能です。アシスト付き権限を有効にすると、言語モデルが各ツール呼び出しのリスクを評価し、ツールを自動実行するか、ユーザーの承認を求めるかを判断します。

この設定を有効にすると、エージェントホスト上で動作するエージェントの権限ピッカーに **Assisted permissions** オプションが追加されます。

---

### Agents ウィンドウの改善（プレビュー）

[Agents ウィンドウ](https://code.visualstudio.com/docs/agents/agents-window)にいくつかの更新が加えられ、変更のレビューとチャット管理がより簡単になりました。

#### ファイルレベルの差分統計

**Changes** エディターの各ファイルヘッダーに、ファイルパスの横に挿入行数と削除行数がリアルタイムで表示されるようになりました。複数ファイルの差分をスキャンする際に、各ファイルの変更規模をすばやく把握できます。

#### コンパクトなマルチファイル差分ビュー

マルチファイル差分がよりコンパクトなガターを使用するようになり、コードの前の空白スペースが削除されました。ファイルヘッダー、行番号、未変更領域のコントロールが一貫したアラインメントになり、狭いエディターでも変更をレビューするためのスペースが広がりました。

#### コンパクトなクイックチャット

エージェントホスト上で動作するクイックチャットが、セッションリストでコンパクトな1行表示になりました。通常のセッションは変更統計、ステータス、タイムスタンプを含む2行目を維持するため、クイックチャットをより区別しやすくなり、プロジェクトセッション用のスペースが広がりました。

#### 全エージェントハーネスでのワークツリーサポート

エージェントホスト上で動作するすべてのエージェントハーネスがワークツリー分離をサポートするようになりました。Agents ウィンドウの **New Worktree** チェックボックスは以前 Copilot ハーネスのみでサポートされていましたが、Claude および Codex セッションも Git ワークツリーで実行できるようになり、ハーネスに関係なく同じワークスペースで異なる機能の並列セッションを簡単に起動できます。

---

### チャットタイムスタンプ

**設定**: `chat.verbose`

チャットのリクエストとレスポンスにタイムスタンプが表示されるようになりました。メッセージのツールバーにホバーすると、チャットインタラクションのタイムスタンプと経過時間を確認できます。`chat.verbose` 設定で無効にすることも可能です。

---

### Copilot Business/Enterprise 向け AI クレジット使用量の集計表示

Copilot Business および Copilot Enterprise ユーザーが、現在の請求サイクルにおける AI クレジット使用量の集計を Copilot ステータスメニューで直接確認できるようになりました。

以前は、ユーザーレベルの予算が設定されている場合にのみクレジット使用量が表示されていたため、組織管理のユーザーの多くは消費したクレジット数を把握できませんでした。

ユーザーレベルの予算が設定されていない場合でも、ステータスメニューに請求サイクル中に使用されたクレジットの合計数が表示されるようになりました。エディターを離れることなく使用パターンを把握できます。
